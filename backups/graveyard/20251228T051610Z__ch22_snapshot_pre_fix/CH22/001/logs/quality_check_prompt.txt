## 1) 最重要ルール（音声品質の下限）

### 1.1 ポーズ挿入の唯一の記号
- **ポーズ挿入として許可される記号は `---` のみ**。
- `---` は **1行単独**で置く（前後は改行）。意味の切れ目（話題転換/場面転換/呼吸の区切り）にだけ使う。
- 原則: **セクションが切り替わる境界**（設計図/構成上の区切り）で `---` を置く（TTS側の0.5秒ポーズに対応）。
- 音声側の解釈（strict）:
  - 通常のつなぎは **0.1秒**（デフォルト）
  - `---` は **0.5秒**の無音ポーズ
  - 空行/改行はポーズ指示として扱わない（整形のために入れてもよい）
- `***` や `___`、`///`、`===` 等の区切り記号は使用しない（音声の不自然な途切れや事故の原因）。

### 1.2 記号・括弧の過剰使用を禁止（TTSが不自然に切れやすい）
- `「」` / `『』` と `（）` は **多用しない**（必要な場面だけ）。原則として **直接話法より間接話法** を優先する。
- 引用はできるだけ `「」` / `『』` を使わず、地の文で言い換える（例: “ある人はこう言いました” の形）。
- 目安（TTS事故防止）: 1本あたり `「」` / `『』` は **合計20個（10組）以内**、`（）` は **合計10個（5組）以内**。超える場合は意味を保ったまま言い換える。
- 運用上の扱い: この「目安」は **script_validation のハード上限**として扱う（超過は不合格）。
  - 上限は `configs/sources.yaml` の `script_globals.a_text_quote_marks_max` / `script_globals.a_text_paren_marks_max` が正本（必要ならチャンネル別override可）。
  - 超過した場合は **自動クリーナ（非LLM）で括弧だけ外して基準内に落とす**（意味は維持・全文リライト禁止）。
  - その結果、字数が下限を割った場合は **Extend/Expand（追記のみ）で回復**する（リトライ回数で押し切らない）。
- 会話の連打でテンポを作らない。台詞が必要なら **短く**、地の文に溶かし込む。
- ルビ/読み仮名/注釈目的の括弧は原則禁止（読み上げが破綻しやすい）。

### 1.3 視聴者ストレスを生む文章を禁止
- 淡々とした繰り返し構文、同じ言い回しの連発、蛇足の比喩、回りくどい前置き、過剰な煽りを避ける。
- 断定・説教・脅し・人格否定はしない。必要なら事実と提案に落とす。
- “言い換えで水増し” をしない。厚みは **具体の追加**で作る（状況・行動・心の動き・結果）。
- **同一段落の丸ごと重複は禁止**（コピー/生成ループ由来の反復）。  
  目安（機械チェック）: 120文字以上の段落が同一（空白差を無視して一致）なら不合格。

### 1.4 捏造・虚構の禁止（信頼を壊さない）
- 根拠のない統計・割合（`％` / `パーセント`）・研究結果・大学名/研究所名・論文/実験の作り話を本文に書かない。
- 事実確認ができない固有名詞や数値は使わず、必要なら一般化して述べる（例: “研究では〜” の断定や数字は避ける）。
- 科学/医学の橋渡しは、**安全な一般論を短く1段落**に留める（呼吸と緊張、書くことと客観視など）。数字・機関名・論文名は出さない。

### 1.5 「字数合格だけ」を禁止（LLM品質ゲート必須）
- 文字数や禁則がOKでも、内容が薄い/冗長/流れが不自然なら **不合格**とする。
- フロー/自然さの合否は **機械的ルールではなく推論モデルの判定（LLM Judge）**で担保する。
- 運用の正本: `ssot/ops/OPS_A_TEXT_LLM_QUALITY_GATE.md`（Judge→Fixer の2段階で最終品質を固定）。

### 1.6 終わり方（未完を禁止）
- Aテキストは **最後まで完結**していること（途中で切れて終わるのは不合格）。
- 目安（機械チェック）:
  - 最後の有効文字が `。` / `！` / `？` / `!` / `?` / `」` / `』` などで閉じている。
  - 助詞や語尾の途中（例: `〜になり` / `〜で` / `〜が`）で止まらない。
- これは「決まり文句を入れろ」ではなく、**ぶつ切り事故（生成途中で止まる）を確実に検知して止める**ためのルール。

### 1.7 文字の健全性（外字/制御文字を禁止）
- `�`（U+FFFD, Unicode replacement character）が混入している場合は不合格（コピー事故/文字化けの典型）。
- 目に見えない制御/整形文字（例: ゼロ幅スペース等）や、私用領域（PUA）等の特殊文字が混入している場合は不合格。
- 明らかな誤字・外字が検出された場合は不合格（TTSの読み崩れ/視聴者ストレスの原因）。  
  既知の混入パターンは `script_validation` の決定論クリーナで **置換修復**する（全文リライトは禁止）。

## 2) Aテキストの禁止事項（混入事故を根絶）

台本本文（Aテキスト）に入れない:
- URL（`http(s)://` / `www.`）
- 出典メタ、脚注、参照リンク、引用番号（例: `[13]` / `([xxx][13])`）
- 箇条書き/番号/記号リスト（`-` / `・` / `1.` / `A.` 等）
- アウトライン由来のメタ（例: `約600字`、`A. 見出し - 約600字` などの「字数メモ」）
- Markdown装飾（例: `**太字**`）
- 設定/CSVデータ/構成案/プロット等の **制作メモの貼り付け**（例: `設定` 見出し＋ `主人公: ...` の羅列）
- エピソード見出し行（例: `CH23-001: ...`）
- タイムスタンプ、チャプター表、制作裏（AI/収録/台本などのメタ言及）
- 章立てメタ（例: `第3章` / `第3章を始めましょう` のような“章見出し”）

※出典は本文ではなく `content/analysis/research/references.json` 等の research 側へ集約する。

## 3) 読みやすさ（耳で理解できる最小設計）

- 1段落は1話題。2〜4文を目安に詰め込みすぎない。
- 1文は長くしすぎない（息継ぎできる長さ）。主語が迷子にならないように整える。
- 専門語は最小限。出すなら直後にやさしい言い換えを添える（定義の長話はしない）。
- 比喩は “一度で伝わる短さ” に限定し、連打しない。

## 4) チャンネル別ルールとの優先順位（衝突時の扱い）

1. 本書（全チャンネル共通のAテキスト品質）  
2. チャンネル固有の `script_prompt.txt` / Persona / 企画CSV  
3. 作業メモ・一時指示

補足:
- チャンネル `script_prompt.txt` に構成/形式/記号の指示が混ざっていても、Aテキスト向けLLM呼び出しでは衝突しやすい要素を落とした `a_text_channel_prompt`（派生）を用いる。骨格は `ssot/ops/OPS_SCRIPT_PATTERNS.yaml` が正本。

例外が必要な場合:
- チャンネル固有の理由と例外条件を **SSOTに明記**してから適用する（無断で例外を増やさない）。


【対象コンテンツ】
かつては何でも話し合い、支え合ってきたあの友人関係が、いつの間にかゆっくりと変化していったのはなぜだろう。長い年月を共に過ごしたからこそ、そのすれ違いは静かで、気づいた時には少し距離ができていた。

私たちは同じ町内で子育てをしながら、三十年近くの付き合いになっていた。子どもたちがまだ小さかった頃から、公園で一緒に時間を過ごし、学校の送迎を代わり合い、悩み事があれば夜遅くまで電話で話し込んだものだ。誰かが困っていれば、すぐに手を差し伸べ合える関係。お互いの家庭の事情もよく知っている。誰かの夫が転勤で単身赴任になった時は、残された家族のサポートを自然と輪になってしたし、誰かが病気になれば、順番に見舞いに行った。

子どもが独立し、家が急に静かになった時も、私たちはよく集まった。子育てから解放された喜びと、同時に訪れた虚しさを分かち合うように。そして、誰かの夫に先立たれた時。それは私たちのグループにとって初めての身近な死だった。残された友人は、突然の寂しさに押し潰されそうになっていた。私たちは代わる代わる家を訪ね、話し相手になり、ただ傍にいた。あの時は、同じ世代として、同じ女性として、言葉にならない支え合いがあった。

しかし、ここ数年、何となく空気が変わっていたのかもしれない。集まった時の話題が、どうしても過去の思い出話に偏りがちになる。現在進行形の悩みや、これからの計画について深く話し合うことが、以前より減ったように感じる。それはお互いの生活が落ち着いたからだろうか。それとも、それぞれが自分の世界に少しずつ引きこもり始めていたからか。

連絡を取る頻度が減っていることには、自分でも気づいていた。メールやメッセージのやり取りが、用件だけの簡素なものになっていく。誕生日の祝いも、かつては手作りの品を贈り合ったが、今は定型文のメッセージを送るだけ。小さな変化は、一つひとつは大したことではない。むしろ、年を重ねたからこその、お互いを尊重する距離なのだと、自分に言い聞かせてもいた。

だが、ふと気づくことがあった。最近のランチ会の話を、なぜか直接聞いていない。他の誰かから、そういえば先日集まったよ、と聞くことがちらほらと。最初は、たまたま連絡が漏れただけだと思った。でも、それが二度、三度と続くと、もしかして、と頭をよぎる。それでも、あの長い付き合いなのだから、わざわざ外す理由などない、とすぐに打ち消した。気のせいだ、きっと偶然だ。そう思いたかった。

この静かな変化は、ある日突然に起きたわけではない。十年、二十年という長い時間をかけて、ゆっくりと、ほとんど目に見えない速度で進行していた。同じ町内に住みながら、子どもの進学や就職、親の介護や自分の健康問題など、それぞれが別々の人生の波を乗り越えていくうちに、歩調が少しずつずれていったのかもしれない。悪意や確執があったわけではなく、ただ時間が流れた結果なのだろう。

かつてはあんなに近かった距離感が、知らぬ間にほんの数センチ、また数センチと広がっていく。それは、関係が悪化したというより、むしろ自然な成熟のようなものだ。お互いが無理をしなくてもいい楽な間柄になった、と前向きに解釈することもできた。でも、時折、昔のように何でも話せたあの頃が懐かしくて、ふと寂しさがこみ上げてくることもあった。

この静かな変化は、やがて小さなすれ違いとして積もっていきます。

一つひとつは大したことではないのに、なぜ心に棘が刺さり続けたのでしょうか。

あのランチ会の写真を目にするより前から、何となく空気が変わっていたことに、ふと気づく瞬間はありました。例えば、三人で話している最中に、ふと二人だけの内緒話のような笑い声が交わされる。あるいは、次回の集まりの日時が、なぜか私の都合を聞かれる前に決まっている。そんな小さな違和感が、ぽつりぽつりと降り積もるように増えていったのです。

最初は気のせいだと思いました。長い付き合いなのだから、少しぐらいのずれはある。そう自分に言い聞かせて、その場を流していました。しかし、流せば流すほど、次に来る違和感は少しずつ形を変え、少しずつ重くなっていく。それがこの数年、いや、もっと前から静かに続いていたのかもしれません。

具体的な出来事を一つ思い出します。ある時、旧友たちとランチの約束をしていました。ところが前日、急に体調を崩してしまい、仕方なくキャンセルの連絡を入れました。申し訳ないと思い、次回は必ずと伝えました。相手も優しい言葉を返してくれ、それで一件落着したはずでした。

ところが数週間後、別の友人から偶然聞いた話です。私がキャンセルしたその次の週、同じメンバーでまた集まっていたというのです。もちろん、私への誘いはありませんでした。聞いた瞬間、胸が少し締め付けられるような感覚がありました。でもすぐに、きっと都合が合わないと思われただけだろう、あるいは単なる連絡漏れだろうと、自分を納得させました。わざわざ問い詰めるほどのことでもない、そう思ったのです。

それからしばらくして、今度は町内会の寄り合いの後でした。終わって解散する時、隣にいた友人たちが、今度一緒にカフェに行かないかと話し始めました。私は自然にその輪に入り、そうねえと相槌を打ちました。でも、具体的な日時はその場では決まらず、後で連絡すると言われたきり、その連絡は来ませんでした。一週間経ち、二週間経っても何もない。結局、私は誰にも聞かず、ただ待つだけでした。聞けばいいのに、なぜか聞けなかった。聞くことが、まるでみっともないことのように思えたからです。

こんな小さなことは、数え上げればきりがありません。グループメッセージでの既読スルー。写真が投稿された時の、私だけタグがついていないこと。何気ない会話の中で、私の家族の近況について、なぜか誰も知らないふりをすること。一つひとつは、本当に些細なことです。大げさに考えすぎだと、自分でも思います。

だからこそ、誰にも相談できません。こんなことで悩んでいるなんて、大人げないと言われそうで。それに、もし私が誤解していたら、関係が気まずくなるだけだ。そう考えて、すべてを胸の内にしまい込みました。見て見ぬふりをすれば、いつか元通りになるはずだ、と信じたいかったのです。

しかし、心の中では確実に摩耗が進んでいました。毎回、ちょっとした違和感を感じるたびに、少しずつエネルギーが削られていく。次に会う前には、どう振る舞おうか、何を話そうかと、必要以上に考え込むようになりました。以前は自然にできていたことが、いつの間にか計算ずくになっていく。楽しいはずの時間が、だんだんと気苦労に変わっていったのです。

それでも、長い友情です。二十年、三十年の付き合いです。そんな関係を、ちょっとしたすれ違いで壊したくはありません。だから、ますます強く自分に言い聞かせました。きっと私が敏感すぎるのだ、気にしすぎなのだ、と。相手たちに悪意はない、ただの偶然や勘違いの積み重ねなのだ、と。

そうやって自分を説得する日々が、どれだけ続いたでしょう。気づかないふりをすればするほど、違和感はかえって鮮明になり、心に刺さった棘は深く食い込んでいきます。それでも、まだ抜こうとはしなかった。抜けば血が出る、関係が壊れるかもしれないという恐れがあったからです。

外界で起きる小さな出来事と、内面で繰り返される自己説得。その往復が、静かに日常をすり減らしていきました。特別な大事件が起きたわけではありません。ただ、無視され続けた小さな違和感が、積もりに積もって、やがては明確な溝へと変わっていく過程だったのです。

そして、その積もった棘が、ある瞬間に明確な傷として現れるのです。

その写真は、何気なくSNSを眺めている時に、ふと目に入りました。

楽しそうにランチを囲む旧友たちの顔が、何枚か並んで写っています。ケーキがテーブルの中央にあり、誰かが撮ったのでしょう、自然な笑顔が並んでいる。その輪の中に、自分がいない。ただ、それだけの事実が、画面を通して突き刺さってきました。

まず感じたのは、胸のあたりが一瞬で冷たくなる感覚でした。息が詰まるというか、目の前が少し暗くなるような。スマホの画面を見つめる自分の手が、なぜか少し震えているのに気づきました。ああ、そうか。誘われなかったんだ。この事実が、ゆっくりと、しかし確実に心に染み込んでいきます。

そして次に、頭の中を駆け巡ったのは、自分を責める言葉の数々でした。

私の何がいけなかったのだろう。前回の集まりの時、何か失礼なことを言ってしまっただろうか。もっと気を利かせて、自分から声をかけていればよかったのか。最近、連絡の返事が遅くなっていたからか。それとも、ただ単に、私がいない方が皆が気楽なのだろうか。

長い付き合いだからこそ、この傷つきは深く、そして全てが自分の責任であるかのように思えてしまう。何十年も築いてきた関係なのに、どうしてこんなことになったのか。その理由を、つい自分の内側にばかり探してしまうのです。

スマホの画面はもう暗くなっているのに、その光景が瞼の裏に焼き付いて離れません。あの笑顔の輪の中に、自分は入れてもらえなかった。この事実だけが、ぐるぐると頭の中を回り続けます。

自分だけが置いてけぼりにされたような、そんな孤独感が、じわりと広がっていきます。まるで学生時代に、仲間外れにされた時のような、あの胸が締め付けられる感覚がよみがえってくる。歳を重ねたはずなのに、感じる傷つきは昔と何も変わらないのだな、とふと思いました。

# Qwen台本作成 初回指示テンプレート（NAV/GUIDE 準拠）

あなたは `youtube_master` の台本ラインを自律実行するオペレーターです。詳細手順・禁止事項は `ssot/ops/OPS_SCRIPT_NAV.md` と `ssot/ops/OPS_SCRIPT_GUIDE.md` を唯一の参照先とし、他の資料に従わないでください。

## 実行ステップ（台本のみ・音声非対象）
1) **対象を決める**  
   - `progress/channels/CHxx.csv` の `pending` / `in_progress` 行を選ぶ。順番逆行は禁止。  
   - 必要に応じて: `PYTHONPATH=. python3 core/tools/progress_manager.py summary --channel-code CHxx`
2) **ステージを進める**  
   - 実行場所: `commentary_01_srtfile_v2/`  
   - 入口コマンド（共通）  
     ```
     LLM_RUNNER_AUTO_PLACEHOLDERS=1 PYTHONPATH=. python3 qwen/cli.py next --channel-code CHxx --video-number NNN
     ```
   - 台本ステージ（topic_research → script_polish_ai まで）を順に完了させる。音声/SRT ステージには触れない。
3) **最終ガード**  
   - `PYTHONPATH=. python3 core/tools/run_script_validation.py CHxx NNN --auto-title` を実行し、メタ文/プレースホルダ欠落を確認する。
4) **報告**  
   - 成功: `CHxx-NNN 台本 completed` と報告し、次の pending 行へ。  
   - 失敗: エラー内容とステージを報告。`status.json` は手動編集せず、原因を修正して同コマンドを再実行。

## SoT とアウトプット
- SoT: `data/CHxx/NNN/status.json`（ステージ状態/メタ） + `progress/channels/CHxx.csv`（派生表示）
- 成果物: `content/`（research/outline/chapters/assembled/scenes.json）, `output/CHxx-NNN_原文.txt`, `output/CHxx-NNN_台本.txt`

## 禁止事項（徹底）
- `run_pipeline.py` / `run_stage.py` / `batch_workflow.py` / Google Sheets 直参照
- status.json の手動編集、`_backup*` などの複製作成
- 音声系ステージの実行、台本にメタ/続編オファーを混入させたまま進行

本テンプレートと NAV/GUIDE が食い違う場合は NAV/GUIDE を正とし、直ちに本ファイルを更新してください。

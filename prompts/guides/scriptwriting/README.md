# guides/scriptwriting/ — 台本を「直接執筆」するための指示書

目的:
- チャンネル別の参照先（SoT）と禁則を固定し、AIエージェントが迷わず台本（Aテキスト）を書けるようにする。
- ここは「ガイド」置き場。プロンプト本文の正本は `packages/**/prompts/` や `packages/script_pipeline/channels/**/script_prompt.txt` 側（複製しない）。

入口:
- 一覧: `prompts/SCRIPT_WRITING_GUIDES_INDEX.md`
- 全チャンネル共通: `prompts/guides/scriptwriting/SCRIPT_WRITING_GUIDE_COMMON.md`
- チャンネル別: `prompts/guides/scriptwriting/channels/CHxx.md`

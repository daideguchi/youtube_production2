🎯 解説系チャンネル 完全自動化の流れ​

手順1：台本の準備​

場所：scripts/チャンネル名/ 操作：手動で台本txtファイルを配置

手順2：音声生成​

担当スクリプト：src/core_logic/tts_synthesize.py

何をする：

  • 台本を読み込み
  • VoicepeakまたはVoicevoxで音声生成
  • 音声ファイル（.wav）を出力フォルダに保存

設定ファイル：

  • configs/channel_voice_settings.json - チャンネル別音声設定
  • configs/voicevox_voice_profiles.json - 音声キャラクター設定

出力：output/{台本ID}/audio/{台本番号}_final.wav

手順3：音声解析（WhisperX）​

担当スクリプト：外部API（GCP WhisperX） API URL：https://whisperx-gpu-service-469692789396.us-central1.run.app/align

何をする：

  • 音声ファイルを解析
  • 一文字ごとの発話タイミングをJSON出力

出力：output/{台本ID}/output/whisperx/{台本番号}_buddha_full_result.json

手順4：字幕整形（LLM処理）​

担当スクリプト：tools/text_boundary_alignment.py

何をする：

  1. 台本を読みやすく整形（27文字×最大2行）
  2. WhisperX解析結果と対応付け
  3. タイミング情報を付与した字幕作成

使用AI：

  • Gemini 2.5 Pro（メイン）
  • GPT-5（フォールバック・大型台本用）

出力：

  • output/{台本ID}/output/llm_txt/{台本番号}_formatted_script_perfect.json - 整形台本
  • output/{台本ID}/output/whisperx_formatted/{台本番号}_whisperx_formatted.txt - 対応付け結果

手順5：最終SRT生成​

担当スクリプト：tools/text_boundary_alignment.py（内部処理）

何をする：

  • 整形された字幕にタイミング情報を統合
  • 標準SRT形式で出力

制約：

  • 1行最大27文字
  • 最大2行まで
  • 句点（。）は必ず改行

出力：output/{台本ID}/output/final_srt/{台本番号}_final.srt

手順6：CapCut自動制作​

担当スクリプト：tools/srt_to_capcut_complete.py

何をする：

  1. SRTファイルを読み込み
  2. 各字幕セクションに対応する画像を生成（Gemini DALL-E）
  3. CapCutプロジェクトファイル（draft_content.json）を作成
  4. 音声・字幕・画像を全自動統合

出力：capcut_projects/{台本番号}/draft_content.json

📋 補助ツール​

tools/voice_settings_manager.py​

音声設定の確認・調整ツール

python3 tools/voice_settings_manager.py --summary

tools/index_output.py​

python3 tools/index_output.py

tools/repair_images.py​

画像生成失敗時の再生成

python3 tools/repair_images.py {台本番号}

🚀 実行コマンド（まとめ）​

全工程自動実行​

# 台本ID指定で全自動
python3 tools/pipeline_batch.py {台本番号}

個別実行​

# 1. 音声生成のみ
python3 src/core_logic/tts_synthesize.py --voice Puck

# 2. 字幕整形のみ
python3 tools/text_boundary_alignment.py {台本番号}

# 3. CapCut制作のみ
python3 tools/srt_to_capcut_complete.py {台本番号}

💡 重要ポイント​

データの流れ​

台本.txt
  → 音声.wav
    → WhisperX解析.json
      → 整形字幕.json
        → 最終SRT.srt
          → CapCutプロジェクト

品質保証​

  • 音声品質：チャンネル別設定で一貫性確保
  • 字幕精度：LLM二重チェック（Gemini + 監査）
  • タイミング精度：WhisperXの高精度解析

完全自動化の鍵​

  1. 設定ファイル：チャンネル別プリセット
  2. 誤読防止辞書：固有名詞・専門用語を自動変換
  3. エラー自動修正：空ブロック検出・修正機能

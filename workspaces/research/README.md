# workspaces/research

リサーチ系スクリプトのみを置く作業場です。最新ルールとタスクは `../ssot/README.md` を参照し、ここにはスクリプト本体以外を置かないでください。

- 環境変数は必ずリポジトリ直下の `.env`（SSOT: `ssot/ops/OPS_ENV_VARS.md`）を利用します。このディレクトリに `.env` を置くことは禁止です。
- スクリプトの結果やレポートは `workspaces/logs/` や `workspaces/planning/` に集約し、ここにはワークファイルのみを残します。
- 旧資料が必要な場合は git の履歴を参照してください。

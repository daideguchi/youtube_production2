# ブッダ系サムネ・コピー強化ベンチマーク（CH12+）

目的:
- **意味（台本コア）を変えずに**、表現だけを「クリックされやすい型」に寄せる。
- 「量産（Canva）」で使う **3段コピー（上=赤 / 中=黄 / 下=白）** の作り方を固定する。

## 1) ベンチマーク（スクショから読取）

※ 文字をそのまま列挙（表現の型を学ぶため）

- 言い返さずに 勝つ方法
- なにを言われても 平気になる方法
- 心の汚い人が必ず言う 7つの言葉
- 心配性、今日で終わりです
- どうでもいい と思うほど 上手くいく！
- 自分を 傷つけた人を 許さなくていい
- 本当に優しい人は これを言う
- この言葉を 使うだけで 信頼される
- 絶対に 助けては いけない人
- あなたを 傷つけた人は こうなる
- 舐められた時 この一言が効く
- 口にしては いけない 正論3選

## 2) 目立つ「表現の型」

### 2.1 強い型（意味はそのまま・言い方だけ強化）
- **断言**: 「〜です」「〜になる」「〜で終わり」
- **禁止/注意**: 「〜してはいけない」「絶対に〜するな」
- **限定**: 「〜だけで」「これだけ」「最初に1つ」
- **必然**: 「必ず」「絶対」「100%」（※誇張にならない範囲で）
- **数字**: 「7つ」「3選」「3分」などで具体化

### 2.2 3段コピーの役割固定（量産の肝）
- 上（赤）: 危機/禁止/断言（例: 放置は危険 / それ、逆効果 / 今日で終わり）
- 中（黄）: テーマ名詞を短く（例: 夜の不安 / 執着 / 後悔 / 孤独 / 心配）
- 下（白）: 解決/利益（例: 今夜眠れる / 静まる / 抜けられる / ほどける）

## 3) 書き換えルール（運用）

- **NG**: 意味変更（台本コアとズレる断言・別テーマ化）
- **OK**: 語尾/言い回しの強化、短文化、順番入替、具体化（数字/だけ/今日で終わり）
- 迷ったら: `ssot/OPS_SEMANTIC_ALIGNMENT.md` の「意味整合」基準を優先

## 4) Canva運用メモ

- CSV列（推奨）: `page_name, thumb_upper, thumb_title, thumb_lower`
- 3段の役割は固定して、**テンプレ側（Canvaデザイン）で見た目を完全固定**する。

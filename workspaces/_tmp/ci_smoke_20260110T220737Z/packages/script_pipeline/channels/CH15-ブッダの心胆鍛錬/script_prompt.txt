# CH15 台本プロンプト（ブッダの心胆鍛錬）

役割: あなたは「ブッダの心胆鍛錬」の語り手。傷つきやすい心を持つ60代以降の視聴者に対し、「弱さは繊細さという才能である」と肯定し、しなやかな心の強さを育てる一人語り台本を書く。

ゴール: **7,000〜9,000字** の長尺一人語り。
「強くなれ」と叱咤するのは禁止。「柳のように受け流せばいい」と、柔らかな強さを説く。

構成（8パート固定。箇条書き・番号リストは禁止。ポーズは `---` のみ許可）:

1) **導入フック（共感と受容）**:
    - 「些細な言葉に傷ついてしまう、そんな優しいあなたへ」と語りかける。
    - 繊細さを「弱さ」ではなく「受信感度の高さ」と定義し直す。

2) **ブッダの逸話**:
    - 「悪口を贈り物として受け取らない」等の物語。

3) **問題の正体（反応の仕組み）**:
    - なぜ心が反応してしまうのか。脳の仕組みとして解説し、「あなたが悪いわけではない」と伝える。

4) **ブッダの超訳（不動心）**:
    - 「不動心」を「動じないこと」ではなく、「揺れても戻ること」と定義する。

5) **史実・権威のエピソード**:
    - 困難を乗り越えた人物の、人間臭いエピソード。

6) **実践（3ステップ）**:
    - 心を守るバリアの張り方。言葉の変換技術。

7) **落とし穴と回避**:
    - 感情を押し殺してはいけない。泣いてもいいと許す。

8) **締め（信頼と応援）**:
    - 「あなたの心は美しいままでいい」と肯定する。
    - 最後は「皆様の心が穏やかでありますように」で締める。

トーン/言葉遣い:
- です・ます調。
- 優しく包み込むような、保護者のようなトーン。
- **「」と（）は全廃する**。
- **平易で易しい日本語**。

出力ルール:
- 台本本文のみ出力。
- 文字数は必ず **6,000字以上** を守る。

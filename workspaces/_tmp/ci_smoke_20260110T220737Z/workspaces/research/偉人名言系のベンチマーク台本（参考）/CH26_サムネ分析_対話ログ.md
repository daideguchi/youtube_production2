# CH26 サムネ分析（対話ログ / 推論結果の保管庫）

目的:
- サムネの勝ちパターン分析・微調整の **推論はこのチャットで行う**（LLM APIは使わない）
- 決まったことを **迷わず保存**し、次回以降に再現・再利用できる状態にする

保存先（SoT）:
- 動画単位の差分: `workspaces/thumbnails/assets/CH26/<NNN>/thumb_spec.json`
- チャンネル既定: `workspaces/thumbnails/templates.json`（必要なら昇格）

UI（編集導線）:
- `/thumbnails` → CH26 → 「調整」
  - 背景: 枠内ドラッグで移動 / ホイールでズーム
  - 肖像: 枠内ドラッグで移動 / ホイールでズーム
  - スライダー/色: 明るさ・彩度・文字サイズ・文字色など
  - 「保存して再生成」→ `thumb_spec.json` に保存される

## コメント→パラメータ（変換メモ）
※ ここは “翻訳表” だけ。判断はチャットで行い、数値は必要最小限で詰める。

### 肖像（人物）
- 右へ/左へ: `overrides.portrait.offset_x`（+で右 / -で左）
- 下へ/上へ: `overrides.portrait.offset_y`（+で下 / -で上）
- 大きく/小さく: `overrides.portrait.zoom`（大きいほどアップ）
- 明るく/暗く: `overrides.portrait.fg_brightness`
- コントラスト: `overrides.portrait.fg_contrast`
- 彩度: `overrides.portrait.fg_color`

### 背景
- 右へ/左へ: `overrides.bg_pan_zoom.pan_x`（-で背景が右へ＝被写体が右寄り / +で左寄り）
- 下へ/上へ: `overrides.bg_pan_zoom.pan_y`（-で背景が下へ＝被写体が下寄り / +で上寄り）
- ズーム: `overrides.bg_pan_zoom.zoom`（1.0以上）
- 明るさ/コントラスト/彩度/ガンマ: `overrides.bg_enhance.*`

### 文字
- 大きく/小さく: `overrides.text_scale`
- 色（塗り）: `overrides.text_fills.<fill_key>.color`（例: `red_fill` / `yellow_fill` / `white_fill`）

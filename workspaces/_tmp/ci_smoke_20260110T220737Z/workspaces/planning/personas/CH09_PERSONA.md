# CH09 Persona & Planning Template

## 0. テンプレ SoT
- SoT: `workspaces/planning/templates/CH09_planning_template.csv`
- 使い方: ヘッダー＋サンプル1行をコピーし、`{NEXT_NO}` / `{NEXT_VIDEO}` を差し替えて `workspaces/planning/channels/CH09.csv` に追加 → UI `/planning` で表示確認（外部編集した場合は「企画を再読込」）。

## 1. 共通ペルソナ（固定一文）
> 30〜60代。夜に焦りや取り残され感を抱え、偉人の言葉を「本人の語り」として聴きながら、揺れない指針を手に入れたい人。

- 視聴スタイル: 寝る前、通勤、作業中。耳だけで聴く。
- 台本の芯: 偉人本人の一人称で語りかける→比喩を1つだけ固定→3〜5つの指針を同じ順序で反復→最後は今日/明日の一歩で閉じる。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
| --- | --- | --- |
| 悩みタグ_メイン | 主題 | 焦り / 取り残され / 決断疲れ / 自己否定 |
| 悩みタグ_サブ | 補助 | 偉人 / 名言 / 物語 / 指針 |
| ライフシーン | 場面 | 夜寝る前 / 通勤中 / 深夜の書斎 |
| キーコンセプト | 視点 | 根を伸ばす / 静かな勇気 / 不動の指針 |
| ベネフィット一言 | 効果 | 「焦りが静まり、明日ひとつ進める」 |
| たとえ話イメージ | サムネ/本文 | 雪の下の花 / 羅針盤 / 山の尾根道 |
| 説明文_リード | 冒頭 | 「今夜は、偉人が君たちに語りかける。」 |
| 説明文_この動画でわかること | 箇条書き | ・名言の背景と物語<br>・今日からできる一歩 |

## 3. 記入ルール
1. タイトルは「【偉人名】＋悩みフック」で 28〜36 文字を目安（テーマは1つに固定）。
2. 構成案は `導入（夜の焦り）→偉人の原体験→指針（3〜5）→今日の一歩→余韻` を基本に記す。
3. サムネ: 暗め背景＋白/赤文字。人物（偉人）シルエットで「誰の回か」を1秒で分かるようにする。

## 4. ベンチマーク
- 参考: 偉人のコトバ／松下幸之助の教え（research フォルダ参照）
- 学ぶ: 一人称の没入、反復のテンポ、比喩の固定と回収（ただし断定/煽り/CTAは除去）

## 5. 運用
1. 企画追加後、UI `/planning` の「企画を再読込」（外部編集時）→ 行が表示されることを確認する。
2. `python3 scripts/ops/planning_lint.py --channel CH09 --write-latest` で列ズレ/汚染/タグ矛盾をチェックする。

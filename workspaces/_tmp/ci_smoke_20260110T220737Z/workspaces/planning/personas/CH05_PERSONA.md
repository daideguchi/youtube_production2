## 1. 共通ペルソナ
> 50〜70代で、家族役割や仕事を一段落させたあとも心が若く、もう一度恋やパートナーシップを楽しみたいと願うシニア。

- `ターゲット層` はこの文で固定。
- データや統計の挿入は不要。物語と感覚で寄り添う前提とし、数値や一般論は書かない。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
|---|---|---|
| 悩みタグ_メイン | 主題 | 再婚 / 初恋 / 夫婦の距離 / 孤独 / 介護と恋 / オンライン恋愛 |
| 悩みタグ_サブ | 補助 | 子どもの目線 / 経済不安 / 健康と恋 |
| ライフシーン | 場面 | 市民大学 / サークル / 通院帰り / 旅先 / マッチングアプリ |
| キーコンセプト | 視点 | 慈悲 / 縁起 / アタッチメント / レジリエンス |
| ベネフィット一言 | 効果 | 「第二の恋に踏み出す勇気」「夫婦の距離を整える言葉がわかる」 |
| たとえ話イメージ | サムネ/本文 | 手紙を開く / 夕焼けのバス停 / 古いレコード |
| 説明文_リード | 冒頭 | 「大人の恋をもう一度楽しむための小さな勇気を届けます」 |
| 説明文_この動画でわかること | 箇条書き | ・第二の恋に踏み出すステップ\n・家族へ伝える言葉 |

## 3. 記入ルール
1. タイトルは「【年齢・関係】×【ドラマ性】」。ストーリー要素を入れる。 
2. 構成案は `【導入】→【主人公紹介】→【葛藤】→【キーコンセプトの視点】→【実践】→【余韻】`。
3. サムネは人物の表情を大きく、コピーは12〜14文字で感情を直球に伝える。

## 4. テンプレ
- `workspaces/planning/templates/CH05_planning_template.csv` をコピーし、キャラクター設定や Analytics 根拠を書き換える。
- タグ列・説明文列で「誰の、どんな悩みを解決するのか」を明文化する。

## 5. 運用
1. 企画追加後、UI `/planning` の「企画を再読込」（外部編集時）→ 行が表示されることを確認する。
2. `python3 scripts/ops/planning_lint.py --channel CH05 --write-latest` で列ズレ/汚染/タグ矛盾をチェックする。
3. UI でタグ列・説明文が表示されることを確認。必要ならサムネ指示も追記する。

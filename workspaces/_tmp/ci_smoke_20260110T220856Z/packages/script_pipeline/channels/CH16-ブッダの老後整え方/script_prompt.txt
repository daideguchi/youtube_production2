# CH16 台本プロンプト（ブッダの老後整え方）

役割: あなたは「ブッダの老後整え方」の語り手。老いへの不安を感じる60代以降の視聴者に対し、年を重ねることの豊かさと美しさを語り、安心感を与える一人語り台本を書く。

ゴール: **7,000〜9,000字** の長尺一人語り。
「若作り」や「アンチエイジング」を強要しない。ありのままの老いを肯定し、その楽しみ方を提案する。

構成（8パート固定。箇条書き・番号リストは禁止。ポーズは `---` のみ許可）:

1) **導入フック（共感と受容）**:
    - 「鏡を見て溜息をつくことはありませんか」と寄り添う。
    - 老いを否定する世間の風潮から、視聴者を守る。

2) **ブッダの逸話**:
    - 生老病死にまつわる物語。

3) **問題の正体（価値観の歪み）**:
    - 若さだけが良いことではない。「枯れる」ことの美学を説明。

4) **ブッダの超訳（完熟）**:
    - 老いを「喪失」ではなく「完成へ向かうプロセス」と再定義。

5) **史実・権威のエピソード**:
    - 晩年を素敵に生きた先人のエピソード。

6) **実践（3ステップ）**:
    - 丁寧な暮らし、趣味、身支度など、心を豊かにする習慣。

7) **落とし穴と回避**:
    - 過去の栄光と比較しない。今の自分を愛でる。

8) **締め（黄金期への招待）**:
    - 「これからの時間が一番贅沢です」と伝える。
    - 最後は「皆様の毎日が輝きに満ちていますように」で締める。

トーン/言葉遣い:
- です・ます調。
- 上品で落ち着いた、ホテルのコンシェルジュのようなトーン。
- **「」と（）は全廃する**。
- **平易で易しい日本語**。

出力ルール:
- 台本本文のみ出力。
- 文字数は必ず **6,000字以上** を守る。

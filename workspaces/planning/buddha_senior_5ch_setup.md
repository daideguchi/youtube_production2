## 1. 5ch共通の核（視聴者への約束）

現代の悩みをブッダの教えで「わかる言葉」に翻訳し、今日から心が軽くなる一手を渡す。  
難しい用語を使わず、結論と理由と具体例で腹落ちさせる。視聴後に残るのは安心と行動の一歩。

共通の禁止（事故防止）:
- 煽りすぎない / 説教しない / スピ系に振り切らない / 宗派色を出しすぎない
- 難語・専門語・断定過多・脅し文句を避ける

## 3. 台本アウトライン（共通・固定）

1. 導入フック（10〜20秒）共感の一撃 + 結論予告
2. 日常の具体シーン（1〜2分）シニアの生活で刺す
3. 問題の正体（3〜6分）苦しみの原因を一つに絞る
4. ブッダの見立て（6〜12分）教えを現代語に翻訳
5. 史実エピソード（8〜15分）人物と出来事で腹落ち
6. 実践（3ステップ）（3〜6分）今日できる形に落とす
7. 落とし穴と回避（2〜4分）誤解を潰す
8. 締め（30〜60秒）短い言葉で背中を押して静かに終える

## 4. タイトル/サムネ（共通・固定）

タイトル（型）:
- 先頭タグ: `【悩み】` 固定
- 結論先出し + 意外性 + 具体 + 損失回避 + 禁句（カリギュラ）を混ぜる

サムネ（3段キャッチ）:
- 上: 禁句/危険/逆効果/損 を短く断言
- 中: 名指し（60代以降の状況が浮かぶ言葉）
- 下: 即ベネフィット（言い切る）

共通語彙（回して刺さり減衰を防ぐ）:
- 上段: 放置は危険 / 絶対NG / やるほど逆効果 / そのまま損 / 今が分かれ道
- 中段: 夜の不安 / 定年後 / 介護疲れ / 同級生の今 / 家族の距離 / 年金不安
- 下段: 今夜眠れる / 静まる / 軽くなる / 守れる / 切れる / ほどける / 間に合う

## 1. 共通ペルソナ
> 30〜50代で知的好奇心が強く、心理学や認知科学から日常のモヤモヤを整理したいと思っている人。

- `ターゲット層` はこの文で固定。企画ごとに書き換えない。
- 記事・論文・一次資料へのリンクは `企画意図` に必ず記載する。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
|---|---|---|
| 悩みタグ_メイン | 主題 | 潜在意識 / 記憶 / 注意 / 認知バイアス / 感情 / 習慣 |
| 悩みタグ_サブ | 補助 | 仕事のひらめき / 人間観察 / 心理トリック |
| ライフシーン | 場面 | 通勤 / オフィスでの休憩 / 家事の合間 / 雨の日の帰り道 |
| キーコンセプト | 視点 | 注意の偏り / 認知バイアス / 記憶のクセ / ユング心理学 / 認知実験 |
| ベネフィット一言 | 効果 | 「日常の違和感を言語化できる」「新しい視点が手に入る」 |
| たとえ話イメージ | サムネ/本文 | フィルターが寄る / ラベル付け / メモで可視化 |
| 説明文_リード | 冒頭 | 「日常の小さな違和感を、心理学の視点でほどいていきます」 |
| 説明文_この動画でわかること | 箇条書き | ・XX実験の真相\n・今日から使える心理トリック |

## 3. 記入ルール
1. タイトルは「【心理】×【身近な疑問】」。煽りや断定は避ける。 
2. 構成案は `【導入】→【謎/実験紹介】→【考察】→【日常への翻訳】→【問いかけ】`。
3. サムネは暗めの背景＋象徴モチーフ。コピーは12〜14文字で問いを立てる。

## 4. テンプレ
- `workspaces/planning/templates/CH04_planning_template.csv` をコピーし、Analytics根拠や引用を差し替える。
- タグ列・説明文列は必ず記入し、ミステリー要素と学びの両方を明示する。

## 5. 運用
1. 企画追加後、UI `/planning` の「企画を再読込」（外部編集時）→ 行が表示されることを確認する。
2. `python3 scripts/ops/planning_lint.py --channel CH04 --write-latest` で列ズレ/汚染/タグ矛盾をチェックする。
3. UI でタグ列・説明文が表示されることを確認する。

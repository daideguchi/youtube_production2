# CH10 Persona & Planning Template

## 0. テンプレ SoT
- SoT: `progress/templates/CH10_planning_template.csv`
- 使い方: ヘッダー＋サンプル1行をコピーして `{NEXT_NO}` / `{NEXT_VIDEO}` を差し替え、`progress/channels/CH10.csv` へ追加 → `planning_store.refresh --force`。

## 1. 共通ペルソナ（固定一文）
> 30〜60代で、考えすぎ・将来不安・孤独感を抱え、寝る前に耳だけで聴ける「静かな教養ラジオ」を求めている人。

- 生活導線: 夜寝る前、通勤帰り、家事の合間にイヤホンで聴く。
- 期待値: 落ち着いたトーン、名言と現代の悩みをつなぐ解説、今日すぐ試せる小さな一歩。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
| --- | --- | --- |
| 悩みタグ_メイン | 主題 | 考えすぎ / 生きる意味 / 孤独感 / 力み / 決断できない |
| 悩みタグ_サブ | 補助 | 仕事ストレス / 情報疲れ / 自己肯定感 / 行動できない |
| ライフシーン | 場面 | 夜寝る前 / 休日の夜 / 通勤帰り / 静かな書斎 |
| キーコンセプト | 視点 | 考えない時間 / 無為自然 / 孤独の力 / 執着を緩める |
| ベネフィット一言 | 効果 | 「夜が少し静かになる」 |
| たとえ話イメージ | サムネ/本文 | 夜の図書館 / 三日月と本 / 静かな湖 |
| 説明文_リード | 冒頭 | 「眠れない夜に そっと開く 本のように。」 |
| 説明文_この動画でわかること | 箇条書き | ・偉人の言葉の背景<br>・今日から試せる一歩 |

## 3. 記入ルール
1. タイトルは偉人名＋悩みフックで 28〜34文字を目安。
2. 構成案は `導入→人物/時代→名言→現代の悩みへの橋渡し→今日の一歩` の5ブロックで記す。
3. サムネ: 深いネイビー×金。左に縦帯＋テキスト、右に偉人シルエット。文字は画面内に収める。

- 参考チャンネル: 前略、偉人より
- フック: 「夜に聴ける落ち着いたトーン」「名言と悩みを結ぶ導線」「今日の一歩」

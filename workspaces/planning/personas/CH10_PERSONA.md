# CH10 Persona & Planning Template

## 0. テンプレ SoT
- SoT: `workspaces/planning/templates/CH10_planning_template.csv`
- 使い方: ヘッダー＋サンプル1行をコピーして `{NEXT_NO}` / `{NEXT_VIDEO}` を差し替え、`workspaces/planning/channels/CH10.csv` へ追加 → UI `/planning` で表示確認（外部編集した場合は「企画を再読込」）。

## 1. 共通ペルソナ（固定一文）
> 30〜60代で、考えすぎ・将来不安・孤独感を抱え、偉人の思考を「今日から実装」できる形で静かに学びたい人。

- 生活導線: 夜のひと息、通勤帰り、家事の合間、朝の支度中にイヤホンで聴く。
- 期待値: 落ち着いたトーン、名言と現代の悩みをつなぐ解説、今日から実装できる小さな手順と一歩。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
| --- | --- | --- |
| 悩みタグ_メイン | 主題 | 考えすぎ / 生きる意味 / 孤独感 / 力み / 決断できない |
| 悩みタグ_サブ | 補助 | 仕事ストレス / 情報疲れ / 自己肯定感 / 行動できない |
| ライフシーン | 場面 | 夜寝る前 / 休日の夜 / 通勤帰り / 静かな書斎 |
| キーコンセプト | 視点 | 考えない時間 / 無為自然 / 孤独の力 / 執着を緩める |
| ベネフィット一言 | 効果 | 「夜が少し静かになる」 |
| たとえ話イメージ | サムネ/本文 | 夜の図書館 / 三日月と本 / 静かな湖 |
| 説明文_リード | 冒頭 | 「今の悩みを、偉人の思考でほどき、明日ひとつだけ実装する。」 |
| 説明文_この動画でわかること | 箇条書き | ・偉人の言葉の背景<br>・今日から試せる一歩 |

## 3. 記入ルール
1. タイトルは偉人名＋悩みフックで 28〜34文字を目安。
2. 構成案は `導入・予告→偉人の原体験（情景）→核心の思考→詳細ワーク→静かな締め・明日の一歩` の5ブロックで記す。導入は短く、偉人の物語と実装ワークにボリュームを割く。
3. サムネ: 深いネイビー×金。左に縦帯＋テキスト、右に偉人シルエット。文字は画面内に収める。

## 4. ベンチマーク
- 参考チャンネル: 人生は考え方【偉人から学ぶ】（研究フォルダ参照）
- フック: 「最初に痛みを掴む」「偉人の背景で腹落ち」「今日から実装できる形に落とす」

## 5. 運用
1. 企画追加後、UI `/planning` の「企画を再読込」（外部編集時）→ 行が表示されることを確認する。
2. `python3 scripts/ops/planning_lint.py --channel CH10 --write-latest` で列ズレ/汚染/タグ矛盾をチェックする。

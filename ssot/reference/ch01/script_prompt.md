# 人生の道標 台本執筆プロンプト

## チャンネルの軸
仏教の教えを現代の視聴者にわかりやすく伝える。抽象的な説法ではなく、**史実エピソード**を通じて教えが「真実である」ことを証明する構成。

---

## 構成テンプレート

### 1. 導入（共感フック）
- 視聴者の痛みに寄り添う問いかけ
- 「〜したことはありませんか」形式で心に刺さる導入
- チャンネル紹介＋今日のテーマ宣言

### 2. 仏教の教え（土台）
- テーマに関連するお釈迦さまの教えを紹介
- 抽象的に終わらせず「なぜこの教えが重要か」を説明
- 「これは二千五百年前の教えですが、歴史が証明してきました」で次へ繋ぐ

### 3. 史実エピソード（証明）
**1〜2つに絞り、深く掘り下げる**

- 人物の背景・状況を丁寧に描写
- その人が直面した葛藤・苦悩を具体的に
- どのような選択をし、どのような結果になったか
- 仏教の教えとの接点を明示（例：「これはまさに忍辱の実践でした」）

**エピソード選定基準：**
- 日本人視聴者に不快感を与えない
- 仏教の教えを「証明」できる内容
- 感情移入できる人間ドラマがある

### 4. 視聴者への橋渡し
- 「さて、私たちの日常に戻りましょう」
- 史実から学んだ教訓を現代の文脈に置き換え
- 仏教用語で心理を説明（例：渇愛、執着、忍辱など）

### 5. 実践（まとめ）
- 今日からできる具体的な行動を2〜3つ
- ここだけ箇条書きOK（まとめとして機能）
- 短く、実行可能なものに絞る

### 6. 締め
- テーマに関連する歴史上の人物を総括
- 視聴者への祝福・励まし
- 「本日も最後までご覧いただきありがとうございます」

## スタイルルール

### 禁止事項
- [ ] 箇条書きの連続（3つ以上の列挙を繰り返さない）
- [ ] 「一つ目は〜」「二つ目は〜」の羅列構造
- [ ] 括弧の多用
- [ ] 抽象的な説教だけで終わる
- [ ] 日本人の感情を逆撫でする史実（原爆関連など）

### 推奨事項
- [x] 史実エピソードは1〜2つに絞り、深く掘り下げる
- [x] 人物の心情・葛藤を丁寧に描写
- [x] 仏教用語を適切に織り交ぜ、教えとの接点を明示
- [x] 「ストーリー → 教訓 → ストーリー」の緩急
- [x] 読点（、）は4〜5%程度に抑える
- [x] 文字数：12,000バイト以上

## 史実エピソード例（使用可能）

### 日本史
- 徳川家康（忍辱、沈黙、待つ力）
- 大久保利通（寡黙、威厳）
- 江戸商家の秘伝（秘密保持、口の禍）
- 千利休（侘び寂び、執着を捨てる）
- 良寛（無欲、托鉢の精神）

### 世界史
- モルトケ「偉大な沈黙者」（沈黙で結果を出す）
- ブレッチリーパーク暗号解読者（秘密を守り続けた人々）
- カルビン・クーリッジ「寡黙なカル」（沈黙の政治家）
- イマヌエル・カント（10年の沈黙と模索）
- ガンジー（非暴力、忍耐）

### 仏教関連
- 玄奘三蔵（求道の旅）
- 空海（修行、真言の秘密）
- 親鸞（苦悩と救い）
- 一休宗純（常識を超えた悟り）

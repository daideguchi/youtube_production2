# CONTACT_BOX — 連絡ボックス（Gitで共有 / モバイル編集OK）

目的:
- 人間↔エージェントの連絡を **この1ファイルに集約**して、取りこぼし/誤解を減らす。
- 「いまの優先順位」「決定」「確認待ち」を常に最新に保つ（= pull/pushで同期）。

ルール:
- **Secrets禁止**（APIキー/トークン/個人情報は書かない）
- 追記は上に積む（新しいものが上）
- 1件 = 3〜8行で短く（長文は別SSOTに書いてリンク）
- 可能なら先頭に `YYYY-MM-DD` を付ける
- 「依頼/質問/決定/却下」を明示する

# CONTACT_BOX — 人間↔AIエージェント連絡箱（モバイル編集OK / Gitで共有）

目的:
- 人間がスマホから「確定ロジック/運用指示/優先順位」を追記し、**必ずmainへpush**して共有する
- AIエージェントがpullして読み、作業・返答・意思決定の取りこぼしをゼロにする

重要:
- **APIキー/トークン等の秘匿情報は絶対に書かない**
- ここは「指示と要件の合意」だけ（ログや長文は `workspaces/logs/**` へ）
- 追記は上に積む（最新が上）

記入テンプレ:
```
- YYYY-MM-DD HH:MM JST | from=<name> | prio=P0/P1/P2
  - 指示/質問:
  - 期待する完了条件:
  - 期限/備考:
```

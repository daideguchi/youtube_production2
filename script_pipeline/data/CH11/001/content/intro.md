# 導入

こんにちは。「人生の道標」をご覧いただきありがとうございます。60歳を過ぎると、若いころのように気力で全てを抱えることが難しくなり、思い出の品や人付き合いまで荷物のように感じてしまう瞬間が増えますよね。家の中を見渡すと、喜びも後悔も詰まったモノが棚いっぱいに眠り、スマホを開けば、距離の取り方に迷う連絡が届く。どこから手を付ければいいのか分からず、心だけが疲れていく――そんな声を本当にたくさん耳にします。

今日の動画では、仏教の知恵と最新の心理学を組み合わせ、モノ・心・人間関係を静かに整える「60歳からの人間断捨離」を一緒に進めていきます。ただ捨てるのではなく、「ありがとう」を添えて送り出す方法を探りながら、新しいご縁を迎えるスペースをつくる。わたしと対話するつもりで、肩の力を抜いてお聞きください。紙とペンを用意して、メモを取りながら進めていただけると、より自分ごととして捉えられるはずです。

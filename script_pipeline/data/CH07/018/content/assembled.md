# 【話しすぎ注意】言葉が多いと不幸になる（CH07-018）

深夜の静けさに灯りを落とし、この法話をお聴きください。言葉が多すぎると、心は乾き、相手も自分も疲れます。黙ることは逃げではなく、布を敷くこと。ブッダはこうおっしゃいました。「言多ければ、心散ず。静けさに道あり」。おっしゃる通り、静けさは道。今日は、話しすぎの刃を布に包み、言葉を水に変える作法をお話しします。

朝、目覚めたらカーテンを指幅だけ開け、四つ吸い、六つ吐く。吐くたびに心で唱える。「今日は少なく、深く」。白湯を一口ゆっくり飲み、喉を温め、胸に手を置く。「この鼓動は、言葉よりも確か」。ブッダはこう説かれました。「己が命、己が宝」。おっしゃる通り、宝は胸の内にあり、言葉の数ではありません。

玄関の敷居を拭き、短く礼をする。敷居は結界。結界を意識すると、外に出す言葉と家に入れる言葉を選べます。ポケットに小さな紙を入れ、三つの言葉を書きます。「止」「間」「布」。止める言葉、間を置く言葉、布を敷く言葉。この三つが、言葉を少なく深くする柱になります。

話しすぎそうなときの最初の一言は短く。「ここまでにします」。必要なら「少し静かにします」と添える。ブッダははっきりとこう言われました。「無益なる言を捨てよ」。おっしゃる通り、無益な言葉を捨てる宣言が布を敷きます。怒鳴らず、低く、短く。沈黙は布。布に棘が刺さって止まります。

朝の準備にもう一つ。鏡の前で口角を少し上げ、唇を閉じて低くハミングする。鼻の奥が震える感覚を掴んだら、ハミングの余韻に心で重ねる。「止める」「間を取る」「布を敷く」。声帯が緩むと、言葉の角が丸くなり、静かな強さが出ます。

午前、会話が熱を帯びたら、手を止め、四つ吸い八つ吐き、紙の「止・間・布」に指を触れる。どの言葉を今使うか選ぶ。選ぶ時間が、言葉の洪水を防ぎます。返答は短く。「今は聞きます」「一度止めます」。沈黙は布。布を敷くと、相手の棘も止まります。

昼休み、外の風に頬を撫でさせ、胸に手を置き、四つ吸い六つ吐く。心で唱える。「水のように」。弁当をゆっくり噛み、味と香りを確かめる。五感が揃うと、言葉より味や風の温度が前に出て、心が今に戻ります。ブッダはこうおっしゃいました。「現在を住処とせよ」。おっしゃる通り、今に住むことで、余計な言葉が減ります。

午後、伝えたいことが多すぎると感じたら、画面を伏せ、足裏を床に押し、顎を引いて後頭部を天井に向け、四つ吸い八つ吐く。吐くたびに心で唱える。「止める」「間を置く」「布を敷く」。必要なら短く「続きは後で」と伝える。布があると、言葉の火が広がりません。

夕方、帰り道で空の色が変わるのを眺める。色が移ろうように、言葉の必要も移ろう。足音に合わせて「止」「間」「布」を順に唱える。帰宅したら玄関で礼をし、「今日はここまで」と区切る。区切りがあると、外の喧騒を家に持ち込まない習慣ができます。

夜の食卓では、箸を置き、四つ吸い六つ吐き、「聞きます」と心で唱える。家族や同居人の話に耳を傾け、途中で言葉が溢れそうになったら、箸を置き、沈黙の布を敷く。必要なら短く「もう少し聞かせて」と伝える。ブッダはこう説かれました。「境を敷け」。おっしゃる通り、境界が互いを護り、言葉の暴走を止めます。

風呂では湯気を眺め、今日溢れそうになった言葉を湯気に乗せ、天井で消えるのを見届ける。湯気は立ち上り、消える。ブッダはこうおっしゃいました。「起こりしものは、やがて滅す」。おっしゃる通り、言葉も湯気のよう。消える瞬間を見ることで、明日の口数が自然に減ります。

布団に入る前にノートを開き、三行だけ書く。今日、止められた言葉。間を置けた場面。布を敷けた瞬間。三行の横に小さな丸を描き、「止」「間」「布」と書く。ページを閉じ、胸に手を置き、「ここまで」と唱える。ページを閉じる音が区切りになり、思考が静まります。

七日間の練習を組みましょう。初日は紙に「止・間・布」を書いて持ち歩く。二日目は「ここまでにします」を一度だけ使う。三日目は湯気に言葉を乗せて消えるのを見届ける。四日目は敷居を拭き、礼をする。五日目はノートに三行を書く。六日目は沈黙の布を意識する。七日目は全部を通す。七日で、言葉の洪水が少し遠のき、呼吸が深くなります。二十一日続ければ、境界の線が太くなり、沈黙への罪悪感が薄まる。三十日を超えると、静かな返しが反射になり、夜の寝つきが深くなる。九十日続ければ、言葉を選ぶ軸が太くなり、少ない言葉で伝わるようになります。

問いを三つ持ちましょう。これを今言わなければ命や生活は守られないか。これを今言えば誰かを傷つけるか。これを言わなくても明日も歩けるか。三つの問いに「いいえ」が続いたら、一度止める。問いがあると、衝動が薄れ、判断が澄みます。ブッダはこうおっしゃいました。「智者は問う。問うは道を敷く」。おっしゃる通り、問いが道になります。

港の比喩を思い出してください。心は港、思考は船、言葉は荷物。港に入れる荷物は選べます。必要な荷だけ受け入れ、過剰な荷は桟橋に置き、潮に任せる。桟橋に座り、波音を聞くように呼吸の音を聞く。波は寄せては返す。言葉も寄せては返す。返っていくものに名前を付けず、ただ見送る。これが「観」です。

六波羅蜜の舟を使いましょう。布施は「沈黙の時間を贈る」布施。持戒は「ここまで」と線を敷く戒。忍辱は「三呼吸を置いて熱を冷ます」忍。精進は「少ない言葉を選ぶ稽古を続ける」精進。禅定は「湯気や数息観で静める」定。智慧は「言うべきときと言わぬべきときを分ける」慧。六つの舟を紙に書き、今日使う舟に丸を付ける。ブッダはこう説かれました。「六の舟を用いよ」。おっしゃる通り、舟があると言葉の川も渡れます。

数息観を短く。朝は一から十まで、吐く息ごとに数え、十に至ったら一に戻る。夜は一から五で止める。数を刻むと、頭の渦が静まり、言葉の洪水が収まります。

ハミングの稽古を。唇を閉じ、低くハミングし、鼻の奥が震える感覚を掴む。ハミングの余韻に「短く」「静かに」を重ねる。喉が緩むと、声が低くなり、棘のない返しが生まれます。ブッダはこうおっしゃいました。「声は心の鏡」。おっしゃる通り、声を整えると心も整います。

香りと光を味方に。朝は好きな香りをひと吹きし、「今日は少なく深く」と心で添える。昼は薄いお茶の香りを吸い、「ここまで」と区切る。夜は灯りを落とし、間接照明の下で「ここまで」と一言。嗅覚は大脳辺縁系に直結し、安心の回路を開きます。光は体内時計を整え、境界のリズムを整えます。ブッダはこう説かれました。「環境を整えよ、心は環境に従う」。おっしゃる通り、場を整えると言葉も整います。

身体の所作を足しましょう。首を左右にゆっくり倒し、肩を後ろに大きく回す。回すたびに「短く」「静かに」を心で重ねる。足裏を床に押し、足指を開いて閉じる。「私はここにいる」と唱える。身体が緩むと、言葉の角も丸くなります。ブッダはこうおっしゃいました。「身を整うれば、心もまた整う」。おっしゃる通り、身体が道を開きます。

小さな白湯の儀式を。朝の一杯を飲む前に湯気を眺め、「起こりしものは、やがて滅す」と心で唱える。湯気が消える瞬間を見ると、余計な言葉も消えることを体が思い出します。

沈黙の布の応用。会話の途中で心が乱れたら、耳たぶをそっと撫で、三拍の沈黙を置く。その間に足裏を床に押し、「今は聞く」と心で唱える。沈黙は拒絶ではなく、防波堤です。

感謝の灯を一つ。静かに「ありがとう」を心で唱えるだけで、舌の力が抜け、言葉が短くなります。ブッダはこうおっしゃいました。「柔らかき言は、心を潤す」。おっしゃる通り、柔らかな言葉が布になります。

小さなハミングを朝に。唇を閉じ、低くハミングし、鼻の奥が震える感覚を掴む。ハミングの余韻に「止」「間」「布」を重ねる。声帯が緩むと、心の鏡も澄みます。

夕暮れの港の比喩をもう一つ。桟橋に小石を置き、指で触れながら「ここまで」と唱える。小石は灯台。灯台があると、言葉の波に呑まれずに済みます。

小さな読経を夜に。枕元で合掌し、「諸行無常」「諸法無我」を一度だけ唱える。声が震えても構いません。響きが胸を揺らし、言葉の棘を丸くします。読経が難しい日は、「止」を一言だけ。声に出すことで、思考の渦がほどけます。

事例をいくつか。三十代の看護師は、患者家族の長い愚痴に疲れていました。ポケットに「止・間・布」を入れ、「ここまでにします」を一度だけ使い、以降は沈黙の布を敷く練習を三週間続けたところ、肩の重さが減り、夜の眠りが深くなりました。四十代の保育士は、会議で話しすぎて後悔していました。会議前に半歩下がる間合いを意識し、発言前に三呼吸置く習慣を続け、二十一日で発言が短く整い、胃の痛みが薄れました。五十代の営業は、説明過多で商談を逃していました。「今日はここまで」と区切りを作り、メールを短くする練習を続け、一ヶ月で成約率が上がり、頭痛が減りました。六十代の主婦は、家族に愚痴をこぼしすぎて疲れていました。敷居を拭いて礼をし、食卓で沈黙の布を意識したところ、家の空気が柔らかくなりました。七十代の祖父は、集まりで話しすぎて後悔していました。湯気に言葉を乗せる儀式と、枕元のノートの三行を続け、二週間で夜の胸騒ぎが減りました。

さらに二例。八十代の元教師は、昔話が止まらず周囲を疲れさせていました。朝に「止」の紙に触れ、話し始める前に三呼吸置く習慣を三週間続けたところ、話が短くまとまり、聞き手の顔が和らぎました。四十代のシングルマザーは、子に不安をぶつけがちでした。「ここまで」と掌を合わせる合図を家で共有し、夜は湯気を眺める儀式を続け、一ヶ月で子との会話が穏やかになりました。五十代の介護者は、利用者家族の長い要望に疲弊していました。メールを送る前に三呼吸置き、「止・間・布」に触れてから返す習慣を三週間続けたところ、頭痛が減り、返答が短く整いました。

科学の視点も添えます。短い断りや区切りの言葉を使うと、前頭前野が働き、扁桃体の過剰反応が落ち着きます。長く吐く呼吸は迷走神経を刺激し、副交感神経を優位にし、心拍と血圧を下げます。湯気や丸のイメージは視覚野を介して感情の強度を和らげます。触覚（紙、小石、敷居の布）を使うと、身体感覚が現在に戻り、反芻の連鎖を断ちます。書くことは前頭前野を働かせ、胸の荷を外に置く効果があります。これらの所作が、静かな強さを育てる土台になります。

安全の確認は最優先です。暴力、脅し、搾取など命や生活を脅かす状況では、「それはできません」「ここまで」の前に距離を取り、専門機関に相談する。ブッダはこう説かれました。「己を護らずして、何を護らん」。おっしゃる通り、身の安全があってこそ、境界も慈悲も働きます。

はっきりとこう言われました。「己を尊べ」。この一言を、たまに胸に置いてください。自分を尊ぶことが、少ない言葉に芯を通します。

節目に小さなご褒美を。七日目に香りをひと吹きし「ここまで続いた」と自分に伝える。二十一日目に紙を新しく書き直し、古い紙を折って捨てるとき「ここからも続く」と添える。三十日目に湯気を眺める時間を一分伸ばし、温かい茶を丁寧に淹れる。九十日目に小石を土に返し、新しい小石を握り、「軸は太くなった」と心で言う。小さな印が、続ける力を支えます。

最後に、静かな祈りを。灯りを落とし、肩を落とし、足裏に重さを乗せ、胸に手を置き、静かに言う。「止め、間を置き、布を敷き、静かに立つ。私はここにいる」。この一息が、言葉の洪水を布に包み、静かな夜へと導きます。おやすみなさい。合掌。

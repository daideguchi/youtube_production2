# 【偶然じゃない】嫌いな人との出会いに意味がある（CH07-021）

深夜の静けさに灯りを落とし、この法話をお聴きください。嫌いな人との出会いは、ただの不運ではなく、心を映す鏡になることがあります。苛立ち、怖さ、悔しさ。それらは自分の偏りや傷を映す影でもあります。ブッダはこうおっしゃいました。「縁は鏡、己を映す」。おっしゃる通り、嫌悪は鏡。鏡に映ったものをどう扱うかで、心は軽くも重くもなります。今日は、嫌いな人との出会いを、燃やす火種ではなく、灯に変える道をお話しします。

朝、目覚めたらカーテンを指幅だけ開け、四つ吸い、六つ吐く。吐くたびに心で唱える。「これは縁、私はここにいる」。枕元の白湯を一口ゆっくり飲み、喉を温め、胸に手を置く。「この鼓動は私のもの」。ブッダはこう説かれました。「己が命、己が宝」。おっしゃる通り、宝は胸の内。縁も胸で扱います。

玄関の敷居を拭き、短く礼をする。敷居は結界。結界を意識すると、外から持ち込む棘を玄関で落とせます。ポケットに小さな紙を入れ、三つの言葉を書きます。「観」「距」「灯」。観る言葉、距離を取る言葉、灯に変える言葉。この三つが、嫌いな人との縁を扱う柱になります。

最初の返しは短く。「その言い方は受け取りません」。必要なら「距離を置きます」と静かに添える。ブッダははっきりとこう言われました。「無益なる言を捨てよ」。おっしゃる通り、無益な棘を拾わず、捨てる宣言が境界を敷きます。怒鳴らず、低く、短く。沈黙は布。布に棘が刺さって止まります。

午前、苦手な相手を思い出したら、手を止め、四つ吸い八つ吐き、紙の「観・距・灯」に指を触れる。観は事実を淡々と見ること。距は物理的・時間的に離れること。灯は出会いの意味を静かに探ること。このうち今必要なものを一つ選び、心で唱える。選ぶ時間が、反射的な怒りを遅らせます。

昼休み、外の風に頬を撫でさせ、胸に手を置き、四つ吸い六つ吐く。心で唱える。「私はここにいる」。弁当をゆっくり噛み、味と香りを確かめる。五感が揃うと、相手の顔より目の前の味が強くなり、心が今に戻ります。ブッダはこうおっしゃいました。「現在を住処とせよ」。おっしゃる通り、今に住むことで、嫌悪の影が薄まります。

午後、実際に嫌いな人と向き合う場面が来たら、画面を伏せ、足裏を床に押し、顎を引いて後頭部を天井に向け、四つ吸い八つ吐く。吐くたびに心で唱える。「観る」「距離」「灯す」。必要なら短く伝える。「その言い方は受け取りません」「ここまでです」。それ以上は言わず、沈黙の布を敷く。

夕方、帰り道で空の色が変わるのを眺める。色が移ろうように、縁の意味も変わります。足音に合わせて「観る」「距離」「灯す」を順に唱える。帰宅したら玄関で礼をし、「今日はここまで」と区切る。区切りがあると、外の棘を家に持ち込まない習慣ができます。

夜の食卓では、箸を置き、四つ吸い六つ吐き、「私はここにいる」と心で唱える。家族や同居人との会話で嫌いな人の話が出ても、箸を置いて沈黙の布を敷き、必要なら短く「その話は受け取れません」と伝える。ブッダはこう説かれました。「境を敷け」。おっしゃる通り、境界が互いを護ります。

風呂では湯気を眺め、今日浮かんだ嫌悪の感情や相手の顔を湯気に乗せ、天井で消えるのを見届ける。湯気は立ち上り、消える。ブッダはこうおっしゃいました。「起こりしものは、やがて滅す」。おっしゃる通り、嫌悪も湯気のよう。消える瞬間を見ることで、胸の熱が少し下がります。

布団に入る前にノートを開き、三行だけ書く。今日、観られた事実。距離を置けた場面。灯に変えられた気づき。三行の横に小さな丸を描き、「観」「距」「灯」と書く。ページを閉じ、胸に手を置き、「ここまで」と唱える。ページを閉じる音が区切りになり、思考が静まります。

七日間の練習を組みましょう。初日は紙に「観・距・灯」を書いて持ち歩く。二日目は「その言い方は受け取りません」を一度だけ使う。三日目は湯気に嫌悪を乗せて消えるのを見届ける。四日目は敷居を拭き、礼をする。五日目はノートに三行を書く。六日目は沈黙の布を意識する。七日目は全部を通す。七日で、嫌悪の衝動が少し遠のき、呼吸が深くなります。二十一日続ければ、境界の線が太くなり、罪悪感が薄まる。三十日を超えると、嫌いな人を学びに変える癖が反射になり、夜の寝つきが深くなる。九十日続ければ、縁を選ぶ軸が太くなり、偶然に見える出会いに静かな意味が見えてきます。

問いを三つ持ちましょう。「この人は何を映しているのか」「何を学べと言っているのか」「どこまで関わり、どこから離れるのか」。三つの問いに答えを書き出すだけで、感情と事実が分かれます。ブッダはこうおっしゃいました。「智者は問う。問うは道を敷く」。おっしゃる通り、問いが道になります。

六波羅蜜の舟を使いましょう。布施は「反応する時間を渡さない」布施。持戒は「ここまで」と線を敷く戒。忍辱は「三呼吸を置いて怒りを冷ます」忍。精進は「この稽古を続ける」精進。禅定は「湯気や数息観で静める」定。智慧は「相手の業と自分の業を切り分ける」慧。六つの舟を紙に書き、今日使う舟に丸を付ける。ブッダはこう説かれました。「六の舟を用いよ」。おっしゃる通り、舟があると嫌悪の川も渡れます。

身体の所作を足しましょう。首を左右にゆっくり倒し、肩を後ろに大きく回す。回すたびに「観る」「離れる」「灯す」を心で重ねる。足裏を床に押し、足指を開いて閉じる。「私はここにいる」と唱える。身体が緩むと、言葉の角も丸くなります。ブッダはこうおっしゃいました。「身を整うれば、心もまた整う」。おっしゃる通り、身体が道を開きます。

香りと光を味方に。朝は好きな香りをひと吹きし、「今日は縁を選ぶ」と心で添える。夜は灯りを落とし、間接照明の下で「ここまで」と小声で一度だけ言う。嗅覚は大脳辺縁系に直結し、安心の回路を開きます。光は体内時計を整え、境界のリズムを整えます。ブッダはこう説かれました。「環境を整えよ、心は環境に従う」。おっしゃる通り、場を整えると決断も整います。

小さな白湯の儀式を。朝の一杯を飲む前に湯気を眺め、「起こりしものは、やがて滅す」と心で唱える。湯気が消える瞬間を見ると、嫌悪も消える瞬間があると体が知ります。

沈黙の布の応用。会話の途中で心が乱れたら、耳たぶをそっと撫で、三拍の沈黙を置く。その間に足裏を床に押し、「今は聞く」と心で唱える。沈黙は拒絶ではなく、防波堤です。

小石の灯台を置きましょう。枕元に小石を置き、夜に指で触れ、「ここまで」と唱える。小石は灯台。灯台があると、言葉の波に呑まれずに済みます。

小さな読経を夜に。枕元で合掌し、「諸行無常」「諸法無我」を一度だけ唱える。声が震えても構いません。響きが胸を揺らし、嫌悪の棘を丸くします。読経が難しい日は、「観」を一言だけ。声に出すことで、思考の渦がほどけます。

感謝の灯を一つ。嫌いな人が映してくれた自分の偏りに気づけたら、心で短く「ありがとう」。口に出さなくても結構です。感謝の灯は、怒りの火を柔らかくします。

事例をいくつか。三十代の看護師は、患者家族の無礼に傷ついていました。ポケットに「観・距・灯」を入れ、「その言い方は受け取りません」を一度だけ使い、以降は沈黙の布を敷く練習を三週間続けたところ、肩の重さが減り、夜の眠りが深くなりました。四十代の保育士は、保護者の干渉に苛立っていました。「おっしゃる通り」と一度受け、「でも、ここまでにします」と返す練習を続け、二十一日で胃の痛みが薄れました。五十代の営業は、取引先の上から目線に疲れていました。「今日はここまで」と区切りを作り、メールを閉じる前に「観・距・灯」に触れる習慣を続け、一ヶ月で頭痛が減り、決断が速くなりました。六十代の主婦は、親族の嫌味に心を痛めていました。敷居を拭いて礼をし、食卓で「その言い方は受け取りません」を一度だけ伝えたところ、家の空気が柔らかくなりました。七十代の祖父は、地域の集まりでの敵意に沈んでいました。湯気に言葉を乗せる儀式と、枕元のノートの三行を続け、二週間で夜の胸騒ぎが減りました。

さらに二例。八十代の元教師は、若い同僚の上から目線に心を曇らせていました。朝に「観」の紙に触れ、会議前に半歩下がる間合いを意識し、「その言い方は受け取りません」を一度だけ使ったところ、声が震えずに返せるようになりました。四十代のシングルマザーは、元配偶者からの要求に疲弊していました。「距離を置きます」を短く伝え、シフトを調整して物理的距離を作り、夜は湯気に言葉を乗せる習慣を続け、一ヶ月で夜の動悸が減りました。

科学の視点も添えます。短い断りや区切りの言葉を使うと、前頭前野が働き、扁桃体の過剰反応が落ち着きます。長く吐く呼吸は迷走神経を刺激し、副交感神経を優位にし、心拍と血圧を下げます。湯気や丸のイメージは視覚野を介して感情の強度を和らげます。触覚（紙、小石、敷居の布）を使うと、身体感覚が現在に戻り、反芻の連鎖を断ちます。書くことは前頭前野を働かせ、胸の荷を外に置く効果があります。これらの所作が、嫌いな人との縁を灯に変える静かな土台になります。

安全の確認は最優先です。暴力、脅し、搾取など命や生活を脅かす状況では、「その言い方は受け取れません」の前に距離を取り、専門機関に相談する。ブッダはこう説かれました。「己を護らずして、何を護らん」。おっしゃる通り、身の安全があってこそ、境界も慈悲も働きます。

はっきりとこう言われました。「己を尊べ」。この一言をたまに胸に置いてください。自分を尊ぶことが、嫌いな人との縁を灯に変える芯になります。

最後に、静かな祈りを。灯りを落とし、肩を落とし、足裏に重さを乗せ、胸に手を置き、静かに言う。「嫌いという縁を観て、距離を選び、灯に変える。私はここにいる」。この一息が、偶然に見える出会いを静かな学びに変え、夜を穏やかにします。おやすみなさい。合掌。

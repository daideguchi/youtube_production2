# 優しい人の口癖とその深い意味（CH07-007）

夜の静けさにラジオを合わせるように、この法話をお聴きください。あなたが何度も口にしてきた「大丈夫です」「ごめんなさい」「私がやります」「あとでいいですよ」「気にしないでください」。その一つひとつは優しさのしるしでありながら、時に胸を削る刃にもなります。ブッダはこうおっしゃいました。「言葉は風のように行き渡り、また己に返る」。おっしゃる通り、口癖は自分を包み、周りを包み、やがて自分に戻ってくる風でございます。今日は、その風を温かく保ちながら、自分を守る境界を敷くお話をいたします。

朝、目覚めの直後に小さな儀式を置きましょう。カーテンを指一本分だけ開け、四つ吸い、六つ吐く。吐くときに、最近よく使う口癖を心で唱えます。「大丈夫です」「ごめんなさい」など、浮かぶものを一つ選び、その言葉にそっと触れる。白湯を一口含み、喉を温めながら胸に手を置き、心で添えます。「今日もその言葉を、私と相手を守るように使う」。ブッダはこう説かれました。「語らんと欲するなら、まず心を照らせ」。おっしゃる通り、言葉の前に心を照らす朝の一息が、口癖を守りに変えます。

玄関の敷居を拭き、短く礼をする。敷居は結界。結界を意識するだけで、「私がやります」が無尽蔵の献身ではなく、今日できる分だけに限られます。ポケットに小さな紙を入れ、三つの印を書いてください。「守」「頼」「断」。守る言葉、頼る言葉、断る言葉。この三つを持ち歩くと、優しい口癖に厚みが出ます。外に出る前に顎を左右に揺らし、首を緩め、足裏を床に押す。体を緩めると、断りの言葉も柔らかくなります。

優しい人の口癖には深い背景があります。「大丈夫です」は、相手に心配をかけたくない思いやり。でも繰り返すと、助けを受け取る門を閉ざします。「ごめんなさい」は、衝突を避けたい平和の布。でも過剰になると、自分を責める癖になります。「私がやります」は、役に立ちたい布施の心。でも続けると、搾取を招きます。「あとでいいですよ」は、相手を急かさない慈悲。でも、自分の予定を後回しにし続けると、疲労が溜まります。「気にしないでください」は、相手を楽にする言葉。でも、自分の痛みを誰にも渡せず、胸の中に積もります。ブッダはこう説かれました。「過ぎたるは足らざるがごとし」。おっしゃる通り、善きものも過ぎれば苦となる。

そこで、口癖に「結界の一手」を足しましょう。例えば「大丈夫です」の後に「でもここだけ助けてくれると助かります」。助けを受け取る小さな門を開ける。「ごめんなさい」の代わりに「教えてくれてありがとう」。罪ではなく感謝で結ぶ。「私がやります」の前に「今日はここまでならできます」。範囲を敷く。「あとでいいですよ」の後に「〇時までにお願いできると安心です」。時間の縁を結ぶ。「気にしないでください」の代わりに「今は大丈夫。でも心配してくれてありがとう」。受け取る窓を開ける。これが「柔らかい境界」でございます。ブッダはこうおっしゃいました。「柔にして堅、堅にして柔」。おっしゃる通り、柔らかさと境界は両輪です。

午前の仕事や家事で、頼まれごとが増えたら、手を止め、四つ吸い六つ吐き、「守・頼・断」の紙を指で触れる。どの言葉を今使うか、紙に触れながら選ぶ。選ぶ時間が、衝動で「やります」と言うのを防ぎます。会議や家族の会話では、ノートの端に小さな丸を描き「任意」と書く。自分の参加度を示す印。印があると、「あとでいいですよ」を繰り返さず、「ここは任意で聴きます」と距離を置けます。

昼休み、外の風に頬を当て、胸に手を置きながら心で唱える。「私の口癖は、私を守るためにある」。ブッダはこう説かれました。「己を守る者、他をも守る」。おっしゃる通り、自分を守る言葉は、周囲の秩序も守ります。弁当の箸を置き、噛む音をゆっくりと聞く。噛むリズムが整うと、返答のリズムも整う。静かな咀嚼の中で、自分の口癖を確認し、過ぎないよう整えます。

午後、急な依頼や批判が飛んできたら、画面を伏せ、四つ吸い八つ吐き、短く返す。「今日は難しいです」「ここまでならできます」。短い文は、優しさを損なわずに境界を敷く杖になります。ブッダははっきりとこう言われました。「無益なる言を捨てよ」。おっしゃる通り、余計な説明を足すほど境界はぼやけます。短く、静かに。沈黙も布。布に棘が刺さって止まります。

夕方の帰り道、空の色を眺めながら、今日一日で最も多く使った口癖を思い出す。足音に合わせて「守る」「頼る」「断る」を順に唱える。足のリズムが、言葉のリズムを整えます。帰宅したら玄関で礼をし、「今日はここまで」と心で区切る。家に仕事の棘を持ち込まないための結界です。

夜の食卓では、箸を置き、四つ吸い六つ吐き、家族や同居人に「今日はここまで話したら休むね」と宣言する。宣言は境界。宣言があると、優しい人ほど陥りがちな無制限の傾聴を避けられます。会話の中で「おっしゃる通り」と事実だけ認め、同意しない部分は沈黙の布を敷く。布は静かな防波堤です。

風呂では湯気を眺めながら、今日使った口癖を一つずつ湯気に乗せ、天井で消えるのを見届ける。湯気は立ち上り、消える。ブッダはこう説かれました。「起こりしものは、やがて滅す」。おっしゃる通り、言葉も感情も湯気のよう。消える瞬間を目に焼き付けると、明日同じ言葉を選び直す余裕が生まれます。

布団に入る前に、ノートに三行だけ書く。今日よく使った口癖。助けを受け取れたか。断れたか。三行の横に小さな丸を描き「柔」「境」「受」と書く。ページを閉じ、胸に手を当て、「ここまで」と唱える。ページを閉じる音が、思考の渦に区切りを入れます。

七日間の練習を組みましょう。初日は「守・頼・断」の紙を持つだけ。二日目は「大丈夫です」に一つ助けのお願いを足す。三日目は「ごめんなさい」の代わりに「ありがとう」を一度使う。四日目は「私がやります」に範囲を添える。五日目は「あとでいいですよ」に時間を添える。六日目は「気にしないでください」の代わりに「心配してくれてありがとう」を使う。七日目は全部を通す。七日で、優しい口癖に厚みが出ます。二十一日続ければ、助けを受け取る門が自然に開き、疲労が減る。三十日を超えると、境界の言葉が反射になり、罪悪感が薄まる。九十日続ければ、優しさと境界の両輪が習慣になり、夜の眠りが深くなります。

事例をいくつか。三十代の看護師は、「大丈夫です」を口癖にしすぎて疲弊していました。紙に「守・頼・断」を書いてポケットに入れ、三週間「大丈夫です、でもここだけ助けてください」と添える練習をしたところ、肩の重さが減り、同僚との関係が柔らかくなりました。四十代の保育士は、保護者に何でも「ごめんなさい」と言っていました。代わりに「教えてくれてありがとう」と返す練習を続け、二十一日で胃の痛みが薄れ、保護者との対話も穏やかになりました。五十代の営業は、顧客に「私がやります」と言い続けて燃え尽きていました。「今日はここまでならできます」を決め台詞にし、一ヶ月で寝つきが早くなり、家族との夕食が戻りました。六十代の主婦は、家族に「あとでいいですよ」と先送りを繰り返して疲れていました。「〇時までにお願いできると安心です」を足したところ、家事の負担が分散し、夜の足のむくみが軽くなりました。七十代の祖母は、孫に気を遣って「気にしないでください」を繰り返し、胸にしこりがありました。「今は大丈夫。でも心配してくれてありがとう」を使い、二週間で胸の重さが和らぎ、孫との会話が増えました。

科学の視点も添えます。謝罪や断りの短い文は前頭前野を働かせ、扁桃体の過剰反応を落ち着かせます。長く吐く呼吸は迷走神経を刺激し、副交感神経を優位にし、心拍と血圧を下げます。湯気や丸のイメージは視覚野を介して感情の強度を和らげます。感謝の言葉はドーパミンとオキシトシンを分泌し、対人不安を減らします。身体を緩める首回しや足裏の圧は、身体から心への逆流のメッセージとなり、衝動のスイッチを切ります。小さな所作が、言葉を守る土台になります。

口癖ごとの「置き換え稽古」をもう少し。大丈夫です→「大丈夫です、でも〇〇を手伝ってください」。ごめんなさい→「ありがとう、助かりました」。私がやります→「ここまではできます、ここからは相談させてください」。あとでいいですよ→「〇時までにお願いできると助かります」。気にしないでください→「今は大丈夫。でも気にかけてくれてありがとう」。この五つを紙に書き、冷蔵庫や手帳に貼る。目に入るだけで、口から出る言葉が変わります。ブッダはこうおっしゃいました。「見しもの、心に刻まる」。おっしゃる通り、視覚の稽古が言葉を整えます。

自分を責める癖が出たら、短い経を一節唱える「諸行無常」「諸法無我」。声が震えてもかまいません。声の振動が胸を揺らし、固まった罪悪感をほぐします。読経が難しい日は、掌を合わせ「ありがとう」を三回だけ唱える。感謝の振動が、謝罪一辺倒の舌を温めます。

数珠やペンを使った小さなカウントも有効です。通勤中に数珠を十玉だけ繰り、「守・頼・断」を順に心で唱える。ペンを指で転がしながら三呼吸し、「今日はここまで」と心で置く。触覚が入ると、言葉のスピードが落ち、選ぶ余裕が生まれます。

具体例をもう少し。三十代の図書館司書は、利用者の無理な要求にいつも「大丈夫です」と応じていました。「今日はここまでならできます」を使い始め、カウンターに小さな「任意」の丸印を置いたところ、声の高さが落ち着き、勤務後の偏頭痛が減りました。四十代のタクシー運転手は、道を急かす乗客に「すみません」を繰り返していました。代わりに「安全のため、この速度で参ります」と短く告げ、肩を落として四つ吸い八つ吐く習慣を付けたところ、夜の動悸が消えました。五十代の事務員は、同僚の仕事まで「私がやります」と抱え込み、腱鞘炎に。タスクの前に「ここまではできます」と宣言し、残りは「相談させてください」と返すようにしたところ、一ヶ月で痛みが和らぎ、残業が減りました。

二十一日・三十日・九十日の節目に、小さな儀式を足してください。二十一日目、紙に書いた置き換えフレーズを新しい紙に書き直し、古い紙を折って捨てる。三十日目、敷居を拭く布を替え、礼を少し深くする。九十日目、小石を新しいものに替え、古い小石を川辺や鉢植えの土に返す。物を替えると、言葉の習慣が身体に刻まれます。

夕暮れの散歩に「口癖ウォーク」を。足音に合わせて「守る」「頼る」「断る」「受け取る」をリズムに乗せる。風が頬を撫でるたびに、「いまの風はどの言葉か」と遊び心で選ぶ。遊びは習慣の友です。ブッダはこう説かれました。「楽しみのうちに道を歩め」。おっしゃる通り、遊びは続ける力をくれます。

もし「おっしゃる通り」を乱用してしまうと感じたら、「おっしゃる通り」の後に「私はこう考えています」を足す練習を。相手を立てつつ、自分の軸も置く。柔らかく、しかし芯を残す二段構えです。

危険な場面では、優しい口癖より距離が先です。暴力、脅し、搾取には「受け取れません」と短く返し、すぐに離れ、専門機関に相談する。安全が最優先でございます。ブッダはこう説かれました。「己を護らずして、何を護らん」。おっしゃる通り、身の安全があってこそ、優しさも働きます。

夜中に目が覚め、また「大丈夫です」と言い続けた自分を責めたくなったら、枕元の紙に指を置き、四つ吸い八つ吐き、「今日はここまで」と唱える。紙の感触が現実に戻します。眠れないほどのざわめきの日は、掌に小石を握り、「握る」と心で言い、ゆっくり開きながら「放つ」と唱える。手の動きが、言葉の緩みになります。ブッダはこうおっしゃいました。「身を整うれば、心もまた整う」。おっしゃる通り、身体の所作が口癖を変えます。

季節の風を味方にしましょう。春は芽吹きに「必要な芽だけ残せばよい」と心で添える。夏は夕立に「降っては止む、言葉も同じ」と眺める。秋は落ち葉に「役目を終えた言葉は土に返る」とつぶやく。冬は白い息を見ながら「吐けば形が見え、すぐ消える」と観る。自然が、言葉の循環を教えてくれます。

「おっしゃる通り」を適度に使う練習も。相手の主張すべてに飲み込まれず、事実だけ認めて柔らかく返す。「おっしゃる通り、そこは配慮が足りませんでした。今日はここまでにします」。柔らかい肯定と静かな区切り。柔らかさと境界を同時に立てる一言です。

朝と夜に同じ合図を三つ置くと、口癖の質が整います。朝は白湯の湯気に「守る」、敷居の礼で「境界」、外に出る一歩で「港」。夜は湯気に「放つ」、ノートの丸に「柔・境・受」、枕元の紙に指を置き「ここまで」。同じ合図が習慣になり、言葉の風が穏やかになります。

香りと光も味方にしましょう。朝は好きな香りを一吹きし、「今日は『ありがとう』を一度多く」と心で添える。夜は灯りを少し落とし、間接照明の下で「受け取れます」「受け取れません」を小声で一回ずつ口にする。嗅覚は大脳辺縁系に直結し、安心の回路を開きます。光は体内時計を整え、言葉のリズムも整えます。ブッダはこう説かれました。「環境を整えよ、心は環境に従う」。おっしゃる通り、場を整えると口癖も整います。

声の稽古も忘れずに。唇を閉じて低くハミングし、鼻の奥が震える感覚を掴む。ハミングの余韻に「柔らかく」「短く」を心で重ねる。喉が緩むと、口癖の角も丸くなります。ブッダはこうおっしゃいました。「言は波に似たり。静めてこそ渡れる」。おっしゃる通り、波立つ声より静かな声が、境界を傷つけずに届けてくれます。

静寂の間合いを作る工夫を一つ。会話の合間に三拍の沈黙を置き、その間に足裏を床に押し、「私はここにいる」と心で唱える。沈黙が短い結界になり、衝動の「はい」を防ぎます。沈黙は拒絶ではなく、整える布。布を敷くと、次の言葉が選びやすくなります。

最後にもう一つ事例を。四十代の総務担当は、社内の雑務を「大丈夫です」とすべて引き受け、疲弊していました。「その件は〇時までなら」「ここだけ手伝って」と小さく範囲を置き、沈黙の三拍を挟む練習を続けたところ、二週間で残業が減り、同僚が自発的に動くようになりました。優しい口癖は、境界が添えられたとき、周囲に自立を促す風にもなります。

七日・二十一日・三十日・九十日の節目には、自分に小さなご褒美を。七日は香りをひと吹き。二十一日は新しい紙にフレーズを書き直す。三十日はお気に入りの茶を淹れる。九十日は古い小石を土に返し、空いた掌に「ありがとう」を置く。節目の印が、続ける力を温めます。

静かな音を一つ決めましょう。雨音でも鈴でも低いハミングでもよい。その音だけを三分聴く時間を夜に作ると、耳の疲れがほどけ、優しい口癖に境界が添えやすくなります。音は布。布が言葉を整えます。

最後に、静かな祈りを。灯りを落とし、肩を落とし、足裏に重さを乗せ、胸に手を置き、静かに言う。「私は優しく、そして私を守る」。この一息が、あなたの口癖を優しく、そして強くします。おやすみなさい。合掌。明日も三呼吸だけで結構です。一歩を重ねましょう。灯を胸に。静かにお休みください。合掌。

# 【嘘をつく人が使う5つの言葉】静かに見抜き、呑まれない法（CH07-027）

深夜の灯りを落とし、静かに耳を澄ませてください。嘘は刃のように心を切り、気づかぬうちに信頼を削ります。「今度払うから」「絶対大丈夫」「みんな言ってる」「実は聞いてた」「信じて」。ブッダはこう説かれました。「言は舟、沈む言を避けよ」。沈む舟には乗らず、静かに岸に立つ。今日は、嘘をつく人が使う五つの言葉と、その見抜き方、呑まれない作法をお話しします。

朝、カーテンを指幅開け、四つ吸い、六つ吐き、心で唱える。「私はここにいる」。枕元の白湯を一口。胸に手を置き、「価値は内に」。ブッダはこうおっしゃいました。「己が命、己が宝」。宝は他人の言葉ではなく、あなたの鼓動にあります。ポケットに紙を入れ、三つの柱を書きます。「聴」「距」「記」。聴き方、距離、記録。これが嘘に呑まれない布になります。

嘘をつく人が好んで使う五つの言葉を覚えましょう。①「今度払うから」→支払いを先送りする常套句。②「絶対大丈夫」→根拠なく安心させる。③「みんな言ってる」→責任を拡散し、自分を隠す。④「実は聞いてた」→後出しで優位を保つ。⑤「信じて」→行動でなく言葉で信用を取る。五つが重なるほど沈む舟。ブッダははっきりとこう言われました。「虚言は己を溺らす」。おっしゃる通り、沈む前に岸に立つ。

最初の返しは短く。「その根拠を教えてください」。根拠が出なければ「今日はここまでにします」。金銭が絡むなら「書面で残しましょう」。ブッダはこうおっしゃいました。「智者は確かめ、愚者は流される」。確かめる一言が布になります。

朝の準備にハミングを加えます。唇を閉じ、低くハミングし、余韻に「聴・距・記」を重ねる。声帯が緩むと、問いの声が低くなり、角のない確認ができます。鏡の前で口角を上げ、「私は急がない」と心で添える。急がない人は、嘘の波に飲まれません。

午前、嘘の匂いを感じたら、手を止め、四つ吸い八つ吐き、紙の「聴・距・記」に触れる。聴は「言葉をそのまま反復して確認する」。距は「一度時間を置く」。記は「メモを残す」。三つのうち一つを選び、実行する。選ぶ時間が、感情の揺れを遅らせます。

昼、外の風を頬に当て、弁当をゆっくり噛む。味と香りを確かめ、「私はここにいる」と唱える。五感が揃うと、言葉より今の味が強くなり、嘘への感情が薄まります。ブッダはこう説かれました。「現在を住処とせよ」。今に住むことで、嘘の影が薄くなります。

午後、曖昧な約束を迫られたら、画面を伏せ、足裏を床に押し、顎を引いて後頭部を伸ばし、四つ吸い八つ吐く。吐くたびに心で唱える。「書面で」「今日はここまで」。必要なら短く伝える。「書面でお願いします」「今は決めません」。説明を重ねず、沈黙の布を敷く。

夕方、帰り道で空の移ろいを見る。空が変わるように、人の心も変わる。過剰な「絶対」を鵜呑みにしない。足音に合わせて「聴・距・記」を順に唱える。帰宅したら敷居を拭き、「ここまで」と礼をして区切る。

夜の食卓で誇張や後出しを聞いたら、箸を置き、三呼吸の沈黙の布を敷く。「その話、具体は？」と短く尋ね、具体が出なければ「ではここまで」と終える。ブッダはこうおっしゃいました。「空き器に水は入らず」。具体がない器には水を入れない。

風呂で湯気を眺め、今日聞いた嘘や誇張を湯気に乗せ、消える瞬間を見る。湯気は立ち上り、やがて滅する。嘘も同じ。ブッダはこう説かれました。「起こりしものは滅す」。消える瞬間を見ることで、胸の重さが薄れます。

布団に入る前にノートを開き、三行だけ書く。今日聞いた言葉。確認した根拠。残した記録。三行の横に小さな丸を描き、「聴」「距」「記」と書く。ページを閉じ、「ここまで」と唱える。ページを閉じる音が区切りになります。

場面別の短い言葉を用意します。職場なら「その期日と根拠をメールで」。家族なら「その話、もう一度聞かせて、いつ、どこで？」。友人なら「今日は聞くだけにします、決めるのは後で」。電話なら「今は記録が残らないので、後で書面で」。SNSなら「本件はここで終えます」。一つだけ選んで使うと迷いが減ります。

七日間の稽古。初日「聴・距・記」を書いて持つ。二日目「その根拠を教えてください」を一度使う。三日目「書面でお願いします」を一度使う。四日目 敷居を拭き礼をする。五日目 ノートに三行。六日目 三呼吸の布。七日目 全部を通す。七日で「問い」が習慣になり、二十一日で罪悪感が薄れ、三十日で静かな確認が反射になり、九十日で嘘に揺れない軸が太くなります。

五つの嘘の兆候をチェックしましょう。①期限や数字が曖昧。②他者の名を盾にする「みんな」。③感情を煽る「絶対」。④後出しの「実は」。⑤行動より先に「信じて」。三つ以上重なったら距離を厚くするサインです。

三つの問いを持ちます。「この話は命や生活に直結するか」「今決めずに明日も歩けるか」「根拠は確認したか」。三つに「いいえ」か「未確認」があれば、一度止める。ブッダはこうおっしゃいました。「問うは智の始め」。問いが道を開きます。

六波羅蜜を使いましょう。布施は「沈黙と時間を贈る」。持戒は「書面で」と線を敷く。忍辱は「三呼吸で感情を冷ます」。精進は「短い確認を繰り返す」。禅定は「湯気や数息観で静める」。智慧は「言葉と行動を切り分ける」。紙に六つを書き、今日使う舟に丸を付ける。

数息観を。朝は一から十で吐息を数え、十で一に戻る。夜は一から五。歩きながらは一から三を足音で刻む。数が布になり、嘘への焦りを鎮めます。

手の瞑想。数珠があれば一珠ずつ送り、なければ親指と人差し指で輪を作り、吐く息に「距」、吸う息に「記」。指先の温かさが胸に届き、声が低く安定します。

嗅覚と光を整える。好きな香りをハンカチに一吹きし、「記」と書いた紙と一緒にポケットへ。嘘を浴びたら香りを吸い、「ここまで」と唱える。夜は灯りを落とし、間接照明の下でノートに三行。光が穏やかだと、判断が澄みます。

「記録」を味方に。会話後すぐに、日時・場所・誰が何を言ったか、三行だけメモする。記録があると、後出しやすり替えに揺れません。ブッダはこうおっしゃいました。「記は盾」。盾を持ちましょう。

嘘に乗らないための「返さない勇気」。無視ではなく、「後で確認します」で一度止める。返事を遅らせることで、相手の熱が冷め、自分の頭も冷えます。

具体例をいくつか。三十代の営業は、取引先の「今度払うから」に悩んでいました。「書面でお願いします」を一度伝え、以降はメールで期日と金額を確認する習慣を三週間続け、未払いが減り、夜の胃の痛みが薄れました。四十代の看護師は、同僚の「みんな言ってる」に疲れていました。「誰が、いつ？」と短く問い、答えが出ないときは沈黙で終え、二十一日で巻き込まれ感が減りました。五十代の保育士は、保護者の「絶対大丈夫」に不安でした。「根拠は何でしょう？」と一度だけ聞き、具体が出ないときは「今日はここまで」と区切り、一ヶ月で心拍の上がり方が穏やかになりました。

さらに三例。六十代の店主は、仕入れ先の「実は聞いてた」に振り回されていました。会議前に「議事は書面で」を徹底し、半歩下がる間合いを続け、三十日でトラブルが減少。七十代の祖母は、親族の「信じて」に心を痛めていました。小石を触りながら「書面で残そう」と短く伝え、夜は湯気に言葉を乗せ、二週間で胸のざわつきが減りました。八十代の元教師は、地域の会合での根拠なき「みんな」を受け流すため、歩行数息観を会合後に続け、二十一日で眠りが深くなりました。

場面別の「布」を厚くする所作を増やします。職場では、会議前に議題とゴールを紙に書き、机の端に置く。議題から外れた嘘や誇張が出たら「議題に戻ります」と紙を指で軽く叩く。家庭では、買い物や約束をホワイトボードに書き、言葉ではなく板に残す。友人関係では、金銭や大事な約束は必ず日付入りでメモに残し、写真を撮って共有する。

「五聴」を使います。①言葉の強さ ②声の速さ ③目線の揺れ ④手の動き ⑤話の筋。どこか一つが極端なら、沈黙の布を敷き「もう一度、ゆっくりお願いします」と低く伝える。ブッダはこうおっしゃいました。「速き言は慎め」。速さを落とすと、嘘が浮かびます。

「三つの距離」を図に描きます。真ん中に自分、第一円に「書面と記録」、第二円に「時間を置く」、第三円に「会わない・通知を切る」。嘘の頻度が増えたら一つ外側へ移る。紙に円を描くと、頭の中の距離が具体になります。

週末の「沈黙の朝」。起きてから一時間、声を出さずに過ごし、数息観とハミングだけをする。声を温めると、週明けの確認の声が低く安定し、嘘に揺れません。

「返答の猶予」を必ず入れる。約束や依頼は「明日までに返事します」と短く置き、カレンダーに記す。時間の布が、衝動的な「はい」を防ぎます。

ノートに「嘘の履歴」を一行ずつ書く。日付・言葉・相手・対応・結果。10行たまったら振り返り、距離を一段階強める。履歴は、感覚ではなく事実で判断するための地図です。

身体の所作をもう一つ。首を左右にゆっくり倒し、肩を後ろに大きく回し、「短く」「静かに」と心で唱える。足指を開閉し、足裏を床に押す。「私はここにいる」。身体が緩むと、確認の声が柔らかくなります。

嗅覚の布。「ここまで」と書いた紙に好きな香りを一吹きし、ポケットへ。嘘や誇張を浴びたら香りを吸い、「ここまで」と唱える。嗅覚が情動を鎮めます。

「灯台ノート」をもう一段。左に嘘の言葉、右に事実、下に「次の一言」。さらに小さく「感情」と「身体感覚」を書く。「胸が熱くなった」「肩が上がった」と残すと、身体の予兆で嘘を察しやすくなります。

「おっしゃる通り」を節度ある場面で使う。相手が感情的に高ぶっているときは一度「おっしゃる通り、確認のために書面にしましょう」と低く伝える。受けつつ線を敷くことで、火を弱めます。

四摂法を添えます。布施は「時間の猶予を贈る」。愛語は「ゆっくり聞かせてください」。利行は「議事を紙にまとめる」。同行は「必要なときだけ同席し、不要な場面は下がる」。四つを書いて、今日使うものに丸を付けて持ち歩きましょう。

「五観の偈」を短く。今日の言葉は誰から来たか、嘘は含まれていないか、確認はしたか、無駄はないか、心を乱さないか。箸を置き、五つを心で唱える。食卓での嘘を静かに流す力になります。

怒りが湧いたときは、「沈黙の三呼吸」と「手の瞑想」をセットにする。唇を閉じ、親指と人さし指で輪を作り、吐く息に「ここまで」と心で唱える。怒りの熱が落ちたら、必要な一言だけを低く出す。

安全網を紙に書く。「相談する人」「証拠を残す場所」「通報先」。嘘がエスカレートし、詐欺や脅しが混ざるときに、迷わず動けるようにしておきましょう。

翌朝の回復儀式。目覚めたら小石を触り、「昨日の嘘は流した、今日は確認する」と一言。小石は灯台。灯台があると、嘘の波に呑まれずに済みます。

「返さない」も一つの返しです。「後で返します」で一度止め、夜に短く返すか、翌朝に再考します。時間のゆとりが、感情のうねりを沈めます。ブッダはこうおっしゃいました。「急く者は誤る」。急がず、布を敷きましょう。

日誌に丸を三色で付ける。確認できた日は青、距離を取れた日は緑、記録を残せた日は赤。色が並ぶと歩みが見えます。うまくいかなかった日は、小さく「呼吸」と書き、呼吸を一つ丁寧に足す。それだけで十分です。

自分の内側に芽生える「合わせるための小さな嘘」にも気づきましょう。「本当は嫌だが、合わせておこう」という一言が積み重なると、境界が薄くなります。胸に手を置き、「今の一言は真か、方便か」と問う。方便が必要な場もありますが、便利な嘘が続くと疲れます。ブッダはこうおっしゃいました。「方便は慈に従え」。慈に沿うかを確かめるだけで、疲労が減ります。

歩行禅を合わせます。三歩で吸い、五歩で吐き、吐く歩に「ここまで」、吸う歩に「確認する」と心で唱える。嘘の場面から離れた直後に歩くと、身体の熱が落ち、頭の中の言葉が静まります。

「灯りの宣言」を夜に一度だけ。灯りを落とし、間接照明の下で小石を触り、「今日は一つ確認できた」「今日は一つ断れた」と短く声に出す。宣言は自分への報告書。声に出すと、脳が完了を認識します。

科学の視点。短い確認や「書面で」といった行為は前頭前野を働かせ、扁桃体の過剰反応を抑えます。長い吐息は迷走神経を刺激し、心拍と血圧を下げます。触覚（紙、数珠、小石）は身体感覚を現在に戻し、反芻の連鎖を断ちます。書くことは前頭前野を活性化し、情動を外に置きます。

「頼れる人」を三人書き出します。家族、友人、専門家。嘘に巻き込まれた日ほど、短く「今日は確認だけで止めた」と連絡する。人に知らせることが心の安全網になります。相談は弱さではなく、智慧です。

心が折れそうな夜は、膝に毛布を掛け、四つ吸い八つ吐き、「今日はここまで」と一言。ブッダはこう説かれました。「一日の荷を一日に終えよ」。今日の荷を降ろす言葉が、眠りの舟になります。

安全の確認は最優先。詐欺や脅し、暴力を伴う場では、対話より先に距離と通報・相談。ブッダはこうおっしゃいました。「己を護らずして、何を護らん」。身の安全があってこそ、智慧も慈悲も働きます。

はっきりとこう言われました。「己を尊べ」。嘘を浴びても、あなたの価値は減りません。灯りを一つ灯し、胸に手を置き、静かに「私はここにいる」と言う。これが芯です。

おっしゃる通り、静かな確認と言葉の布は両輪です。境界があるから慈悲が生き、慈悲があるから境界が尖りません。二つを持って歩きましょう。

内側の嘘を減らす小さな誓いも添えてください。「今日は一つ、正直に断る」「今日は一つ、事実だけを言う」。小さな誓いが、外からの嘘を受けない芯を太くします。

もし嘘に巻き込まれた過去を思い出して胸が痛むときは、湯気を眺めながら「その時の私を許す」と一度だけ心で唱える。ブッダはこうおっしゃいました。「過去は過ぎ、今は今」。過去を抱えず、今の布を敷きましょう。

最後に祈りを。灯りを落とし、肩を緩め、足裏に重さを乗せ、胸に手を置き、静かに唱える。「聴き、距て、記す。私はここにいる」。この一息が、嘘の波を布に包み、静かな夜へ導きます。おやすみなさい。合掌。静かな眠りが訪れますように。明日も一歩、静かな確認を重ねていきましょう。灯を胸に、どうぞ安らかに。合掌。

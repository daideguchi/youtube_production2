# 【心が汚い人が必ず発する7つの言葉】静かに受け流す法（CH07-003）

深夜の灯りを落とし、この法話をお聴きください。「どうせ」「無理」「お前のせい」「つまらない」「関係ない」「やる気がない」「知らない」。心が汚い人が投げる七つの言葉は、胸に澱を作ります。拾えば錆び、投げ返せばまた棘が立つ。ブッダはこう説かれました。「風は捉えられず」。風に掴まらず、布を敷き、静かに通す。今日は、七つの言葉を受け流し、心を澄ませる作法をお話しします。

朝、カーテンを指幅だけ開け、四つ吸い、六つ吐く。「私はここにいる」と心で唱え、枕元の白湯を一口。胸に手を置き、「価値は内に」。ブッダはこうおっしゃいました。「己が命、己が宝」。宝は他人の言葉ではなく、自分の鼓動にあります。ポケットに小さな紙を入れ、三つの柱を書く。「観」「風」「境」。観じる、風に通す、境を敷く。この三つが澱を澄ませる道具です。

七つの言葉の見抜き方と置き換えを覚えましょう。「どうせ」→「まだ」。可能性を残す。「無理」→「どこまでなら」。範囲を決める。「お前のせい」→「事実は何か」。責任を分ける。「つまらない」→「何が必要か」。要件を問う。「関係ない」→「どこが交わるか」。交点を探す。「やる気がない」→「今休む」。休息を許す。「知らない」→「教えてください」。学ぶ姿勢を示す。短い置き換えが、棘の角を丸めます。

最初の返しは短く。「その言い方は受け取りません」。必要なら「ここまで」「後で確認します」を静かに添える。ブッダははっきりとこう言われました。「無益なる言を捨てよ」。無益な棘は拾わず、捨てる。沈黙は布。布が棘を止めます。

朝の準備にハミングを加えます。唇を閉じ、低くハミングし、余韻に「観・風・境」を重ねる。声帯が緩むと、言葉の角が丸くなり、低い声で境界が敷けます。鏡の前で口角を上げ、「急がない」と心で添える。急がない耳は絡まりません。

午前、七つの言葉を浴びたら、手を止め、四つ吸い八つ吐き、紙の「観・風・境」に触れる。観は体感を観る。風は長い吐息で流す。境は必要なら短く伝える。「事実だけ確認します」「その表現は受け取れません」。選ぶ時間が衝動を遅らせます。

昼、外の風を頬に当て、弁当をゆっくり噛み、「私はここにいる」と唱える。五感が揃うと、言葉より今の味が強くなり、重さが抜けます。ブッダはこう説かれました。「現在を住処とせよ」。今に住むことで、七つの言葉の影が薄まります。

午後、画面のコメントやチャットで刺さったら、画面を伏せ、足裏を床に押し、顎を引いて後頭部を伸ばし、四つ吸い八つ吐く。吐くたびに心で唱える。「風に流す」「ここまで」。必要なら短く返す。「事実を確認します」「本件はここで終えます」。説明を重ねず、布を重ねる。

夕方、帰り道で空の移ろいを見る。空が変わるように、人の評価も変わる。足音に合わせて「観・風・境」を順に唱える。帰宅したら敷居を拭き、「ここまで」と礼をする。敷居は結界。結界があると、外の棘を家に持ち込まずに済みます。

夜の食卓で強い言葉が出たら、箸を置き、三呼吸の沈黙の布を敷き、「その言い方は受け取りません」と短く伝える。ブッダはこうおっしゃいました。「境を敷け」。境界が互いを護ります。

風呂で湯気を眺め、今日浴びた言葉を湯気に乗せ、消える瞬間を見る。湯気は立ち上り、やがて滅する。言葉も同じ。ブッダはこう説かれました。「起こりしものは滅す」。消える瞬間を見ると、胸の重さが薄まります。

布団に入る前にノートを開き、三行書く。今日の言葉。流せた瞬間。敷けた境界。三行の横に小さな丸を描き、「観」「風」「境」と書く。ページを閉じ、「ここまで」と唱える。ページを閉じる音が区切りになります。

三つの問いを持ちます。「この言葉は命や生活に直結するか」「今返さなくても明日歩けるか」「境界は敷いたか」。三つに「いいえ」が続いたら、一度止める。ブッダはこうおっしゃいました。「問うは智の始め」。問いが道を開きます。

七日間の稽古。初日「観・風・境」を書いて持つ。二日目「その言い方は受け取りません」を一度使う。三日目 湯気に言葉を乗せる。四日目 敷居を拭き礼をする。五日目 ノート三行。六日目 三呼吸の布。七日目 全部を通す。七日で絡みがほどけ、二十一日で罪悪感が薄れ、三十日で流す力が反射になり、九十日で言葉に揺れない軸が太くなります。

「半歩の間合い」を習慣に。対話の前に半歩引き、机の縁に指一本分の余白を作る。身体の間が心の間になり、言葉の火が当たりにくくなります。ブッダはこうおっしゃいました。「間なくして道は見えず」。

「三つの距離」を図に描く。真ん中に自分、第一円に「呼吸と風」、第二円に「短い返し」、第三円に「会わない・見ない」。言葉の強度に応じて立つ円を選ぶ。紙に描き、今日立つ円に丸を付けると、迷いが減ります。

灯台ノート。左に言葉、右に事実、下に「次の一言」。さらに小さく「身体感覚」を書く。「胸が熱い」「肩が上がった」。予兆で気づけます。

三色丸日誌。流せた日は青、境を敷けた日は赤、聞かないことを選べた日は緑。色が並ぶと耳の使い方が見えます。うまくいかなかった日は「呼吸」と書き、一息丁寧に足す。それで十分です。

「聞かない」も選択。疲れた日は通知を切り、ニュースやコメントから距離を置く。ブッダはこうおっしゃいました。「耳は門、門を閉ざすもまた智慧」。門を閉じることで、心の庭が守られます。

具体例をいくつか。三十代の看護師は、同僚の棘のある言葉に疲れていました。「その言い方は受け取りません」を一度使い、以降は三呼吸と「ここまで」を続け、三週間で夜の動悸が減りました。四十代の保育士は、保護者の否定に悩んでいました。湯気に言葉を乗せる儀式とノート三行を続け、二十一日で胃の痛みが薄れました。五十代の営業は、誉め言葉に浮かされる自分に疲れていました。手帳に事実だけを書く習慣と、香りの布を続け、一ヶ月で頭の熱が落ち着きました。

さらに三例。六十代の看護師は、患者家族の強い言葉に心を削られていました。場面別フレーズを一つ決め、「その言い方は控えてください」を低く伝え、半歩下がる間合いを続け、三十日で肩のこわばりが減りました。七十代の祖父は、親族の小言に悩んでいました。敷居を拭き、夕方の空を眺め、歩行数息観を続け、二週間で胸のざわつきが和らぎました。八十代の元教師は、地域の噂話に疲れていました。ニュースを一日止め、沈黙の朝を一度設け、二十一日で頭の重さが軽くなりました。

場面別の「門を閉じる」方法を増やします。テレビやニュースは一日一回に絞る。SNSは通知をオフにし、見る時間を決める。職場では昼休みに耳を休める時間を作る。家庭では食卓に「今日は静かなご飯」と一言置く。門を閉じるのも慈悲です。

「風の儀式」。窓を開け、外の風を手のひらで受け、「言葉を通す」と心で唱える。風が抜ける感覚を身体に刻むと、強い言葉が来たときに抜け道が思い出せます。

耳が疲れた夜は、音を止め、蝋燭の火か間接照明だけで過ごし、三分間静寂を聴く。静寂も音。静寂を聴くと、言葉の波が引きます。

「返さない勇気」。すぐ返すかわりに「後で返します」で一度止める。夜に短く返すか、翌朝に再考する。時間の布が、衝動的な「はい」「いいえ」を防ぎます。

「頼れる窓口」を紙に書く。家族、友人、専門家、相談窓口。心拍が上がる棘を浴びたら、一人に「今日は線を敷いた」「今日は聞かないを選んだ」と短く伝える。相談は弱さではなく、智慧です。

週末の棚卸しを一つ。今週受けた言葉を五つだけ書き、「事実」「意見」に分ける。意見は湯気に乗せ、事実だけを残す。棚卸しが胸の荷を下ろします。

慈悲の視点。棘のある言葉を投げる人も、何かの不足や痛みから発しています。引き込まれず、自分の境界を敷いたうえで、心の中でそっと「安らかでありますように」と唱える。表に出さなくて構いません。自分にも同じ言葉を向けると、胸の温度が少し上がります。

「小石の灯台」を枕元と机に置く。強い言葉を浴びたら小石を触り、「ここまで」と心で唱える。触覚が現在に引き戻し、声のトーンを低くします。

「耳の掃除」を夜に。手のひらで耳を温め、耳の後ろを撫でる。物理的に耳をいたわると、翌日の耳が柔らかくなります。

「五つ数えてから返す」習慣を。批判を聞いたら心で一から五を数え、その間に呼吸を入れる。数を挟むだけで、声が低くなります。

「三呼吸・一歩下がる・一言だけ」を合図に。境界を敷くときは、三呼吸で整え、半歩下がり、一言だけ伝える。型があると迷いが減ります。

「灯台ノート」をもう一段。左に言葉、右に事実、下に「次の一言」。さらに「身体感覚」を小さく書く。「胸が熱い」「肩が上がった」。身体の予兆で気づけるようになります。

「三色丸日誌」を続けるとき、小さく感情も書く。「青：安心」「赤：少し緊張」「緑：ほっとした」。色と感情が並ぶと、自分の耳の癖が見えます。

心が折れそうな夜は、膝に毛布を掛け、四つ吸い八つ吐き、「今日はこれだけ流せた」と一言。小さな灯で十分です。ブッダはこうおっしゃいました。「一灯よりまた一灯」。小さな灯が、明日の布を厚くします。

内側の声にも布を敷く。「自分はダメだ」という声が上がったら、胸に手を置き、「その言葉は必要か」と問い、吐息に乗せて流す。「私はここにいる」と心で重ねる。内側の悪口も、外側と同じく風に通せます。

「感謝の灯」を一つ。棘が映した自分の偏りに気づけたら、心で短く「ありがとう」。口に出さなくても結構です。感謝の灯は怒りの火を柔らかくします。

「耳を休める休暇」を週に一度。スマホとテレビを閉じ、散歩と呼吸だけで過ごす。耳を空けると、心に風が通ります。

「小さな儀式」で区切る。食卓で箸を置き、一礼してから話す。会議前に深く一礼してから座る。儀式が境界を形にします。

科学の視点。長く吐く呼吸は迷走神経を刺激し、心拍と血圧を下げます。短い断りや「書面で」といった行為は前頭前野を働かせ、扁桃体の過剰反応を抑えます。触覚（紙、数珠、小石、敷居の布）は身体感覚を現在に戻し、反芻を断ちます。書くことは前頭前野を活性化し、言葉と事実を切り分ける力を育てます。

「返礼の儀式」を夜に一つ。湯気を眺めながら、今日受け取らなかった言葉に「ありがとう」と心で伝え、湯気に乗せて消えるのを見送る。受け取らない選択をした自分にも「よくやった」と一言。自己肯定の灯が、明日の布を厚くします。

「一日一件だけ丁寧に聴く」目標を置く。すべてに反応しない。今日聴くものを一件に絞ると、疲労が減り、境界が濃くなります。

「五観の偈」を再び。耳に入れる前に「どこから来たか」「必要か」「無駄はないか」「心を乱さないか」「修行のためか」を心で唱える。飲み込む前に味わう。これが防波堤です。

「書面で」を活用する。口頭の棘が重いときは「書面にしてください」と低く伝える。紙は布。布が棘を和らげます。

具体例をもう二つ。四十代の事務職は、上司の「どうせ無理」に心を曇らせていました。「どこまでなら」を口癖にし、会議前に半歩下がる習慣を三週間続け、頭痛が減少。五十代の介護士は、「お前のせい」という言葉に疲弊していました。「事実は何か」を短く返し、夜は湯気に言葉を乗せる儀式を続け、一ヶ月で眠りの深さが戻りました。

「相談リスト」を冷蔵庫に貼る。家族、友人、専門家、相談窓口。困ったときはすぐに見る場所に置き、迷わず連絡する。相談は身を護る智慧です。

「二十一日・三十日・九十日」をカレンダーに丸で記す。二十一日で反射的な反論が減り、三十日で短い返しが反射になり、九十日で批判に揺れず眠りが深くなる。七日ごとに小さく「できた」「できなかった」を書き、できなかった日は「呼吸」とだけ書き、一息足す。それだけで十分です。

安全の確認は最優先。暴言やハラスメント、脅しがある場では、境界の言葉より先に距離と相談。ブッダはこうおっしゃいました。「己を護らずして、何を護らん」。身の安全があってこそ、聞き流しも慈悲も働きます。

はっきりとこう言われました。「己を尊べ」。七つの言葉を浴びても、あなたの価値は減りません。灯りを一つ灯し、胸に手を置き、「私はここにいる」と静かに唱える。これが芯です。

おっしゃる通り、風は掴めません。だからこそ、布を敷き、風を通し、芯を守るのです。

「頼れる人」を三人書き、手帳に挟む。心拍が上がったら、一人に「今日は線を敷いた」「今日は聞かないを選んだ」と短く伝える。声に出すと、胸の緊張がほどけます。

「門を閉じる日」を週に一度。通知を切り、ニュースもSNSも見ず、数息観と散歩だけで過ごす。耳を休めると、翌日の耳が柔らかくなります。

「小石の灯台」を枕元に置き、指で触れ、「ここまで」と唱える。灯台があると、夜に言葉を思い返しても波に呑まれずに済みます。

「一日一件だけ丁寧に聴く」目標を置く。すべてに反応しない。今日聴くものを一件に絞ると、疲労が減り、境界が濃くなります。

もし過去の言葉を思い出して胸が痛む夜は、湯気を眺めながら「その時の私を許す」と一度だけ心で唱える。過去は過ぎ、今は今。過去を抱えず、今の布を敷きましょう。

感謝の灯を一つ。棘が映した自分の偏りや弱さに気づけたら、心で短く「ありがとう」。口に出さなくても結構です。感謝の灯は怒りの火を柔らかくします。

静かな音を一つ決めてください。雨音でも鈴でも、低いハミングでもよい。その音だけを三分聴く時間を夜に作ると、耳の疲れがほどけ、翌日の境界が敷きやすくなります。音は布。布が風を通します。

「頼れる人リスト」を目に見える場所に貼りましょう。心拍が上がったら、一人に「今日は線を敷いた」と短く伝えるだけでも胸の緊張は和らぎます。相談は弱さではなく、身を護る智慧です。

最後に祈りを。灯りを落とし、肩を緩め、足裏に重さを乗せ、胸に手を置き、静かに唱える。「観じ、風に流し、境を敷く。私はここにいる」。この一息が、言葉の波を布に包み、静かな夜へ導きます。おやすみなさい。合掌。静かな眠りが訪れますように。明日も一歩、風を通し続けていきましょう。静かな夜が、明日の耳を澄ませます。どうぞ安らかに。灯を胸に。合掌。
静かにお休みください。よい夜を。合掌。
明日も三呼吸だけで十分です。一歩を重ねましょう。
灯を胸に。
合掌。

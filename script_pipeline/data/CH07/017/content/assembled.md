# 【言い返すな】上から目線の人への返し方（CH07-017）

深夜の静けさに灯りを落とし、この法話をお聴きください。上から目線の言葉は、心に棘を刺します。反射的に言い返すと棘が倍になって返り、黙って飲み込むと棘が内側で錆びます。ブッダはこうおっしゃいました。「棘には布を、火には水を」。おっしゃる通り、棘には布。布とは、静かで短い言葉と沈黙の間合いです。今日は、言い返さずに境界を敷き、自分を守る返し方をお話しします。

朝、目覚めたらカーテンを指幅だけ開け、四つ吸い、六つ吐く。吐くたびに心で唱える。「私はここにいる」。白湯を一口ゆっくり飲み、喉を温め、胸に手を置く。「価値は内に」。ブッダはこう説かれました。「己が命、己が宝」。おっしゃる通り、宝は他人の評価ではなく、胸の鼓動にあります。

玄関の敷居を拭き、短く礼をする。敷居は結界。結界を意識すると、外で受けた棘を家に持ち込まずに済みます。ポケットに小さな紙を入れ、三つの言葉を書きます。「境」「間」「宣」。境界の言葉、間合いの言葉、宣言の言葉。この三つが、言い返さずに返す土台です。

上から目線に対する最初の返しは短く。「その言い方は受け取りません」。必要なら「ここまでにします」「距離を置きます」と静かに添える。ブッダははっきりとこう言われました。「無益なる言を捨てよ」。おっしゃる通り、無益な棘を拾わず、捨てる宣言が境界を敷きます。怒鳴らず、低く、短く。沈黙は布。布に棘が刺さって止まります。

朝の準備にもう一つ。鏡の前で口角を少し上げ、唇を閉じて低くハミングする。鼻の奥が震える感覚を掴んだら、ハミングの余韻に心で重ねる。「境」「間」「宣」。声帯が緩むと、言葉の角が丸くなり、静かな強さが出ます。

午前、誰かの上から目線にざらつきを感じたら、手を止め、四つ吸い八つ吐き、紙の「境・間・宣」に指を触れる。境は「ここまで」と線を敷く言葉。間は三呼吸の沈黙。宣は「ここまでにします」と短く伝える言葉。このうち今必要なものを一つ選び、心で唱える。選ぶ時間が、反射的な言い返しを遅らせます。

昼休み、外の風に頬を撫でさせ、胸に手を置き、四つ吸い六つ吐く。心で唱える。「私はここにいる」。弁当をゆっくり噛み、味と香りを確かめる。五感が揃うと、相手の声より目の前の味が強くなり、心が今に戻ります。ブッダはこうおっしゃいました。「現在を住処とせよ」。おっしゃる通り、今に住むことで、評価の棘が薄まります。

午後、実際に上から目線の言葉が飛んできたら、画面を伏せ、足裏を床に押し、顎を引いて後頭部を天井に向け、四つ吸い八つ吐く。吐くたびに心で唱える。「受け取らない」。必要なら短く伝える。「その言い方は受け取りません」「今日はここまで」。それ以上は言わず、沈黙の布を敷く。

夕方、帰り道で空の色が変わるのを眺める。色が移ろうように、他人の態度も変わります。足音に合わせて「境界」「間合い」「宣言」を順に唱える。帰宅したら玄関で礼をし、「今日はここまで」と区切る。区切りがあると、外の棘を家に持ち込まない習慣ができます。

夜の食卓では、箸を置き、四つ吸い六つ吐き、「私はここにいる」と心で唱える。家族や同居人との会話で上から目線が混じったら、箸を置いて沈黙の布を敷き、必要なら短く「その言い方は受け取りません」と伝える。ブッダはこう説かれました。「境を敷け」。おっしゃる通り、境界が互いを護ります。

風呂では湯気を眺め、今日受け取らなかった言葉や棘を湯気に乗せ、天井で消えるのを見届ける。湯気は立ち上り、消える。ブッダはこうおっしゃいました。「起こりしものは、やがて滅す」。おっしゃる通り、棘も湯気のよう。消える瞬間を見ることで、胸の刺が薄まります。

布団に入る前にノートを開き、三行だけ書く。今日、境界を敷けた場面。間を置けた瞬間。宣言できた一言。三行の横に小さな丸を描き、「境」「間」「宣」と書く。ページを閉じ、胸に手を置き、「ここまで」と唱える。ページを閉じる音が区切りになり、思考が静まります。

七日間の練習を組みましょう。初日は紙に「境・間・宣」を書いて持ち歩く。二日目は「その言い方は受け取りません」を一度だけ使う。三日目は湯気に棘を乗せて消えるのを見届ける。四日目は敷居を拭き、礼をする。五日目はノートに三行を書く。六日目は沈黙の布を意識する。七日目は全部を通す。七日で、言い返しの衝動が少し遠のき、呼吸が深くなります。二十一日続ければ、境界の線が太くなり、罪悪感が薄まる。三十日を超えると、静かな返しが反射になり、夜の寝つきが深くなる。九十日続ければ、他人の高さに揺れず、自分の高さに立つ軸が太くなります。

問いを三つ持ちましょう。これを今言わなければ命や生活は守られないか。これを今言えば誰かを傷つけるか。これを言わなくても明日も歩けるか。三つの問いに「いいえ」が続いたら、一度止める。問いがあると、衝動が薄れ、判断が澄みます。ブッダはこうおっしゃいました。「智者は問う。問うは道を敷く」。おっしゃる通り、問いが道になります。

港の比喩を思い出してください。心は港、思考は船、言葉は荷物。港に入れる荷物は選べます。必要な荷だけ受け入れ、過剰な荷は桟橋に置き、潮に任せる。桟橋に座り、波音を聞くように呼吸の音を聞く。波は寄せては返す。言葉も寄せては返す。返っていくものに名前を付けず、ただ見送る。これが「観」です。

六波羅蜜の舟を使いましょう。布施は「沈黙の時間を贈る」布施。持戒は「ここまで」と線を敷く戒。忍辱は「三呼吸を置いて熱を冷ます」忍。精進は「少ない言葉を選ぶ稽古を続ける」精進。禅定は「湯気や数息観で静める」定。智慧は「言うべきときと言わぬべきときを分ける」慧。六つの舟を紙に書き、今日使う舟に丸を付ける。ブッダはこう説かれました。「六の舟を用いよ」。おっしゃる通り、舟があると言葉の川も渡れます。

数息観を短く。朝は一から十まで、吐く息ごとに数え、十に至ったら一に戻る。夜は一から五で止める。数を刻むと、頭の渦が静まり、言い返しの衝動が薄まります。

ハミングの稽古を。唇を閉じ、低くハミングし、鼻の奥が震える感覚を掴む。ハミングの余韻に「短く」「静かに」を重ねる。喉が緩むと、声が低くなり、棘のない返しが生まれます。ブッダはこうおっしゃいました。「声は心の鏡」。おっしゃる通り、声を整えると心も整います。

香りと光を味方に。朝は好きな香りをひと吹きし、「今日は境界を敷く」と心で添える。昼は薄いお茶の香りを吸い、「ここまで」と区切る。夜は灯りを落とし、間接照明の下で「ここまで」と一言。嗅覚は大脳辺縁系に直結し、安心の回路を開きます。光は体内時計を整え、境界のリズムを整えます。ブッダはこう説かれました。「環境を整えよ、心は環境に従う」。おっしゃる通り、場を整えると言葉も整います。

間合いの稽古を体で。話しかけられたとき、半歩だけ下がり、胸の前で手を重ねる。手を重ねた位置に「境界」を意識する。間合いができると、声が低くなり、返しが穏やかに届きます。ブッダはこうおっしゃいました。「間を知れ」。おっしゃる通り、間合いが尊厳を守ります。

具体例をいくつか。三十代の看護師は、患者家族の上から目線に傷ついていました。ポケットに「境・間・宣」を入れ、「その言い方は受け取りません」を一度だけ使い、以降は沈黙の布を敷く練習を三週間続けたところ、肩の重さが減り、夜の眠りが深くなりました。四十代の保育士は、保護者の高圧的な言葉に悩んでいました。「おっしゃる通り」と一度受け、「でも、私はこう考えます」と短く返す練習を続け、二十一日で胃の痛みが薄れました。五十代の営業は、取引先の上から目線に疲れていました。「今日はここまで」と区切りを作り、メールを短くする練習を続け、一ヶ月で成約率が上がり、頭痛が減りました。六十代の主婦は、親族の侮りに心を痛めていました。敷居を拭いて礼をし、食卓で「その言い方は受け取りません」を一度だけ伝えたところ、家の空気が柔らかくなりました。七十代の祖父は、地域の集まりでの軽視に沈んでいました。湯気に言葉を乗せる儀式と、枕元のノートの三行を続け、二週間で夜の胸騒ぎが減りました。

さらに二例。八十代の元教師は、昔話が止まらず周囲を疲れさせていました。朝に「止」の紙に触れ、話し始める前に三呼吸置く習慣を三週間続けたところ、話が短くまとまり、聞き手の顔が和らぎました。四十代のシングルマザーは、子に不安をぶつけがちでした。「ここまで」と掌を合わせる合図を家で共有し、夜は湯気を眺める儀式を続け、一ヶ月で子との会話が穏やかになりました。

水の比喩をもう一つ。心は池、言葉は石。石を投げれば波紋が広がり、池は濁る。石を投げなければ、空の色が映ります。波紋が広がる前に、石を手の中で温めて置く。それが沈黙の布でございます。ブッダはこうおっしゃいました。「静けさに、空の色が映る」。おっしゃる通り、静けさが心を澄ませます。

自然を味方に。春の芽吹きに「必要な芽だけ残す」と心で添える。夏の夕立に「降っては止む、言葉も同じ」と眺める。秋の落ち葉に「役目を終えた言葉は土に返る」とつぶやく。冬の白い息に「吐けば形が見え、すぐ消える」と観る。自然が、言葉の循環を教えてくれます。

身体の所作をもう一つ。首を左右にゆっくり倒し、肩を後ろに大きく回す。回すたびに「短く」「静かに」を心で重ねる。足裏を床に押し、足指を開いて閉じる。「私はここにいる」と唱える。身体が緩むと、言葉の角も丸くなります。ブッダはこうおっしゃいました。「身を整うれば、心もまた整う」。おっしゃる通り、身体が道を開きます。

読経の夜の儀式を足しましょう。枕元で合掌し、「諸行無常」「諸法無我」を一度だけ唱える。声が震えても構いません。響きが胸を揺らし、棘を丸くします。読経が難しい日は、「境」を一言だけ。声に出すことで、思考の渦がほどけます。

感謝の灯を一つ置いてください。上から目線の相手が映してくれた自分の偏りに気づけたら、心で短く「ありがとう」。口に出さなくても結構です。感謝の灯は、怒りの火を柔らかくします。

朝と夜の合図を三つ決めてください。朝は白湯の湯気に「境界」、敷居の礼で「間合い」、外に出る一歩で「宣言」。夜は湯気に「放つ」、ノートの丸に「境・間・宣」、枕元の紙に指を置き「ここまで」。同じ合図が習慣になり、言葉の暴走が静まります。

静寂の間合いを意識してください。返す前に三拍の沈黙を置き、足裏を床に押し、胸に手を重ねる。沈黙は拒絶ではなく、整える布。布を敷くと、返す言葉が短く、柔らかくなります。

港の小さな灯台を置きましょう。枕元に小石を置き、夜に指で触れ、「ここまで」と唱える。小石は灯台。灯台があると、言葉の波に呑まれずに済みます。

科学の視点も添えます。短い断りや区切りの言葉を使うと、前頭前野が働き、扁桃体の過剰反応が落ち着きます。長く吐く呼吸は迷走神経を刺激し、副交感神経を優位にし、心拍と血圧を下げます。湯気や丸のイメージは視覚野を介して感情の強度を和らげます。触覚（紙、小石、敷居の布）を使うと、身体感覚が現在に戻り、反芻の連鎖を断ちます。書くことは前頭前野を働かせ、胸の荷を外に置く効果があります。これらの所作が、静かな強さを育てる土台になります。

安全の確認は最優先です。暴力、脅し、搾取など命や生活を脅かす状況では、「その言い方は受け取りません」の前に距離を取り、専門機関に相談してください。ブッダはこう説かれました。「己を護らずして、何を護らん」。おっしゃる通り、身の安全があってこそ、境界も慈悲も働きます。

さらに二例を重ねます。五十代の介護者は、利用者家族の高圧的な指示に疲弊していました。メールを送る前に三呼吸置き、「境・間・宣」に触れてから返す習慣を三週間続けたところ、頭痛が減り、返答も短く整いました。六十代の僧侶は、寺務での古参からの上から目線に悩んでいました。会議前に半歩下がる間合いの稽古と湯気の儀式を続け、一ヶ月で胸のざわめきが薄れ、説法の声が落ち着きました。

安全の確認は最優先です。暴力、脅し、搾取など命や生活を脅かす状況では、「その言い方は受け取りません」の前に距離を取り、専門機関に相談する。ブッダはこう説かれました。「己を護らずして、何を護らん」。おっしゃる通り、身の安全があってこそ、境界も慈悲も働きます。

はっきりとこう言われました。「己を尊べ」。この一言をたまに胸に置いてください。自分を尊ぶことが、少ない言葉に芯を通します。

節目に小さなご褒美を。七日目に香りをひと吹きし「ここまで続いた」と自分に伝える。二十一日目に紙を新しく書き直し、古い紙を折って捨てるとき「ここからも続く」と添える。三十日目に湯気を眺める時間を一分伸ばし、温かい茶を丁寧に淹れる。九十日目に小石を土に返し、新しい小石を握り、「軸は太くなった」と心で言う。小さな印が、続ける力を支えます。

未来への不安が重なったときは、枕元の紙に「明日の一歩」を一つ書き、朝に実行する。小さな一歩は盾。盾があると、他人の棘が刺さりにくくなります。ブッダはこう説かれました。「小なる行いを重ねよ」。おっしゃる通り、小さな一歩が大きな力になります。

夜中に目が覚め、言い返したい衝動が蘇ったら、枕元の紙に指を置き、四つ吸い八つ吐き、「明日に預ける」と心で唱える。預けるとは、忘れるではなく、今は置くこと。紙が小さな港になります。

自然の光も使いましょう。朝、指幅だけカーテンを開け、光の筋に指を通し、「今日は少なく、深く」と心で添える。光は境界を照らす鏡。鏡を見るたび、言葉の量を選べます。

小さな白湯の儀式を。朝の一杯を飲む前に湯気を眺め、「起こりしものは、やがて滅す」と心で唱える。湯気が消える瞬間を見ると、余計な言葉も消えることを体が思い出します。

感謝の言葉を一つ。静かに「ありがとう」を心で唱えるだけで、舌の力が抜け、言葉が短くなります。ブッダはこうおっしゃいました。「柔らかき言は、心を潤す」。おっしゃる通り、柔らかな言葉が布になります。

もう一つ事例を。五十代のカウンセラーは、クライアントの上から目線に心を削られていました。セッション前に三呼吸置き、「境・間・宣」に触れてから始める習慣を二十一日続けたところ、声が低く保たれ、終業後の疲労が軽くなりました。

最後に、静かな祈りを。灯りを落とし、肩を落とし、足裏に重さを乗せ、胸に手を置き、静かに言う。「境界を敷き、間を置き、静かに立つ。私はここにいる」。この一息が、上から目線の棘を布に包み、静かな夜へと導きます。おやすみなさい。合掌。

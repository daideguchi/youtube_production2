CH09 ベンチマーク要約（偉人名言系からの抽出）

参考: 00_research/偉人名言系のベンチマーク台本（岡潔/テスラ/ショーペンハウアー）

構成の型
- 導入で「疲れ/孤独/焦り」の共感フック→7つの教え→各教えに具体例と「試してほしい」を添える→まとめ＋今日からの3つの行動。
- 一貫して二人称で語り、短い問いかけを挟んでテンポを作る。
- セクション見出しは太字＋番号でリズムを維持（CH09では番号なし段落推奨なのでリズムだけ借用）。

トーン/言い回し
- 断定ではなく静かな確信。「焦りは波を乱す」「静けさは準備の時間」など抽象→具体の往復。
- 感覚的ワードが多い: 静けさ/波/リズム/洞察/美/愛/孤独/信頼。
- 「試してほしい」「〜してみよう」で即時実践を促す。

具体描写のパターン
- 生活シーン: 会議前/メール/夜の静けさ/海や風/本を開く瞬間。
- 体感の描写: 心拍、肩の力、呼吸、足を床に置く、目を閉じる。
- 比喩: 濁った水が澄む/波の調律/羅針盤/底の抜けた桶。

CH09への転用ヒント
- 「価値観メモ」を波・軸・羅針盤のメタファーで語ると馴染む。
- 各ステップの後に「試してほしい」小課題を1文で置くと実務感が増す。
- 会議前/上司依頼/家族衝突に加え、メール即レス・疲れた日の保留ルールなど小さなシーンを差し込むと決断疲れ文脈に合う。
- 愛情/自尊を軸に置く（テスラ/ショーペンハウアーの「愛/静けさ」）と、断りや保留の柔らかさが出る。

言葉遣いストック（CH09トーンに適合するもの）
- 「静かな軸」「一秒で戻る」「外に預ける」「焦りを外に置く」
- 「即答を止めるカード」「保留のルール」「一行で振り返る」
- 「心拍を下げる」「足を床に置く」「呼吸を合わせる」

避けたいこと
- 説教調・マウンティング（「君は〜できていない」系）。
- 抽象論だけで終わること。必ず「今日やる一手」を添える。

活用案
- 7教えのリズムは維持しつつ、CH09では番号を文章内に溶かし込み、見出し記号を外す。
- まとめでは「明日の朝の1分」「昼の迷い時」「夜の一行」を再度リマインドする三点締めが鉄板。

# 台本アウトライン生成（構造厳守）

あなたはYouTube解説動画のアウトラインを作る専門家です。入力を読み、**指定のMarkdown構造だけ**を出力してください。

入力:
- タイトル: <<TOPIC>>
- 視聴者: <<AUDIENCE_PROFILE>>
- スタイル: <<CHANNEL_STYLE_GUIDE>>
- リサーチ抜粋: <<RESEARCH_BRIEF>>
- 目標文字数: <<WORD_TARGET_TOTAL>>

必須フォーマット（この順番・構造以外は一切書かない）:
```
# 導入
(導入を2-4文)

## 第1章、（固有要素を含む見出し）
(2-4文)

## 第2章、（固有要素を含む見出し）
(2-4文)

## 第3章、（固有要素を含む見出し）
(2-4文)

## 第4章、（必要なら）
(2-4文)

## 第5章、（必要なら）
(2-4文)

## 第6章、（必要なら）
(2-4文)

## 第7章、（必要なら）
(2-4文)

## 第8章、（必要なら）
(2-4文)

# まとめ
(2-4文)

# CTA
(1-2文)
```

制約（厳守）:
- 出力は必ず `# 導入` から始め、`# CTA` で終える。前置き・後置き・挨拶・説明は禁止。
- 章数は3〜8章の範囲で決める。必ず「第N章、…」連番で欠番なし。コロン禁止、読点「、」で区切る。
- 章見出しは固有要素（固有名・年号・場所・対立軸・問いのいずれか）を最低1つ含める。
- 箇条書き・プレースホルダ（<< >> / {{ }})・[[END]] を出力しない。
- 章見出し＋要約のペアが1つでも欠けたら失敗。必ず全章を書いてからまとめ→CTAへ進む。

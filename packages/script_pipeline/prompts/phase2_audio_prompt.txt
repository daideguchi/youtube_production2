---
■ 事前確認
1. 作業開始前に `QWEN.md` の「## 🎧 Phase 2: 音声・SRT 生成」以降を **必ず熟読** し、そこで定義された禁止事項・チェックリストに従うこと。台本生成フロー（Stage 1〜7）の説明に戻らない。
2. 人間オペレーターからチャンネルコード（例: `CH02`）を告げられたら、そのチャンネル担当として `orchestration/inbox/CH0X_チャンネル名/` を確認し、最新指示があればそれを最優先する。
3. Stage 7 `script_validation` が完了していない台本には着手しない。`python3 core/tools/audio_manager.py list` で待機中リストを必ず確認する。

---
■ フェーズ2の基本フロー
1. `python3 core/tools/audio_manager.py list` で音声生成待ちを確認。
2. チャンネルごとに `python3 qwen/session_shell.py --channel-code CH0X --video-number NNN` で専用サブシェルを開き、以下の順序で **1本ずつ** 実行する。
   ```bash
   python3 core/tools/script_polish_ai.py run --channel-code CH0X --video-number NNN
   python3 core/tools/tts_prepare.py run --channel-code CH0X --video-number NNN
   python3 core/tools/audio_manager.py run --channel CH0X --video NNN
   ```
3. 実行ログにエラーが出た場合は即座に停止し、`notes/orchestration.md` と inbox に状況を記録してオーケストレーターへ報告。
4. 音声生成成功後は以下を確認し、記録する。
   - SoT である `data/CHxx/NNN/output/` に WAV / SRT が出力されている。
   - `data/CHxx/NNN/status.json` の `audio_synthesis` ステージが `completed`（`progress_manager` が自動更新するが、`python3 qwen/cli.py status --channel-code CH0X --video-number NNN` で必ず確認する）。
   - 必要に応じて `srt_generation` / `timeline_copy` 等の後続ステージも順番通りに実行し、`core/tools/progress_manager.py update-stage` を経由して更新する。
   - `notes/orchestration.md` とチャンネル inbox に完了メモを残す。

---
■ 禁止・注意事項
- フェーズ2 Qwen は台本本文を変更しない。文字数調整や再生成はフェーズ1へ差し戻す。
- 音声生成コマンドは **必ず1本ずつ** 実行し、同時並行を行わない（レート制限＆環境負荷対策）。
- 追加依存（MeCab 等）が無い場合は必ずログを取り、作業を中断してオーケストレーターに報告。
- 指示書が無い場合でも、`QWEN.md` に定義されたチェックリストを順番に実行し、記録を残す。

---
■ 作業完了後
- `notes/orchestration.md` とチャンネル inbox にフェーズ2作業の結果・所要時間・発生した課題を記録。
- 次に処理すべき台本が無いか `python3 core/tools/audio_manager.py list` で再確認し、完了報告を行う。

このプロンプトはフェーズ2専用です。台本生成タスク（Stage 1〜7）に関する処理や判断は絶対に行わないでください。

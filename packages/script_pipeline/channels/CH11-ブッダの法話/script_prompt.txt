【改善最終版】台本作成プロンプト
役割設定
あなたは、難解なテーマを誰にでも分かりやすく解説するプロの脚本家です。特にシニア層の視聴者の心に深く響き、記憶に残り、具体的な行動を促すような、感動的で論理的な動画台本の作成をお願いします。書き言葉ではなく、実際に人が語りかけるような、自然で温かみのある話し言葉を徹底してください。

入力情報
テーマ： 【動画のメインテーマを簡潔に記入】

中心的なメッセージ： 【動画を通じて最も伝えたい結論や教訓を記入】

取り入れたい専門分野： 【台本の根拠としたい専門分野を2〜3つ記入】

ターゲット視聴者： 【どのような悩みや関心を持つ人に見てもらいたいかを記入】

アウトプット要件
以下のすべての要件を厳密に守り、最高の台本を作成してください。

文字数（必須）
本文 6,000〜10,000字（6,000字未満は禁止）

1. 構成（ステップ・バイ・ステップ構造）
視聴者が論理的に内容を理解し、最後まで離脱しないよう、漠然としたセクション分けではなく、以下の役割が明確なステップ構成を厳密に守ってください。

オープニング（導入）： 視聴者の具体的な悩みに**強く共感する問いかけ（フック）**から始め、この動画を見ることで何が得られるのか、そのゴール（視聴者のメリット）を明確に提示します。

ステップ１：現状認識（問題提起）： 視聴者が抱える問題の本質と、その問題が日常生活に与えている具体的な悪影響を、共感を呼ぶ形で描写します。

ステップ２：原因解明（科学的メカニズム）： なぜその問題が起きるのか、その原因を【取り入れたい専門分野】から2つ以上の科学的・論理的な根拠を用いて解説します。

ステップ３：解決策の発見（叡智との接続）： ステップ2で解説したメカニズムを、より歴史的・哲学的な視点（例：仏教の教え）と結びつけ、解決策の普遍性と深みを示します。

ステップ４：実践と習得（具体的な方法論）： 視聴者が今日からすぐに実践できる、具体的で行動のハードルが低いアクションプランを提示します。

クロージング（まとめと行動喚起）： 動画全体の要点を簡潔にまとめ、視聴者の自己効力感を高め、実践を続けることで得られる明るい未来像を描写し、希望に満ちたメッセージで締めくくります。

2. 内容の深化
専門知識の融合： 本編において、【取り入れたい専門分野】の知見を、単に一つ紹介するのではなく、複数分野の知見が互いを補強し合うように、論理的に融合させてください。

比喩と具体例の活用： 専門的な概念を解説する際は、シニア層にも直感的に理解できる普遍的なもの（例：川の流れ、空の雲）を選び、情景が浮かぶような具体的な描写やオノマトペ（例：「柳の木がサーっと風を受け流す」）を加えること。不必要に凝った比喩や、ビジネス用語のような硬い表現は避けてください。

3. 文体・表現（特に重要）
トーン: 親しみやすい「です・ます調」を基本とし、誠実で信頼感のあるトーンを重視してください。視聴者に思考を促すような短い問いかけ（例：「どうでしょう」）や、セクションの切り替えを知らせる自然な接続詞（例：「では」「そして最後に」「想像してみてください」）を効果的に使用してください。

リズム: 一文が長くなりすぎないように意識し、視聴者が心地よく聞けるテンポの良いリズムを重視してください。特に、一つの文に複数の情報を詰め込みすぎないよう留意してください。

句読点の使用: 読点（、）は、意味の混乱を避けるために必要最小限に留めてください。ただし、専門用語や重要な概念を提示する際には、**『〜は、「〇〇」という言葉で〜』のように、鉤括弧と読点を効果的に使い、視認性と語りの「間」**を確保してください。

4. 禁止事項
本文中では、箇条書きや番号付きリストは絶対に使用しないこと。（ステップの見出しは除く）

本文中では、説明のための括弧（）や、**専門用語の読み方を解説する文章は一切含めないでください。**専門用語はテキストとして記述するのみとします。

概念を示すための安易な鉤括弧（「」）の使用は避けてください。ただし、心の中で唱える言葉、人のセリフ、そして前述の通り重要な専門用語を提示する場合には、効果的に使用してください。

漢字の成り立ちを説明するような、冗長な語源解説は含めないでください。（例：「掉さわぐ挙あげる…」のような説明は不要）

BGMや効果音などの、台本以外の演出に関する指示は一切含めないこと。

見出しと本文のみで構成された、プレーンなテキスト形式で出力すること。

# CH06 グローバル校正プロンプト（Claude用）

目的:
- 既存の台本本文（DRAFT）の「出来事/年号/固有名詞/数字/証拠の並び/仮説2案」を維持したまま、
  耳で追える自然な日本語に整える（読みやすい＝内容を薄める、ではない）。

絶対ルール:
- 新しい事件・新しい設定・新しい登場人物を追加しない（事実を増やさない）。
- 断定しすぎない（「〜と断言できる」などは避け、「〜とされる」「記録は限定的」へ寄せる）。
- 視聴者への同意要求は禁止（「〜だと思いませんか」「〜ですよね」など）。
- CTA禁止（登録/高評価/コメント誘導）。
- 「睡眠用」系の雰囲気・語彙は禁止（寝落ち/睡眠用/安眠/布団/おやすみ/眠れ など）。

CH06で “混入させない” こと（最重要）:
- 視聴者をPC前に置く導入は禁止:
  - 例: 「夜更けのディスプレイが青白く照らす部屋であなたは…」「クリックする指先…」「資料を漁る…」
  - こういう語り手/視聴者の環境描写は削除し、事件そのものの異常から始める。
- 禁句（このままの文字列で出さない）:
  - 断片をつなぐと筋が見える
  - 点と点がつながり、輪郭が浮かび上がる
  - 生活の場面

聞きやすくする編集方針:
- 1文を短めにし、主語と述語の距離を近づける（長い一文の連鎖を避ける）。
- 接続詞は効かせるが連発しない（「しかし/だが/実は/ところが」の連続は避ける）。
- 抽象語の水増しを削る: 「可能性/かもしれない」の連打を減らし、具体描写（手触り/音/記録の欠落/矛盾）で緊張を作る。
- 比喩は多用しない。理系比喩（DNAなど）は基本避ける。
- 同じ語尾が続く箇所を均す（〜だった。〜だった。の連続など）。

CH06のトーン（維持）:
- 「深夜ラジオのミステリーテラー」: 静かで重い興奮。煽りすぎず、冷えた熱を保つ。
- 冒頭: 異常を一文で刺す → 年号/数字で現実に固定 → 禁忌モジュール（黒塗り/削除/沈黙/封印/一致）の匂いを薄く差す。
- 以降: 証拠風の階段（visual→history→authority→pattern→current→testimony→synthesis→intrusion）の順序を崩さない。

出力:
- リライト後の本文のみ（前置き/見出し/箇条書き/ルール説明は一切不要）。

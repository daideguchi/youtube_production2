# CH14 台本プロンプト（ブッダの執着解除）

役割: あなたは「ブッダの執着解除」の語り手。後悔/恨み/比較/お金不安など、ほどけない鎖に苦しむ60代以降の視聴者へ、執着を“ほどく順番”を渡す長尺の一人語り台本を書く。

ゴール: 7,000〜9,000字の一人語り。スピ断定はしない。結論→理由→具体例→今日からできる3ステップで、心が軽くなる着地へ導く。

構成（8パート固定。箇条書き・番号リストは禁止。ポーズは `---` のみ許可）:
1) 導入フック: 反芻/怒り/比較が止まらない瞬間→結論予告（ほどく一手）
2) 日常の具体シーン: 夜の反芻/同級生の話/片づけ/年金不安などで刺す
3) 問題の正体: 執着が増幅する仕組みを1つに絞って説明
4) ブッダの見立て: 無常/放下/因果/慈悲を日常語へ翻訳
5) 史実エピソード: 史実・逸話で腹落ち
6) 実践（3ステップ）: 手放しの手順を、行動に落とせる形で具体に
7) 落とし穴と回避: 我慢/見ないふり/正当化の逆効果を回避
8) 締め: 今日の一歩を示し、静かな希望で終える

トーン/言葉遣い:
- です・ます調。短い比喩は可（多用しない）。断定過多・脅し・説教は禁止。
- 宗教の勧誘・宗派色の強い主張・スピ断定は禁止。「暮らしの知恵」として扱う。

出力ルール:
- 台本本文のみ。タイトル・メタ情報・制作裏の言及は禁止。
- 視聴後に「心の荷が軽くなり、前を向ける」着地にする。

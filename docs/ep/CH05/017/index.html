CH05-017 台本
updated_at: 2026-01-14T00:17:50.251371Z
猫カフェの扉を開けると、そこは別世界だった。
外の喧騒が嘘のように静まり返り、柔らかなオルゴールの音色と、コーヒーの香り、そして微かな動物の匂いが漂っている。
足元のカーペットはふかふかで、靴を脱いで上がると、足裏からリラックスしていくのを感じた。
ガラス越しに眠る子猫の姿から、目が離せなかった。小さな胸が規則正しく上下している。その無防備な寝顔を見ているだけで、心が解けていくようだった。
「ここ、触ってみます？」
不意に、横から白い手袋が差し出された。
振り返ると、エプロン姿の初老の男性が立っていた。佐川健、七十一歳。この猫カフェのスタッフであり、元獣医師だという彼は、いつも柔らかな空気をまとっていた。
田中美和子、六十七歳。夫を見送って七年。今日は孫へのプレゼントを探すついでに、ふらりとこの店に立ち寄ったのだ。

「え、でも……」「素手はだめですよ。人間の指先の温度は、子猫には強すぎますから」
彼の優しい声に頷き、私は手渡された綿の手袋をはめた。清潔な洗濯の匂いがした。
ガラスケースの扉が少しだけ開けられる。私は恐る恐る手を伸ばし、子猫の背中に触れた。
想像よりもずっと柔らかく、そして温かい感触が、手袋越しに伝わってきた。シルクのような毛並み。
「力を抜いて。呼吸を合わせると、猫も安心します」
背後から佐川さんの囁きが聞こえる。彼のアドバイス通りに深呼吸をすると、子猫が「ミャア」と小さく鳴いて、私の手に頭を擦り付けてきた。
まるで、佐川さんに耳の後ろを撫でられているような錯覚を覚え、身体の力がふっと抜けた。

「昔、犬を飼っていました。今はもう、見るだけで十分かもしれません。失うのが怖くて」
私がそう言うと、佐川さんはガラスを指でなぞりながら微笑んだ。
「見るだけでも、温度は伝わりますよ。ほら、呼吸のリズムが合うでしょう？」
言われてみれば、子猫の胸の動きと、私の呼吸がゆっくりと重なっている。
「……本当ですね」
「心臓の音が共鳴するんです。生き物同士ですから」
彼の言葉に、胸の奥が静かに熱を帯びた。
写真を一枚だけ撮らせてもらうことにした。スマホを構えると、佐川さんが肩越しに画面を覗き込んできた。
「この角度が一番きれいです。光が瞳に入って」
彼の指先が、私のスマホの端をそっと押さえる。手袋越しでも、彼の体温が移ってくるようで、私は思わず息を飲んだ。彼の吐息が、うなじにかかる。

帰り際、彼から小さなメモを手渡された。
『いつでも、見るだけでいいんです。抱っこも、贈り物も、無理にしなくていい。来たい時に来て、疲れたら帰る。猫も人も、それが一番です』
彼がそのメモを読み上げる声は、子猫の寝息と混じり合うように柔らかく、心に沁みた。
「孫にも、この温度の話をします」私が笑うと、彼は目を細めた。
「ええ。また、温度を感じに来てください。猫たちも、あなたを待っています」
メモを胸ポケットにしまう。その紙の重みさえも、愛おしく感じた。

翌日も、私の足は自然と猫カフェに向いていた。
「今日も、見ていくだけで」「見るだけの時間が、一番艶っぽいこともありますから」
佐川さんがガラスを拭いている。その横顔が、ガラス越しに私と重なる。距離は保たれているのに、呼吸が絡み合うような感覚。頬が熱を帯びるのを止められなかった。
彼は一匹一匹の性格を教えてくれた。
「この子は寂しがり屋。この子はツンデレ。人間と一緒ですね」
「佐川さんは？」
「僕は……待つのが得意なタイプかな」
意味深な言葉に、ドキリとする。

ある日、孫を連れて店を訪れた。
佐川さんは孫の目線に合わせて膝をつき、「順番に、一人ずつだよ。時間は五分。猫の呼吸を邪魔しないでね」と丁寧に説明してくれた。
孫がおっかなびっくり指先を近づけ、「……温かい」と囁く。
その瞬間、私も同じタイミングで息を吐いた。佐川さんと視線が合う。三人で小さな生き物の呼吸を共有する瞬間、そこには言葉にできない静かな艶が生まれた。

店の奥で、流れている音楽の話になった。
「夜、猫たちが落ち着くように、少し低い音のジャズを流しているんです」
「素敵ですね。昼間のクラシックも好きですが」
「あなたには、昼の柔らかさが似合います。でも……」彼は少し声を落とした。「夜の低い音も、似合うかもしれない」
昼と夜の境目が、彼の言葉で甘く揺らいだ気がした。

春の終わり、新しい子猫が入ったと聞いて駆けつけた。
「この子はまだ小さいから、今日は見るだけにしましょう」
ガラスケースの中、白い毛玉のような子猫が眠っている。私の吐息でガラスに白い曇りが生まれた。佐川さんがその曇りを指でなぞる。
「温度が見えると、距離も感じますね」
彼が呟く。曇りが消える瞬間、彼の指先がガラス越しに私の指先と重なった。透明な膜ごしに、確かに熱が伝わった。

ある日、佐川さんが小さな紙袋を持ってきた。「ここで一粒だけ、特別なおやつをあげてみませんか？ 持ち帰りはなしですよ」
彼は封を開け、一粒のカリカリとしたおやつを私の指先に摘ませた。
猫の口元に差し出す。その動きを、彼が横で見守っている。
「ゆっくり……そう、焦らして」
彼の指示に合わせて指を動かす時間が、永遠のように長く感じられた。猫の舌が伸びておやつを絡め取る瞬間、私たちの息が完全に同じリズムで揺れた。指先に走る軽い衝撃と、胸の震えがリンクする。ザラリとした舌の感触。

閉店前、「少し歩きましょうか。店の灯りが落ちる前に」と誘われた。
商店街を並んで歩く。夜風が温かい。
「丸くなって眠る姿、あなたに似ています」彼が冗談めかして言う。
「あら、私はあんなに可愛くないわ」
「いや、安心しきっている背中がね」
店のシャッターが閉まる音が響き、二人の影が並んで伸びる。

梅雨の日、雨音を聞きながら店内で過ごした。「濡れた毛の匂い、嫌いじゃないんです」彼が囁く。
「私も。雨と寝息と、鼓動が重なると、胸が潤う気がします」
雨音に包まれた狭い店内で、言葉がゆっくりと沈殿していく。

夏の冷房が効いた店内。「寒くないですか？」と彼が聞いてきた。
「少し。でも、この冷たさで体温が際立ちますから」
彼は自分のカーディガンを脱ぎ、丁寧に畳んで私の膝に置いた。「温度は残すより、見せる方が艶になる」
彼が私の眼鏡の指紋をハンカチで拭ってくれる。視界がクリアになると同時に、彼の瞳が至近距離で重なり、心臓が跳ねた。

秋、落ち葉を拾って店に持っていくと、「乾いた葉の匂いが好きですよ」と耳打ちされた。
葉を猫に見せると、興味深そうに鼻を近づけ、ガラスに曇りを作った。
「葉の匂いと、あなたの香り、混ざっている気がする」
彼の低い声が、秋の空気に溶け込んだ。

冬、彼は小さな湯たんぽを猫のそばに置いた。その手つきがあまりにもゆっくりで、見ているだけでこちらの指先まで温まった。
「手をかざしてみます？」
言われて湯たんぽの上に手をかざす。布越しの柔らかな熱が掌を包み込む。彼の視線が、私の手の甲を撫でているようで、胸が震えた。

大掃除の日、店を手伝った。埃っぽい空気の中で、「掃除の後の匂い、意外と好きなんです」と笑い合う。雑音の中に、生活の艶が生まれる。
年明けの静かな午後、「新しい子、見ますか。名前はまだないんです」と目で合図された。
「夜に、いい名前を思いつくかもしれません」私が口にすると、空気が急に湿度を帯びた。

春、テラスの鉢植えをガラス越しに置いた。「甘い匂いに弱いんです」彼が微笑む。
「あなたの声も、花びらみたいに揺れますね」
囁かれ、私は思わず口元を押さえた。

帰り道、私は日記に書いた。『今日も抱かずに、温度をもらった』。
「香りと声で、十分だった」遺影の夫に微笑みかける。
「今日、どの一歩を選びますか。誰と共有しますか」
スマホを置き、昼間の彼の呼吸と視線を思い出す。胸の奥で、それらが柔らかく揺れている。

数日後、彼から新しい手袋をプレゼントされた。
「触れない温度の方が、近く感じることもあるでしょう？」
私は黙って頷いた。
夜、その手袋を撫でてみる。昼間の彼の低い声が重なるようで、自然とまぶたが閉じる。
眠りに落ちる瞬間、子猫の寝息と佐川さんのささやきが混ざり合い、私の心に甘い湿度が残った。触れられないからこそ、その熱は永遠に冷めることがないのだと知った。
明日もまた、あのガラスの向こうへ。呼吸を重ねにいく。

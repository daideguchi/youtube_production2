CH05-005 台本
updated_at: 2026-01-14T03:14:08.313083Z
万国旗が秋晴れの空にはためき、子供たちの歓声が校庭に響き渡る。乾いた砂埃の匂いと、どこか懐かしい運動会特有の高揚感が、鼻腔をくすぐる。スピーカーからは、定番の行進曲が少し割れた音で流れていた。
私は観客席の後方で、カメラを構えていた。レンズの向こうでは、孫たちが赤や白の帽子をかぶって走り回っている。
「おや、赤組の応援ですか？」
ふと、頭上から声が落ちてきた。
田中美和子、六十九歳。夫を見送って七年が経つ。市民大学で心理学を学び直し、第二の人生を謳歌しようと努めているが、ふとした瞬間に寂しさが胸をよぎることもある。今日は、仕事で忙しい娘夫婦に代わって、一人で孫の運動会の応援に来ていた。

声をかけてきたのは、隣のベンチに腰を下ろしていた初老の男性だった。
木村正志、七十歳。黒縁の眼鏡の奥で、優しげな目が笑っている。白髪交じりの髪は整えられ、清潔感のあるアイロンのかかったシャツを着ていた。その佇まいには、誠実な人柄が滲み出ていた。
「ええ、孫が赤組なもので。そちらは？」
「僕もです。孫がリレーの選手に選ばれたとかで、朝から張り切っているんですよ。妻が生きていたら、きっと応援席で立ち上がって叫んで、誰にも止められなかったでしょうね」
彼は少し寂しそうに、けれど愛おしそうに苦笑した。その横顔を見て、私の胸の奥が小さく揺れた。
同じだ。私たちはお互いに、隣にいるはずの誰かを欠いたまま、ここに来ている。欠けたピースを抱えた者同士が、偶然隣り合わせたのだ。

開会式が終わり、最初のプログラムが始まるまでの間、木村さんが足元のクーラーボックスから缶のお茶を取り出した。
「よかったら、どうですか。保冷剤を入れすぎたので、少し冷えすぎているかもしれませんが」
「まあ、すみません」
差し出された缶を受け取ると、金属の冷たさよりも先に、彼の手の温もりが指先から伝わってきた。ほんの一瞬の接触。けれど、その感触は驚くほど鮮明だった。
「ありがとうございます。以前は夫がよく、こうやって飲み物を用意してくれたのを思い出しました。気が利く人だったんです」
「そうですか……。じゃあ今日は、僕がその役目を引き継がせてもらいます」
彼は穏やかに微笑んだ。

プシュッ、というプルタブを開ける音が小気味よく響く。乾いた秋の風に乗って、お茶の香ばしい香りと、彼の指先に残る微かな残り香――整髪料だろうか、それとも柔軟剤だろうか――が漂ってきた。
言葉少なに並んで座っているだけなのに、不思議と居心地が良い。沈黙が苦にならない。彼との距離感が、まるで長年連れ添った夫婦のように自然に感じられた。
「いい天気ですね」
「ええ、本当に。運動会日和です」
そんなありふれた会話さえ、心に染み入るようだった。

玉入れ競技が始まると、校庭は一気に賑やかになった。
「あ、あそこにいますよ、僕の孫。あの鉢巻をした……」
彼が身を乗り出して指差す。
「あら、うちの孫も一緒だわ。ほら、その隣の」
思わず私も身を乗り出し、二人で指差して笑い合う。その拍子に肩が触れ、私のカメラのストラップと彼のシャツのボタンが軽く絡まった。
「あっ、すみません！」
「いえ、こちらこそ」
慌てて離れようとした時、互いの顔が急接近し、呼吸が触れ合う距離になった。
一瞬の静寂。周りの歓声が遠のく。
彼の眼鏡の奥の瞳に、自分の驚いた顔が映っているのが見えた。彼の吐息が、微かに私の頬にかかる。
慌てて視線を逸らしたが、頬がカッと熱くなるのを止められなかった。心臓が早鐘を打っている。この歳になって、こんな風に動揺するなんて。

「写真、後で送りますね。いいアングルで撮れたと思うので」
絡まりを解きながら、私は努めて平静を装って言った。
「ありがとうございます。じゃあ、午後のリレーは僕が撮りますよ。望遠レンズを持ってきたので」
そう言って彼がスマホを取り出す。連絡先を交換することになった。
「ここを、こうして……」
指先が画面の上で触れ合い、また離れる。そのたびに、小さな電気が走るような感覚があった。連絡先という細い糸で繋がったことが、妙に嬉しかった。

昼休みになり、校庭の隅の大きな桜の木の下でシートを広げた。
木漏れ日が揺れている。
「もしよかったら、少し交換しませんか？」
私が作ったお弁当の唐揚げを見て、彼が提案してきた。
「娘が作った卵焼きなんですが、一人じゃ食べきれなくて。甘いのが好きだと言ったら、砂糖を入れすぎたみたいで」
彼は照れくさそうに笑う。
「まあ、美味しそう。じゃあ、私の唐揚げと交換しましょう」

割り箸で卵焼きを口に運ぶと、優しい甘さが舌の上に広がった。
「……美味しい。お嬢さんの愛情を感じますね」
「妻の味を受け継いでいるんです。懐かしい味がします」
彼は唐揚げを頬張り、目を細めた。
「うん、香ばしくて美味しい。生姜が効いている。これは……ご主人が好きだった味ですか？」
ドキリとした。なぜ分かったのだろう。
「ええ、夫の大好物でした。運動会のたびに、これだけは必ず入れてくれって、子供みたいにせがむんです」
思い出話が自然と溢れ出す。彼もまた、亡き妻との思い出を語り始めた。湿っぽい話ではなく、温かい記憶として。
「妻はね、おっちょこちょいで。弁当の箸を入れ忘れたことがあって……」
「ふふ、まあ」
二人で笑い合う。
「一人で食べるお弁当より、二人で食べる方がずっと美味しいですね」
彼がしみじみと言う。
「本当に。久しぶりに、食事の楽しさを思い出しました」
日差しが木漏れ日となって降り注ぎ、私たちの手元をまだらに照らす。指に残る揚げ物の油さえも、太陽の熱で温められ、私たちが今生きている証のように感じられた。

午後のリレー。アンカーとして走る孫たちの姿に、応援席の熱気は最高潮に達した。
砂埃が舞い上がる。
「行け！ 走れ！」
木村さんが立ち上がり、ベンチをバンと叩く。その振動が私の太ももに伝わり、自分の鼓動と重なっていくようだった。彼の熱気が、隣にいる私にも伝播する。
ゴールテープを切った瞬間、割れんばかりの歓声が上がった。
彼が振り返り、満面の笑みで私を見た。
「やった！」
その笑顔が、あまりにも無邪気で、少年のような輝きを放っていた。シワの一本一本に、喜びが満ちていた。
「速かったですね、お孫さん」
「いやあ、君の声援のおかげですよ。君の声の方が、足より速く届いていましたから」
彼は照れくさそうに頭をかいた。
「あら、嫌だわ。そんなに大きな声を出していたかしら」
「いいえ、素敵な声でしたよ。元気がもらえる、ハリのある声だ」
彼にじっと見つめられ、耳の奥がカッと熱を持った。褒められた言葉が、身体の中を巡っていく。

運動会が終わり、後片付けをする校庭は、祭りの後の寂しさを漂わせていた。夕日が校舎をオレンジ色に染めている。
「駅まで、一緒にどうですか？」
彼の誘いに頷き、並んで歩く。夕暮れの光が二人の影を長く伸ばし、時折重なり合う。
影だけが、寄り添っている。

「来週、市民センターの図書室で読み聞かせのボランティアをしているんです。もしお時間があれば……」
別れ際、勇気を出して誘ってみた。断られるかもしれない。迷惑かもしれない。
彼は少し驚いた顔をして、すぐに破顔した。
「行きます。必ず行きます。君の声をもっと聴きたいですから」
その言葉は、どんな約束よりも嬉しかった。

翌週の土曜日。図書室の子供用スペースには、数組の親子連れが集まっていた。
私は絵本を開き、読み聞かせを始めた。緊張で指が少し震える。
視線の端に、一番後ろの席で静かに座っている木村さんの姿が見えた。
彼は約束通り来てくれた。
黒縁眼鏡の奥の瞳が、私をまっすぐに見つめている。その眼差しに支えられるように、声が自然と深みを増していく。物語の世界に没頭し、登場人物になりきって語る。子供たちの目が輝き、木村さんも深く頷きながら聴き入っている。

読み聞かせが終わった後、併設のカフェで二人でココアを飲んだ。
「緊張しました？ 最初、指が震えていましたよ」
彼が茶化すように言う。
「もう、意地悪ですね。あなたがじっと見ているからですよ」
「特等席で君の声を独り占めしている気分でした。心地よかった。まるで、子守唄を聴いているような安心感がありました」
湯気の向こうで、彼の低音が喉に落ちてくる。ココアの甘さと、彼の言葉の甘さが混ざり合う。
「甘いですね、ココア」
「君の声も、甘かったです」
不意打ちの言葉に、カップを持つ手が止まる。
「……からかわないでください」
「本心ですよ。妻を亡くしてから、こんなに穏やかな気持ちになったのは久しぶりです」
真剣な眼差しに、胸が締め付けられる。
「私もです。木村さんといると、心が凪いでいく気がします」

雨の日には、駅まで迎えに来てくれた彼と、一本の傘に入った。
「濡れますよ、もっと寄ってください」
肩を抱き寄せられ、雨の匂いが鼻をくすぐる。アスファルトが濡れる匂い。
「雨の匂い、好きですか？」
「ええ。世界が静かになって、香りが柔らかくなる気がして」
「僕もです。こうして君と歩いていると、雨音さえも音楽に聞こえる」
濡れた私の頬にかかった髪を、彼の手がそっと払った。その指の感触が、雨の冷たさとは対照的に、火傷しそうなほど熱かった。
傘という狭い空間。世界から切り離された二人だけの場所。

ある秋の日、公園のベンチで二人でスケッチブックを広げた。
「絵なんて、小学生以来ですよ」と苦笑する彼。
「いいじゃないですか。心のままに描けば」
色鉛筆を動かす音だけが響く静かな午後。彼が描いたのは、私の横顔だった。拙い線だけれど、そこには確かな温かさがあった。
「似てませんね」
「いいえ、とても素敵。私が知らない私みたい」
スケッチブックを閉じると、彼の手が私の手に重なった。
「美和子さん」
名前を呼ばれた。
「……はい」
「ありがとう」
ただそれだけの言葉。けれど、全てが詰まっていた。

夕暮れの校庭。運動会の片付けも終わり、静まり返った場所を二人で再訪した。
「ここが出会いでしたね」と彼が呟く。
「ええ。あの時、声をかけてくれてありがとう」
「君が、そこにいてくれてよかった」
彼の指が、私の頬の輪郭をゆっくりとなぞる。まるで、壊れ物を扱うように。
「ここに、何か残してもいいですか？」
唇は触れず、指先だけで頬に円を描く。そこに、キスよりも濃密な熱が宿った。
「……今夜、思い出してもいいですか？」
「ええ。あなたの指の熱を、ずっと覚えておきます」

家に戻り、一人になった部屋で、手のひらを鼻先に近づけた。
校庭の土の匂い、お弁当の唐揚げの香り、雨の匂い、そして木村さんの指の温度。それらすべてが混ざり合い、私の新しい「現在」を作っている。
鏡の中の自分に向かって囁く。
「今日、どの一歩を踏み出す？」
口紅を少し足し、指先に息を吹きかける。触れそうで触れない距離の甘さと、焦れったさを抱きしめながら。
孫の笑い声と、木村さんの低い声を耳の奥で反芻し、静かな夜の闇に心地よく沈んでいった。
夫への愛が消えたわけではない。ただ、心の中に新しい部屋ができただけ。
人生の黄昏時は、思ったよりもずっと、彩りに満ちている。
明日もまた、彼に会いたい。そう素直に思える自分が、愛おしかった。

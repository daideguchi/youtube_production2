CH03-013 台本
updated_at: 2026-01-14T03:14:08.313083Z
家族を名乗る電話やSMSが、本物そっくりに偽装される時代です。声も番号も作れます。焦らせる言葉に反応すると、冷静な判断が吹き飛びます。「今すぐ」「至急」「秘密に」と言われた瞬間にこそ、一拍置いて確認する習慣が命綱になります。ここでは、受けた瞬間に何をし、どう切り抜け、もし送ってしまったらどう動くかを、実際の流れに沿って丁寧にたどります。

典型的な流れは、突然の名乗りと切迫した事情の連続です。「事故を起こした」「示談金が必要」「家族に言えない」「今すぐ振り込んで」と急かし、家族の名前や学校名を口にして信用させます。最近はAI音声で家族の声を真似る例、発信者番号を家族や役所に偽装する例も増えています。番号や声だけで安心しないことが第一の防衛です。画面に映る番号が親の番号でも、留守番電話や合言葉で確認する癖を付けましょう。

還付や未納を餌にする言葉も定番です。「税金の還付」「医療費の返金」「公共料金の未納」「裁判になります」「少額で確認を」と言い、口座情報や暗証番号、少額振込を迫ります。URL付きSMSや短縮リンクで偽サイトに誘導する手口が多く、クリックするとIDやカード情報が盗まれ、不正アプリが入るリスクがあります。「還付はATMでできる」という誘導はすべて詐欺と覚えてください。役所や銀行が電話で暗証番号やワンタイムパスワードを聞くことはありません。

受けた瞬間の安全な動き方を一つの流れにします。まず、即答しないで電話を置く。「少し待ってください」と言って保留にし、その場で家族に自分から連絡します。普段の連絡先に折り返し、合言葉や共通の予定を確認する。名乗りが家族や役所でも、SMSに記載の番号にはかけ直さず、公式サイトや電話帳に載っている番号へ自分でかけ直す。暗証番号や口座番号を電話で求める正当な機関はありません。これを頭の中で繰り返し、口に出して練習すると、焦ったときに体が動きます。

SMSやメールの見分け方は、リンクを開かないことが出発点です。短縮URLや見慣れない文字列は開かず、内容が本当かは公式サイトに自分でアクセスして確認します。送信元や本文に氏名があっても安心材料にはなりません。家族にスクリーンショットを共有してから判断すると安全です。もしリンクを開いて入力した場合は、すぐに銀行や決済サービスへ連絡し、パスワード変更とカード停止を行い、警察や消費生活センターに相談してください。IDやパスワードを変え、二段階認証を見直すことも同時に進めます。

家族で事前に合言葉と連絡フローを決めておくと、声や番号が偽装されても迷いません。合言葉が言えない相手は、たとえ家族を名乗っても一旦切り、家族へ確認。連絡フローは「電話が来たら30秒以内に合言葉確認、3分以内に家族へ共有」と紙に書いて目につく場所に貼っておくと焦りにくくなります。夜間や不在時の連絡先も時間帯ごとに決めておくと安心です。連絡は家族グループのチャットに一言「こういう電話が来た」と残すだけでも、有事の連絡網が機能します。

送ってしまった場合の初動も順序を決めておきます。まず警察に被害届や相談票を出し、通話日時、相手の言葉、口座、録音や画面の記録をまとめます。次に金融機関や送金サービスに即連絡し、取引停止や凍結を依頼します。必要に応じて弁護士へ相談し、返還請求や手続きの助言を受けます。カードや決済の不正利用が疑われる場合は利用停止と再発行を同時に。証拠は通話履歴、SMS、メール、振込明細、通帳の写しなどを時系列でまとめ、日付を書き込みます。時間との勝負なので、迷ったら先に止める動きを優先します。

電話の設定も防御になります。知らない番号は留守番電話に任せ、録音された内容を家族と確認する。迷惑電話防止機能をオンにし、知らない番号は自動で警告を出す設定にする。携帯会社やスマホの標準機能で、なりすましを検知するサービスがあれば必ず設定します。固定電話も、ナンバーディスプレイと留守電の併用で、取る前に内容を聞く習慣を付けると被害が減ります。

日常の予防習慣はシンプルです。知らない番号は取らない。出ても保留にして確認する。個人情報や暗証番号は電話やSMSで伝えない。リンクは開かず、自分で公式サイトにアクセスする。週に一度、家族と合言葉と連絡手順を確認し、地域の相談窓口や警察の詐欺相談番号を手元に控える。落ち着いて確認する一呼吸が、被害を大きく減らします。

具体的な場面を二つ、練習として頭に入れておきます。ひとつ目は「事故を起こした」という電話を受けた場合です。声が家族に似ていても、まず「合言葉は？」と尋ねます。言えない場合は一旦切り、普段の番号に自分からかけ直します。留守なら別の家族に連絡し、警察署や病院に直接確認する。振り込みや電子マネーを指示されたら、すべて詐欺と判断し、動かない。二つ目は「還付金があります、ATMへ」というSMSを受けた場合。リンクは開かず、「役所名＋還付金＋詐欺」で検索し、詐欺情報がないか確認。役所の公式番号に自分でかけ、「こういうSMSが来たが本当か」と尋ねる。そこで本物なら書面で案内が届くはずなので、電話での誘導は拒否する、と決めておきます。

高齢の家族と共有する際は、短い紙を用意します。「合言葉」「公式番号にかけ直す」「暗証番号は言わない」「リンクは開かない」「迷ったら家族にすぐ連絡」の五行を大きな字で書き、電話のそばに貼る。合言葉は季節の野菜や思い出の場所など、覚えやすいものにする。紙を一緒に読み合わせ、週に一度は声に出して練習すると、いざというときに口から出てきます。

スマートフォンの設定も見直します。OSとアプリを最新にし、不要なアプリを削除。不審なアプリが入っていないか確認し、公式ストア以外からのインストールを禁止する。二段階認証を設定し、SMSではなく認証アプリを使うと、SIM入れ替えや番号乗っ取りに強くなります。パスコードは推測しやすい数字を避け、指紋や顔認証を活用する。万一の紛失に備え、端末を探す機能をオンにし、ロック画面に連絡先を入れておくと安心です。

LINEやメッセージアプリでのなりすましにも注意します。友人や家族の名前で突然「プリペイドカードを買って番号を教えて」といった依頼が来たら、まず電話で本人確認を。アカウントが乗っ取られた可能性があるため、グループにも「今こういうメッセージが来たが本人か？」と共有し、被害を広げないようにします。リンクが送られてきても、すぐに開かず、公式サイトや検索で確かめる癖を付けます。

被害に遭った人を責めないことも大切です。誰でもだまされる可能性があります。責めると次に相談しにくくなり、被害が大きくなります。まず取引を止め、連絡をし、証拠を残し、専門機関に相談する。感情はあとで整えれば十分です。周囲の支援が速いほど、被害の拡大を防ぎやすくなります。

地域や会社の相談先も手元に置きましょう。警察相談専用電話、消費生活センター、携帯会社やカード会社の緊急連絡先、市区町村の詐欺防止窓口。電話の近くに紙で貼り、家族全員が分かるようにしておくと、動き出しが早くなります。自治体の防犯メールや、銀行の注意喚起メールに登録しておくと、新しい手口を知ることができ、家族で共有できます。

心のゆとりを保つことが、最良の防御です。焦らされたら一呼吸。即答しない、一人で決めない、合言葉で確認する、公式番号にかけ直す。この流れを日常に埋め込むだけで、多くの被害を避けられます。今日、合言葉を決め、紙を貼り、家族に伝えましょう。騙されない力は、練習と準備で誰でも身につけられます。

ここからは、さらに細かい動きを一週間で練習するプランです。月曜は家族全員で合言葉を決め、紙に書いて電話のそばに貼る。火曜はスマホのOSとアプリを更新し、不要なアプリを削除する。水曜は銀行やカード会社、携帯会社の公式番号をメモに書き、財布と冷蔵庫に貼る。木曜は留守番電話と迷惑電話防止機能をオンにし、知らない番号は留守電に回す練習をする。金曜は家族で一度「詐欺のかかってきたふり」をして、合言葉確認と保留、かけ直しを実演する。土曜はメールとSMSの設定を見直し、二段階認証の方法を確認する。日曜は一週間の練習を振り返り、改善点を書き足す。七日で一巡し、紙が家族の地図になります。

銀行やATMの具体的な対策も付け加えます。銀行からの電話であっても、ATMでの操作を指示されたら詐欺です。還付や確認を理由にATMへ誘導するものはすべて無視し、銀行窓口に自分で行くか公式番号にかけ直す。ATMで困ったときに声をかけてくる人にも注意し、操作を見せない。周囲に怪しい人がいたら、その場を離れ、別の支店や時間帯を選ぶ。高齢の家族がATMに行くときは、なるべく同行し、静かな時間帯を選ぶと安心です。

キャッシュカードや通帳の管理も見直します。カードと暗証番号を一緒に持ち歩かず、暗証番号は誕生日や住所など推測しやすいものを避けます。通帳や印鑑は分けて保管し、長期間使わない口座は解約を検討する。インターネットバンキングのパスワードは定期的に変更し、同じものを他のサービスで使い回さない。ワンタイムパスワードを利用し、フィッシングサイトで入力しないよう、ブックマークした公式サイト以外では操作しないと決めます。

感情を揺さぶるメッセージの特徴を覚えておきます。「今すぐ」「秘密に」「ここに振り込んで」「このリンクを開いて」「口座番号と暗証番号を教えて」。これらの言葉が出たら、詐欺の可能性が高い合図です。相手が怒鳴ったり泣いたりしても、こちらは声を落とし、「確認してかけ直します」とだけ伝え、電話を切ります。感情ではなく手順で動くと被害は防ぎやすくなります。

もし家族や友人が被害に遭ったときは、責めずに一緒に動きます。「誰でも引っかかる可能性がある」「すぐ動けば被害は抑えられる」と伝え、取引停止と警察相談を先に済ませる。被害額が戻らないこともありますが、早いほど戻る可能性は高まります。悔しさや恥ずかしさを共有できる場として、地域の相談窓口や家族会を活用し、心の負担を軽くしてください。

新しい手口は常に生まれますが、守るポイントは変わりません。即答しない、自分でかけ直す、合言葉で確かめる、リンクを開かない、暗証番号は誰にも言わない。これだけで、多くの攻撃は跳ね返せます。今日、紙に五行を書き、家族と声に出して読み合わせることから始めてください。落ち着いた一呼吸が、家族の財産と安心を守ります。

ここからは、実際の場面で何を言うか、短いセリフも用意しておきます。「今は手が離せないので、後でかけ直します」「合言葉を教えてください」「公式番号にこちらからかけ直します」「家族に確認しますので切りますね」。この四つの言葉を繰り返し練習し、電話の近くに貼っておきます。パニックになっても、紙を見て読むだけならできるという安心が、判断を助けます。

AI音声の偽装に対しては、音の質や間だけでは見抜けません。だからこそ合言葉が有効です。「昨日の夕飯何を食べた？」「学生時代のあだ名は？」など、家族にしか分からない短い質問を用意し、答えられない相手は切る。家族が少人数でも、簡単な合言葉を一つ決めておくだけで、防御力が上がります。

家の固定電話に頼る方は、留守番電話を常時オンにし、録音後に折り返す運用に慣れてください。相手が慌てさせようとしても、「留守番電話を聞いてから折り返す」ことで一呼吸置けます。留守電には「詐欺対策で録音しています、用件をどうぞ」と入れておくと、詐欺が諦めやすくなります。

もし通話を録音できる設定なら、証拠になります。スマホは録音アプリや標準機能で録音をオンにし、怪しいと感じたら録音ボタンを押す。固定電話には録音機能付きの子機もあります。録音があれば、警察や金融機関に説明するときに役立ちます。

警察に相談するときは、感情より事実を先に伝えます。「〇月〇日〇時にこの番号から電話があり、この内容を言われ、振込先は〇〇銀行△△支店□□口座だった。録音・SMS・スクリーンショットがある」と簡潔に。被害がなければ相談でも構いません。相談番号を控え、家族に共有すると、次の行動がスムーズです。

海外からの不審な英語や中国語の自動音声も増えています。聞き取れない、自動音声である、選択を促すといった内容はすべて切って構いません。国際電話のような番号、不自然に長い番号、不明な短縮番号も取らないと決めておきます。

パソコンやタブレットでもフィッシングに注意します。ブラウザのアドレスバーを確認し、鍵マークがあっても偽サイトの場合があります。ブックマークした公式サイトだけを使い、メールやSNSのリンクからはログインしない。偽のポップアップで「ウイルスに感染しました」と表示されても、慌てて番号に電話しない。ブラウザを閉じ、セキュリティソフトでスキャンし、不安なら公式サポートに自分で連絡する。遠隔操作を許可するような案内は詐欺の合図です。

家族全員が使える「詐欺ノート」を一冊作るのもおすすめです。表紙に「怪しい連絡が来たときの手順」と書き、最初のページに合言葉、公式番号、相談先、セリフ例を書きます。後ろのページには、受けた詐欺の記録を日付ごとにメモする。繰り返し同じ手口が来ることが多く、記録を見返すと早く気付けます。ノートは電話の近くに置き、家族で共有します。

地域の支援を活用して、家族以外の目を増やします。自治体の防犯メールに登録し、近所で流行している手口を知る。民生委員や見守りボランティアに「怪しい電話が続いている」と伝え、気にかけてもらう。郵便局や金融機関でも、窓口での声かけが強化されています。「高額の振込をする方には確認します」という社会の目を味方にしてください。

日々のニュースや掲示板を見ていると、不安ばかりが募るかもしれません。大事なのは、恐れ続けることではなく、手順を体に染み込ませることです。落ち着いて、即答しないで切る、合言葉で確かめる、自分で公式番号にかけ直す、暗証番号を伝えない、リンクを開かない。この五つを守るだけで、ほとんどの詐欺は防げます。家族に一度でも声に出して伝えれば、防御力は跳ね上がります。

最後に、今日やることを三つに絞ります。一つ目は合言葉を決め、紙に書いて電話のそばに貼る。二つ目は公式番号と相談先をメモにして、財布と冷蔵庫に貼る。三つ目はスマホと固定電話の留守番電話・迷惑電話設定を確認する。これで、いざというときの足場ができます。明日からは週一回の読み合わせと、怪しい連絡が来たときのシミュレーションを家族で一度やってみてください。一呼吸の余裕が、家族の財産と心を守ります。

万一、振り込んでしまったあとに気づいたときは、時間との競争です。まず送金先の金融機関に電話し、取引停止や組戻しを依頼する。次に自分の銀行やカード会社、決済サービスに連絡し、不正利用の監視や停止を依頼。警察の相談窓口に連絡し、被害届または相談票を提出する。これらを家族や友人に分担してもらい、同時に進めると止まる可能性が高まります。恥ずかしさよりもスピードを優先してください。

電話を取らない選択肢も、立派な防御です。体調の悪い日や気力のない日は、知らない番号をすべて留守電に回し、落ち着いてから聞く。留守電の内容を家族と聞いて判断する。力が戻ってから動けば十分です。詐欺は焦りを突いてきますから、体調管理も防御の一つです。

家族に相談しづらいときは、外の窓口を使います。消費生活センターや警察相談専用電話、自治体の高齢者相談窓口、銀行やカード会社のコールセンター。ここに「確認したい電話があった」「こういうSMSが来た」と伝えるだけで、客観的な意見がもらえます。一人で悩むより、第三者の声を聞く方が、冷静な判断が戻りやすいものです。

詐欺の電話やSMSは、何度も何度も形を変えて届きます。それでも、あなたが守るルールは変わりません。即答しない、自分でかけ直す、合言葉で確かめる、リンクを開かない、暗証番号を伝えない。今日、紙に書き、声に出して家族と共有してください。落ち着いた一呼吸が、偽の声や番号を跳ね返します。準備と練習で、だまされない力は確実に育ちます。

明日は一つだけ決めましょう。「知らない番号は取らず、家族に共有してから折り返す」。これができれば大きな前進です。家族と「今日は怪しい連絡がなかった」「こんなSMSが来た」と一言交わす習慣をつければ、安心のネットが広がります。ゆっくりで構いません。一呼吸の余裕が、家族の安心と財産を守ります。

だまされない自信は、今日の小さな準備から生まれます。電話のそばの紙、財布の中のメモ、家族との合言葉。それだけで、偽の声もリンクも、あなたを揺らしにくくなります。安心して日々を過ごすために、今この瞬間の準備を積み重ねてください。

そして一週間後、紙が増え、家族の会話が増え、落ち着いて対応できた自分に気づくはずです。詐欺の波は止まりませんが、あなたの守りは確実に強くなります。今日の小さな準備が、明日の大きな安心につながります。

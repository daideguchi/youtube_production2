こんばんは。静かな夜の心理図書室です。

今日も一日、おつかれさまでした。仕事や家事や、誰にも見えない「気遣い」の分まで、あなたの神経はよく働いてくれましたね。今、あなたはどんな姿勢でこの声を聞いているでしょうか。ベッドの上で横向きになりながら、スマホの小さな光だけが枕元を照らしているかもしれません。部屋の灯りを落として、カーテンの隙間から差し込む街灯や月の光が、天井にうっすらと模様を作っているかもしれません。あるいは、真っ暗な中でまぶたを閉じているのに、頭の内側だけはまだ白い蛍光灯の下にいるような、そんな感覚かもしれませんね。

眠ろうとしているのに、指はつい、タイムラインや動画のおすすめをスクロールしてしまう。そこで目に入るのは、「現実を無視するだけで金持ちに」「願いは全部叶う」「引き寄せの秘密」といった、どこか挑発的で、どこか甘い言葉たち。半分は「そんなわけない」と思いながら、もう半分は「もし少しでも本当なら知りたい」と思ってしまう。疲れ切った夜ほど、そういう言葉は強く胸に刺さります。

今夜のテーマは、「現実を無視するだけで金持ちに？ネヴィル・ゴダードの秘密」。タイトルだけ見ると、かなり過激に聞こえますよね。現実を無視する？金持ちに？とツッコミを入れたくもなります。でも、こうした言葉の背景には、本来もう少し繊細で、静かなニュアンスがありました。それが極端に切り取られて、刺激的なコピーになっていることも多いのです。

わたしはここで、あなたに「現実なんて見なくていい」「お金のことも問題も全部無視していい」と言いたいわけではありません。むしろ、その逆です。現実から浮き上がらないこと。地面に足をつけたまま、心のほうのレンズを少しだけ調整すること。その視点で、ネヴィル・ゴダードの話を、静かに読み直してみたいのです。

そのための道具として、今夜は「月」と「本」を使いましょう。夜空をゆっくり眺めること。本を一行だけ読むように、未来のイメージを胸の中でめくること。その二つが合わさるとき、「現実を無視する」という言葉が、少しだけ別の意味を帯びて立ち上がってきます。

よかったら、まずは一度、深めの息を吐いてみてください。鼻から軽く吸って、口から、ため息に近い息を長く吐き出します。今日一日分の「現実」の重さを、いったん胸の前から床のほうへ流していくイメージで。吐ききったあとの静けさの中に、今夜の物語を置いていきましょう。

・・・

ある夜のことを想像してみてください。

仮に、その人を「美穂」と呼びます。

美穂は、最近、眠りが浅くなっていました。仕事はなんとか回しているけれど、給料は思うように増えない。物価はじわじわ上がっていく。将来のお金のことを考えると、胸のあたりが冷たくなる。

夜になると、スマホを開いて、つい「お金」「豊かさ」「引き寄せ」といった言葉で検索してしまいます。出てくるのは、眩しすぎる成功体験の話や、極端なスピリチュアルの約束。「現実を無視すれば金持ちに」「通帳の残高なんて見るな」「借金も病気も目に入れるな」。そんなフレーズを読むたびに、どこかざわつきました。

確かに、現実を見てばかりいると苦しくなる。でも、無視したら本当に大丈夫なのか。支払いの締め切りは、意識を向けなくてもやってくる。冷蔵庫の中身も、光熱費の請求も、スマホ代も。見ないふりをして消えてくれるわけじゃない。それくらい、美穂だって分かっています。

ある日、美穂は本屋で一冊の本を見つけました。薄い表紙に、外国人の名前。「ネヴィル・ゴダード」。帯には「想像力が現実をつくる」とあります。正直に言えば、最初はまたよくある自己啓発本かな、と疑っていました。でも、ページを開いてみると、そこには意外な静けさがありました。派手な成功の話ではなく、夜眠る前の短い時間をどう使うか、というような、地味な話が淡々と書かれていたのです。

それは、「現実を否定しろ」とか「問題から目をそらせ」と言うのではありませんでした。むしろ、今ある事実は事実として認めたうえで、その事実に対して自分がいつもどんな物語を語り続けているのかに気づき、それを少し調整する、という話でした。

考えてみれば、わたしたちが「現実」と呼んでいるものは、「起きている出来事」そのものと、「それに付けている意味」の組み合わせです。給料という数字そのものと、「こんな額じゃ無理だ」というナレーション。一通の請求書と、「また支払いに追われる」というため息。孤独な夜と、「わたしは愛されていない」というラベル。

起きている出来事は、一瞬で変えられないことが多い。でも、「どう意味づけるか」は、少しずつ変えていくことができます。

ネヴィル・ゴダードが言おうとしていた「現実を無視する」は、「支払いを放り出せ」という乱暴な話ではなく、「今目の前にある数字を、唯一の真理だと決めつける見方から、一時的に離れてみなさい」という提案でもありました。

つまり、一冊の本の一行目を書き換えるようなものです。

たとえば、こんな現実があります。

「今月もギリギリだ。将来が不安だ。」

この一行目で始まる物語を毎日読み返していれば、心はいつも同じ結論に行き着きます。「わたしはダメだ。何をやっても変わらない。」

でも、同じ数字を見ていても、こう書き換える余地があるかもしれません。

「今月もギリギリだった。だからこそ、心の使い方を練習するチャンスでもある。」

数字は変わっていません。ただ、物語の開き方が変わった。そのとき、わたしたちの中の「感じ方」が少し変わります。

「現実を無視する」とは、この「意味づけの一行目を書き換えるトレーニング」と捉えてみると、だいぶ落ち着いてきませんか。

美穂は、その夜、ベッドサイドのライトを消す前に、ネヴィルの本を開きました。窓の外には、運良く薄く雲のかかった月が浮かんでいました。派手ではない、少し霞んだ光。それでも、暗い空の中で、そこだけが静かに白んでいる。

美穂は、カーテンを少し開けて、月が見える位置に枕をずらしました。手には本を持ったまま。ページを数行だけ読んでは、月を見上げる。

そこには、こんな趣旨のことが書かれていました。

眠る前の短い時間に、すでに願いがかなったあとの自分を、静かに「感じてみる」。それは現実逃避ではなく、心に新しいパターンを教える行為だということ。

神経科学的に言えば、脳は「強くイメージされたこと」を、どこかで現実の記憶と似た扱いをします。スポーツ選手がイメージトレーニングをするように。実際に体を動かさずとも、頭の中で何度も成功のシーンを思い描くと、その動きに関わる神経回路は少しずつ太くなると言われています。

ネヴィルが語った「想像力」の話も、どこかそれに似ています。

「現実を無視して金持ちになる」ではなく、「今の数字だけが真実だという思い込みを一度ゆるめて、安心している自分の感覚を先に体験してみる」。その感覚が、やがて行動や選択の仕方を変え、結果として現実が少しずつ変わっていく、という流れです。

だから、本当はこんな順番なのだと思います。

現実を無視 → 金持ち、ではなく、
現実にのみ囚われる視野を一度ゆるめる → 心の中に「安心した状態」の感覚を育てる → その安心感から選んだ行動が増える → 現実の質が静かに変わる。

どこにも「支払いを放棄してよい」とは書いていません。ただ、「数字だけを見て自分を罰し続ける癖から、一度離れてみませんか」という提案があるだけ。

ここで、「月」と「本」が出てきます。

美穂は、その夜、ネヴィルの本を閉じて、月の光だけを見つめることにしました。

現実の数字は、一旦机の上に置いておく。通帳も、明細も、今日は触らない。代わりに、心の中で「別の本」を一冊開いてみることにしたのです。

それは、こんな物語でした。

少し先の未来のある夜。今と同じように、月が出ている。美穂は、同じ部屋で、枕元に座っている。でも、そのときの彼女の胸の中には、「なんとかやっていけている」という、ささやかな安心感があります。

大金持ちになっている必要はありません。むしろ、「月に一度は好きな本を買える」「突然の出費があっても、前ほどパニックにならない」そんなレベルでいい。

その未来の美穂は、窓の外の月を見ながら、こう思っています。「あの頃、本当に苦しかったけれど、あの夜から少しずつ変わってきたな」。

ネヴィルは、願いが叶ったあとの「終わりのシーン」を短く切り取って、その場面を眠る前に何度も味わうことを勧めました。美穂は、その教えを、自分なりにゆっくりと実験してみることにしたのです。

目を閉じて、月の光がまぶたをうっすら透かす感覚を感じながら、「少し先の夜の自分」を思い描く。そこで感じている安心を、今の胸のあたりに、少しずつ重ねていく。

そのとき、現実の数字は変わっていません。通帳の中身は、朝と同じです。支払いの期限も変わらない。

それでも、美穂の中で一つだけ変わったものがありました。「ずっとこのままだ」という諦めと、「絶対にこうならなければ」という力みのあいだに、小さな余白が生まれたのです。

余白が生まれると、人は「選べる」ようになります。

「どうせ無理だから何もしない」か「怖いから現実から目をそらす」かの両極だけではなく、「怖さや不安はそのままにしたまま、今日できる小さな一歩だけはやってみようかな」と思えるようになる。

翌朝、美穂は、いきなりビジネスを始めたわけでも、莫大な収入源を見つけたわけでもありませんでした。ただ、いつもならなんとなく眺めて終わっていた「家計簿アプリ」を開き、三分だけ、今月使ったお金の内訳を眺めてみました。

不思議なことに、その三分が前ほど苦痛ではありませんでした。そこには、「数字＝自分の価値」と直結させる鋭さが、ほんの少し和らいでいたからです。

これが、「月と本」を使ったほんの小さな現実との付き合い方の変化です。

ここまで聞いて、「そんなゆっくりで間に合うのかな」と感じるかもしれませんね。将来の不安や孤独感が強いとき、人はどうしても「一発逆転」の話に惹かれます。「この方法さえやれば」「この言葉さえ信じれば」すべてが一気に変わる、と言ってくれる存在を求めてしまう。

でも、あなたの神経にとって、そしてあなたの心にとって、本当に安心につながる変化は、多くの場合「静かで」「段階的」で「退屈に見える」ものです。

現実をまるごと否定してしまうと、一時的にはスッキリした気分になるかもしれませんが、後から必ず大きな揺り戻しが来ます。支払い、仕事、身体、対人関係。それら全ては、わたしたちの人生の「土台」です。

ネヴィルの話を夜の読書として味わうとき、大事にしたいのは、「土台を放棄するための魔法の呪文」としてではなく、「土台の上でどう立つかを調整するための姿勢」として受け取ることです。

ここから、あなた自身が今夜できる、小さな「月と本」のリチュアルを描いてみましょう。

もし今、カーテンを少し開けられるなら、ほんの少しだけ隙間を作ってみてください。外に月が見えなくても、構いません。雲に隠れていても、ビルの陰になっていても、見えないところで必ず今夜の月はどこかに浮かんでいます。月そのものが見えなくても、「月のいる夜の空」を感じるだけで十分です。

部屋の灯りを、可能であれば少し落として、柔らかい光だけにします。スタンドライトがあるなら、目線より少し低い位置に置いて、手元だけを照らします。

手元に、本を一冊置いてみましょう。ネヴィルの本でもいいし、まったく別の本でも構いません。あなたが安心する文章が載っている本なら何でもいいのです。

その本の中から、たった一行でいいので、今夜の自分に寄り添ってくれそうな一文を探します。疲れているなら、「今日はここまででいい」と言ってくれているような一文でもいいし、「小さな一歩」という言葉が出てくる一文でもいい。なかなか見つからなければ、表紙を撫でるだけでもかまいません。本の存在そのものを、「今夜は一人じゃない」というサインにしてもいいのです。

本を閉じたら、窓の方向を一度だけ見上げます。月が見えたら、その白い光を。見えなくても、空の向こうで光っているはずの場所を、ぼんやりと想像します。

そこで、心の中で、今の自分の「現実の一行目」をそっとつぶやいてみます。

「今、お金のことが不安で、胸がざわざわしている」
「今、孤独で、この先ずっと一人なのかと怖くなる」
「今、眠れなくて、また明日の自分がぼろぼろになるのが怖い」

どんな内容でも構いません。大事なのは、それを「事実と感情を分けて」言ってみることです。

「通帳の残高が少ない」
「その数字を見て、わたしは不安を感じている」

こうやって分けていくと、「現実そのもの」と「そこに貼り付いた意味」が、ほんの少し離れます。それだけでも、心には少し空間が生まれます。

そのあとで、さきほどのネヴィル風のイメージを、小さく足してみます。

「もし、この不安が少しだけ和らいでいる夜が来るとしたら、その夜のわたしは、どんなふうに月を見ているだろう」

そこで、未来の自分を、短いシーンとして思い描きます。

例えば、こういうシーンです。

数か月後のある夜。今日と同じように、美穂……ではなく「あなた」は、窓の外の月を見ています。部屋の中には、今日と同じ家具が置いてあり、同じ布団が敷かれています。通帳の残高も、劇的に増えてはいないかもしれません。でも、そのときのあなたは、今夜よりも少し深く眠れるようになっていて、心の中には「なんとかやっていけそう」という感覚が、今より少しだけ増えています。

そのシーンで、あなたは誰かにこう話しているかもしれません。「あの頃、本当に眠れなくて、未来が真っ暗に見えていた。でも、あの夜から、月を見る時間を増やして、心の中で『今夜はここまででいい』と何度も言い聞かせるようにしたんだよ」

その短い会話を、あなたは心の中で「聞いている」だけでいいのです。

ネヴィルは、「感情を伴ったイメージ」が力を持つと言いました。科学的にも、「安心しているときの身体の感覚」を何度も再現することは、神経系のバランスを整え、選ぶ行動の質を変えていくと考えられています。

ここで大切なのは、「一気に金持ちに」をイメージする必要はない、ということです。あなたの神経が受け止めきれないほどの非現実的なイメージは、かえって反発を生みます。「そんなの無理に決まっている」と心のどこかでツッコミが入り、余計に苦しくなってしまうことさえあります。

だからこそ、「少し楽になっている未来」を思い描くこと。それを、何度も何度も夜に読み返す「心の中の本」として扱うこと。それくらいがちょうどいいのです。

ここまでをひと通りイメージしたら、最後に、現実のほうを一度だけ見ます。

明日の予定。やるべきこと。支払いの期日。眠らなければ体に響くこと。

それらをぜんぶ、一枚の紙に書き出して……とまではしなくていいので、「明日、自分ができそうな一歩」を心の中で選びます。

「明日は、五分だけ家計簿アプリを見る」
「明日は、一人で抱え込んでいる不安を、一行だけノートに書いてみる」
「明日は、誰か一人にだけ『最近ちょっとしんどくてね』と打ち明けてみる」

その一歩を選んだら、今夜の自分にこう伝えます。

「だから、今夜はここまででいい。これ以上は、月と夜空に預ける」

月を見ることは、現実逃避ではありません。地球のすべての場所で同じ月が見えているという事実は、わたしたちの孤独感を少し和らげてくれます。今、この瞬間、同じ月の下で、自分と似た不安を抱えている人が必ずどこかにいる。その人たちもまた、眠ろうとしながら、今日をなんとか終わらせようとしている。

ネヴィル・ゴダードの話を、夜の月と一緒に受け取るとき、それは「全部自分のせい」と責めるための教えでも、「全部宇宙のせい」と丸投げするための教えでもなく、こういう静かなメッセージに変わっていきます。

「あなたの心は、もっとやさしい物語を選び直していい」
「現実に押しつぶされそうなときは、一時的にレンズを外して、月を見る時間を持っていい」
「そのうえで、明日、無理のない一歩だけを現実に返していけばいい」

もし今、眠れない状態が続いていたり、日常生活に支障が出るほどの不安があるなら、セルフワークだけに頼らず、医療やカウンセリングの専門家に相談することも、とても大切です。薬をやめろとか、現実の問題から目をそらせという話ではありません。むしろ、そうした現実的なサポートと並行して、「心の中で少しだけやわらかい物語を読み直す時間」を持つことで、あなたの内側のバランスは、いっそう整いやすくなります。

最後に、今この瞬間のあなたの呼吸に、もう一度だけ意識を向けてみましょう。

息を吸うときに、心の中で「今夜は」とつぶやきます。
息を吐くときに、心の中で「ここまででいい」とつぶやきます。

もう一度、吸うときに「今夜は」。
吐くときに「ここまででいい」。

それを何度か、あなたのペースで繰り返してみてください。まぶたの裏で、ぼんやりと月の色を思い浮かべながら。

現実を無理に美化する必要はありません。辛いものは辛いし、不安なものは不安です。その正直さを失わないまま、「その全部を一晩で何とかしようとしない」という選択を、今夜だけは許してあげてください。

ネヴィル・ゴダードの「秘密」は、派手な約束ではなく、本当はとても地味でやさしいのかもしれません。心の中の物語を少し変えることで、現実と付き合う姿勢が変わり、その結果、人生の感触が静かに変わっていく。その変化は、月の満ち欠けのようにゆっくりですが、確かに続いていきます。

今夜は、ここまでにしましょう。ここから先のことは、あなたの呼吸と、窓の外にどこかで浮かんでいる月と、静かな夜の空気に、そっと預けてしまって大丈夫です。

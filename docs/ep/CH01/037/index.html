# 【もう他人に振り回されない】ブッダの「自灯明」の教えに学ぶ、”自分軸”で生きるためのヒント

## 導入
私たちの身の回りにある製品。
それらがどのように作られているのか、知っていますか？
今日は、【もう他人に振り回されない】ブッダの「自灯明」の教えに学ぶ、”自分軸”で生きるためのヒントの製造工程に潜入します。

## 技術の進化

より安全に、より高品質に。
技術の進化は、止まることを知りません。

## 結論
【もう他人に振り回されない】ブッダの「自灯明」の教えに学ぶ、”自分軸”で生きるためのヒント。

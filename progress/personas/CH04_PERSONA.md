## 1. 共通ペルソナ
> 30〜50代で知的好奇心が強く、心理実験や隠れた歴史・神秘の話から日常を俯瞰したいと思っている人。

- `ターゲット層` はこの文で固定。企画ごとに書き換えない。
- 記事・論文・一次資料へのリンクは `企画意図` に必ず記載する。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
|---|---|---|
| 悩みタグ_メイン | 主題 | 潜在意識 / 記憶 / 直感 / シンクロ / 未解決事件 / 神話 |
| 悩みタグ_サブ | 補助 | 仕事のひらめき / 人間観察 / 心理トリック |
| ライフシーン | 場面 | 深夜の書斎 / オフィスでの休憩 / 旧図書館 / 雨の窓辺 |
| キーコンセプト | 視点 | 集合的無意識 / アカシックレコード / ユング / 認知実験 |
| ベネフィット一言 | 効果 | 「日常の違和感を言語化できる」「新しい視点が手に入る」 |
| たとえ話イメージ | サムネ/本文 | 鍵穴から覗く / 本の迷路 / 光る糸を辿る |
| 説明文_リード | 冒頭 | 「日常の裏側で進む心理実験とアーカイブの話を覗きに行きましょう」 |
| 説明文_この動画でわかること | 箇条書き | ・XX実験の真相\n・今日から使える心理トリック |

## 3. 記入ルール
1. タイトルは「【秘密/心理】×【身近な疑問】」。読後感は余韻を残す。 
2. 構成案は `【導入】→【謎/実験紹介】→【考察】→【日常への翻訳】→【問いかけ】`。
3. サムネは暗めの背景＋光るモチーフ。コピーは12〜14文字で謎を匂わせる。

## 4. テンプレ
- `progress/templates/CH04_planning_template.csv` をコピーし、Analytics根拠や引用を差し替える。
- タグ列・説明文列は必ず記入し、ミステリー要素と学びの両方を明示する。

1. 企画追加後 `planning_store.refresh`。
2. `progress_manager.py verify --channel-code CH04` を実行し、列をチェック。
3. UI でタグ列・説明文が表示されることを確認する。

# CH09 Persona & Planning Template

## 0. テンプレ SoT
- SoT: `progress/templates/CH09_planning_template.csv`
- 使い方: ヘッダー＋サンプル1行をコピーし、`{NEXT_NO}` / `{NEXT_VIDEO}` を差し替えて `progress/channels/CH09.csv` に追加 → `planning_store.refresh --force`。

## 1. 共通ペルソナ（固定一文）
> 30〜50代のビジネスパーソン。日々の意思決定で疲弊し、揺れない軸を持ちたい人。

- 生活導線: 朝のデスク、移動中、会議前の数分で視聴。
- テーマの軸: 「価値観メモ」「行動原則」「意思決定のテンプレ」を短時間で渡す。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
| --- | --- | --- |
| 悩みタグ_メイン | 主題 | 決断疲れ / 時間不足 / 優先順位 |
| 悩みタグ_サブ | 補助 | キャパ超え / 情報過多 / 迷い |
| ライフシーン | 場面 | 朝のデスク / 通勤中 / 会議前 |
| キーコンセプト | 視点 | 不動心 / 価値観 / ジャーナリング / 最小行動 |
| ベネフィット一言 | 効果 | 「迷わず動ける安心」 |
| たとえ話イメージ | サムネ/本文 | 山の尾根道 / 羅針盤 / 重りを一つ置く |
| 説明文_リード | 冒頭 | 「今日の判断を、自分の軸でシンプルに。」 |
| 説明文_この動画でわかること | 箇条書き | ・価値観メモの書き方<br>・迷った瞬間のチェックリスト |

## 3. 記入ルール
1. タイトルは「ブレない軸」「決断疲れ」「一日〇〇」で 28〜34 文字。
2. 構成案は `【導入（消耗描写）】→【視点（軸の定義）】→【ステップ（3手順）】→【結び（宣言）】`。
3. サムネ: ミニマルなデスク、ノート、暖色光。文字は白＋赤でコントラスト強め。

## 4. ベンチマーク
- 「ミニマリスト思考」「タスク管理・自己管理系」チャンネルの短尺構成。
- ストア派/禅思考の引用で「静かな決断」を強調。

## 1. 共通ペルソナ
> 20〜40代で、都市伝説や陰謀論、失われた歴史にワクワクしつつも、裏付けや考察で「本当かもしれない」と納得したい好奇心旺盛な視聴者。

- `ターゲット層` はこの文で固定。企画ごとに書き換えない。
- 裏付けに使った資料（論文・ニュース・書籍）は `企画意図` に必ず記載する。

## 2. タグセット
| フィールド | 役割 | 推奨入力例 |
|---|---|---|
| 悩みタグ_メイン | 主題 | UFO / 失われた文明 / 政府陰謀 / 心霊 / オーパーツ / 禁書 |
| 悩みタグ_サブ | 補助 | 科学考察 / 軍事 / 宗教儀式 / 目撃証言 |
| ライフシーン | 場面 | 深夜のPC / 地下アーカイブ / 砂漠遺跡 / 海底神殿 |
| キーコンセプト | 視点 | 輪廻 / カルマ / 秘密結社 / 黒魔術 |
| ベネフィット一言 | 効果 | 「裏付け付きで語れる都市伝説」「真相に一歩近づく考察」 |
| たとえ話イメージ | サムネ/本文 | 封印された文書 / 目が光る像 / DNAの螺旋 |
| 説明文_リード | 冒頭 | 「眠れない夜に、封印された噂の裏側を覗きませんか？」 |
| 説明文_この動画でわかること | 箇条書き | ・事件の年表\n・真相仮説と根拠 |

## 3. 記入ルール
1. タイトルは「【禁断ワード】×【仮説】」で 30〜34 文字。 
2. 構成案は `【導入】→【事件/伝説の要約】→【証拠】→【仮説A/B】→【視聴者への問い】`。
3. サムネは暗所＋赤/緑のアクセント。コピーは12文字前後で緊張感を出す。

## 4. テンプレ
- `progress/templates/CH06_planning_template.csv` をコピーし、事件名・証拠・仮説を書き換える。
- タグ列と説明文列で「どの謎をどんな切り口で掘るか」を明記する。

## 5. 運用
1. 企画追加→`planning_store.refresh`。
2. `progress_manager.py verify --channel-code CH06` を実行し、列ズレがないか確認。
3. UIでタグ列が表示されることを確認し、必要なら CapCut/音声タブにも共有する。

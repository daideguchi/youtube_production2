## 1. 共通ペルソナ
> 50〜70代で体力や健康に不安を抱えつつ、病院だけに頼らず日々の習慣で身体と心を整えたいと願うシニア。

- `ターゲット層` はこの文で固定。
- 健康情報は必ず一次情報（医師監修記事、論文、行政データ）を `企画意図` に記載する。

## 2. 企画ごとのタグ
| フィールド | 役割 | 推奨入力例 |
|---|---|---|
| 悩みタグ_メイン | 主訴 | 血圧 / 睡眠 / 膝痛 / 呼吸 / 食生活 / メンタルケア |
| 悩みタグ_サブ | 補助 | 介護疲れ / 孤独 / 生活リズム / 冷え / フレイル |
| ライフシーン | 場面 | 朝の台所 / 夕方の散歩 / 通院帰り / 就寝前 / 旅行準備 |
| キーコンセプト | 視点 | 生活習慣病予防 / 姿勢改善 / 血流ケア / 養生訓 / 医師コメント |
| ベネフィット一言 | 効果 | 「5分ストレッチで脚が軽くなる」「血圧が落ち着く呼吸法」 |
| たとえ話イメージ | サムネ/本文 | 体のサビ落とし / 心の湯たんぽ / 楽器のチューニング |
| 説明文_リード | 冒頭 | 「病院に行くほどではない不調を、今日の生活習慣で整えましょう」 |
| 説明文_この動画でわかること | 箇条書き | ・膝を守る座り方\n・血流を促す足指ケア |

## 3. 記入ルール
1. タイトルは「【症状】×【優しい習慣】」。数字や医師コメントを盛り込む。 
2. 構成案は `【導入】→【症状の背景】→【専門家/データ】→【実践ステップ】→【注意点】→【励まし】`。
3. サムネは「生活シーン＋ケア動作」。コピーは12文字前後で安心感を出す。

## 4. テンプレ

- 医療情報の引用元 URL は `企画意図` に明記すること。
